# 电路基础入门

## 一、电流、电压、电阻

### 1.1 用水管类比

把电路想象成一个**水管系统**：

```
  水泵 ←→ 电池（提供动力）
  水流 ←→ 电流（流动的东西）
  水压 ←→ 电压（推动流动的力）
  水管粗细 ←→ 电阻（阻碍流动的程度）
```

| 电路概念 | 水管类比 | 单位 | 符号 |
|---------|---------|------|------|
| 电压 | 水压 | 伏特(V) | U 或 V |
| 电流 | 水流量 | 安培(A) | I |
| 电阻 | 管道阻力 | 欧姆(Ω) | R |

### 1.2 电压是什么？

电压就是**电的"推动力"**，两点之间的电压差让电流从高电位流向低电位。

```
  高电位 ────── 电阻 ────── 低电位
  （正极）      ↑           （负极）
              电流方向
           （从高到低）
```

常见电压参考：
| 来源 | 电压 |
|------|------|
| USB接口 | 5V |
| 手机电池 | 3.7V |
| 干电池(1号/5号) | 1.5V |
| 汽车电瓶 | 12V |
| 家用插座(中国) | 220V |
| 高压输电线 | 110kV-1000kV |

### 1.3 电流是什么？

电流就是**电荷的定向移动**。

- 1安培 = 每秒流过1库仑的电荷
- 1库仑 ≈ 6.24×10¹⁸个电子
- 实际电路中，常用mA（毫安）和μA（微安）

```
  电流方向的约定：
  
  传统电流方向：从正极 → 负极（外部电路）
  电子实际移动：从负极 → 正极（电子带负电，被正极吸引）
  
  我们通常使用"传统电流方向"
```

### 1.4 电阻是什么？

电阻就是**材料对电流的阻碍作用**。

```
  电阻定律：R = ρ × L / A
  
  ρ = 电阻率（材料特性）
  L = 导体长度
  A = 导体截面积
  
  越长 → 电阻越大
  越粗 → 电阻越小
```

常见材料的电阻率：
| 材料 | 电阻率(Ω·m) | 类型 |
|------|-------------|------|
| 银 | 1.6×10⁻⁸ | 导体 |
| 铜 | 1.7×10⁻⁸ | 导体 |
| 铝 | 2.8×10⁻⁸ | 导体 |
| 硅 | 6.4×10² | 半导体 |
| 玻璃 | 10¹⁰~10¹⁴ | 绝缘体 |

## 二、欧姆定律——电路第一定律

### 2.1 公式

$$U = I \times R$$

三个变形：
```
U = I × R     → 已知电流和电阻，求电压
I = U / R     → 已知电压和电阻，求电流
R = U / I     → 已知电压和电流，求电阻
```

### 2.2 实例

```
一个LED串联300Ω电阻，接5V电源：
LED正向压降 = 2V（红光LED）

电阻上的电压 = 5V - 2V = 3V
电流 = 3V / 300Ω = 10mA

判断：10mA对LED来说是安全的（通常最大20mA）
```

## 三、功率

### 3.1 功率公式

$$P = U \times I = I^2 \times R = U^2 / R$$

- 单位：瓦特(W)
- P = 1W 表示每秒消耗1焦耳的能量

### 3.2 实例

```
上面的LED电路：
P(电阻) = 3V × 10mA = 30mW（很小，1/8W电阻就够）
P(LED) = 2V × 10mA = 20mW
总功率 = 5V × 10mA = 50mW
```

## 四、串并联电路

### 4.1 串联电路

```
  R1      R2
──┤├──┤├──
```

**特点：**
- 电流处处相等：I₁ = I₂ = I
- 电压分压：U = U₁ + U₂
- 总电阻：R = R₁ + R₂

```
分压公式：
U₁ = U × R₁ / (R₁ + R₂)
U₂ = U × R₂ / (R₁ + R₂)
```

**应用：分压电路**

```
         VCC (5V)
          │
          R1 (10kΩ)
          │
          ├──── Vout = VCC × R2/(R1+R2)
          │           = 5 × 10/(10+10) = 2.5V
          R2 (10kΩ)
          │
         GND
```

### 4.2 并联电路

```
     ┌── R1 ──┐
  ───┤         ├───
     └── R2 ──┘
```

**特点：**
- 电压处处相等：U₁ = U₂ = U
- 电流分流：I = I₁ + I₂
- 总电阻：1/R = 1/R₁ + 1/R₂（即 R = R₁×R₂/(R₁+R₂)）

```
分流公式：
I₁ = I × R₂ / (R₁ + R₂)
I₂ = I × R₁ / (R₁ + R₂)
```

## 五、电容

### 5.1 什么是电容？

电容就像一个**小电池**，能储存电荷。

```
  结构：两块金属板中间夹着绝缘材料
  
  ┌─────┐
  │金属板│
  ├─────┤
  │绝缘层│
  ├─────┤
  │金属板│
  └─────┘
  
  符号：──┤├── 或 ──||──
```

### 5.2 电容的单位

$$C = Q / U$$

- C：电容，单位法拉(F)
- Q：储存的电荷，单位库仑(C)
- U：两端电压

常用单位换算：
```
1F = 1000mF（毫法）
1mF = 1000μF（微法）  ← 最常用
1μF = 1000nF（纳法）
1nF = 1000pF（皮法）  ← 也很常用
```

### 5.3 电容的特性

**隔直通交：**
- 直流电：电容充满后就不通了（像堵住的水管）
- 交流电：电容不断充放电，看起来像在导通

```
充电过程（接直流）：
  电压 ↑
  Vmax│         ___________  ← 充满
      │        ╱
      │       ╱
      │      ╱
      │     ╱
      │    ╱
      └─────────────────→ 时间
        τ=RC（时间常数）

  τ = R × C
  约5τ后充满（99%）
```

### 5.4 常见电容类型

| 类型 | 容量范围 | 特点 | 用途 |
|------|---------|------|------|
| 电解电容 | 1μF~10000μF | 有极性，容量大 | 电源滤波 |
| 陶瓷电容 | 1pF~1μF | 无极性，高频好 | 去耦、旁路 |
| 钽电容 | 0.1μF~1000μF | 稳定，贵 | 高可靠电路 |
| 薄膜电容 | 1nF~100μF | 无极性，稳定 | 耦合、定时 |

> **重要：电解电容有极性！** 接反会爆炸！长脚为正，短脚为负，外壳有标记。

## 六、电感

### 6.1 什么是电感？

电感就是一根**绕成线圈的导线**，利用电磁感应储存能量。

```
  符号：
  ──UUUU──
  
  结构：
  ┌────┐
  │线圈│ ← 导线绕在磁芯上
  └────┘
```

### 6.2 电感的特性

**阻碍电流变化：**
- 电流增大时，电感产生反向电压阻碍增大
- 电流减小时，电感产生正向电压维持电流

$$V = L \times \frac{dI}{dt}$$

- L：电感，单位亨利(H)
- dI/dt：电流变化率

> **类比**：电感像**惯性**大的飞轮——速度（电流）不能突变，需要时间加速减速。

### 6.3 电感的应用

- 滤波：阻碍高频信号通过
- 储能：开关电源中的储能元件
- 振荡：与电容组成LC振荡电路

## 八、万用表的使用

### 8.1 测量电压（并联）

```
测量方法：万用表并联在被测元件两端

  ┌──── R ────┐
  │           │
  └── 万用表 ──┘
  
量程选择：先选大量程，再逐步减小
```

### 8.2 测量电流（串联）

```
测量方法：断开电路，万用表串联接入

  ── 万用表 ── R ──
  
注意：电流档内阻很小，不能并联在电源上！会短路！
```

### 8.3 测量电阻

```
测量方法：
1. 断电！（不能在线测量）
2. 万用表接在电阻两端
3. 读数

档位选择：
- 200Ω档：测0-200Ω
- 2kΩ档：测0-2kΩ
- 20kΩ档：测0-20kΩ
- 以此类推
```

## 九、焊接基础

### 9.1 焊接工具

| 工具 | 用途 |
|------|------|
| 电烙铁 | 熔化焊锡（常用30-40W） |
| 焊锡丝 | 含松香芯的焊锡（0.8-1.0mm） |
| 松香/焊膏 | 助焊剂，帮助焊锡流动 |
| 吸锡器 | 吸走多余焊锡 |
| 镊子 | 夹持小元件 |

### 9.2 焊接五步法

```
1. 准备：烙铁头镀锡，清洁
2. 加热：烙铁头同时接触焊盘和引脚
3. 送锡：焊锡丝送到接触点（不是烙铁上）
4. 移开：先移走焊锡丝，再移走烙铁
5. 检查：焊点应光滑、明亮、呈圆锥形
```

### 9.3 焊接注意事项

```
好焊点：           坏焊点：
   ╱╲              ╱╲
  ╱  ╲            ╱  ╲
 ╱ 光 ╲          ╱暗淡╲ ← 虚焊（接触不良）
╱______╲        ╱______╲
  明亮            灰色
  
时间不能太长！每个焊点2-3秒
否则会：
- 烫坏元件
- 损坏焊盘
- 焊锡氧化
```

## 十、常见问题

**Q: 为什么电流从正极流到负极？**
A: 这是历史约定。实际上电子是从负极流向正极的（因为电子带负电，被正极吸引）。但电路分析中我们使用"传统电流方向"（正→负）。

**Q: 电压和电位有什么区别？**
A: 电压是两点之间的电位差。电位是某点相对于参考点（通常是地）的电压。就像高度是相对于海平面，而海拔差是两点高度之差。

**Q: 什么是"地"(GND)？**
A: 地是电路的参考零电位点。在单片机系统中，地通常接电源的负极。"接地"就是接到这个参考点。

**Q: 为什么有些电路要加"去耦电容"？**
A: 电源线上会有高频噪声，去耦电容能把这些噪声旁路到地。通常在芯片的VCC和GND之间并联0.1μF陶瓷电容。
