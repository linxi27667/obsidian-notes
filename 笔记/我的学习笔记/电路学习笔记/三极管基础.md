# 三极管基础

## 一、什么是三极管？

三极管是电子电路中最重要的元件之一，它有**放大**和**开关**两大功能。

> **类比理解：水龙头**
> - 水管里的水流 = 集电极到发射极的大电流
> - 你拧水龙头的力 = 基极的小电流
> - 用很小的力（基极电流）就能控制很大的水流（集电极电流）

```
三极管符号（NPN型）：

        集电极 C（Collector）
          │
          │
     ─────┤
     B    │  ← 发射极箭头方向
    基极──┤     表示电流方向
     ─────┤
          │
          ▼
        发射极 E（Emitter）
```

## 二、三极管的结构

### 2.1 两种类型：NPN 和 PNP

三极管由三层半导体组成，有两种排列方式：

```
NPN型：                    PNP型：
  N型（发射区）              P型（发射区）
    │                          │
  P型（基区）                N型（基区）
    │                          │
  N型（集电区）              P型（集电区）

  发射极箭头：向外           发射极箭头：向内
  （电流从C流向E）          （电流从E流向C）
```

**NPN更常用**，我们以NPN为主讲解。

### 2.2 三个极的作用

| 电极 | 英文 | 作用 |
|------|------|------|
| 基极 B | Base | 控制端，输入小电流控制大电流 |
| 集电极 C | Collector | 收集载流子，输出大电流 |
| 发射极 E | Emitter | 发射载流子，大电流流出端 |

### 2.3 内部结构详解

```
        C（集电极）
        │
  ┌─────┴─────┐
  │  N型集电区  │  ← 掺杂浓度低，面积大
  ├───────────┤
  │  P型基区   │  ← 很薄（几微米），掺杂浓度中等
  ├───────────┤
  │  N型发射区  │  ← 掺杂浓度高，面积小
  └─────┬─────┘
        │
        E（发射极）
```

**关键设计特点：**
1. **发射区掺杂浓度最高** → 容易发射电子
2. **基区很薄**（几微米）→ 大部分电子能穿过
3. **集电区面积最大** → 收集电子

## 三、三极管的工作原理

### 3.1 电流分配的魔法

三极管的核心秘密：**发射极发出的电子，大部分（>95%）被集电极收集，只有很少一部分（<5%）从基极流出。**

```
          Ic（集电极电流，大）
          ↑
          │
     ┌────┴────┐
     │  N      │
     │   ↗ ↗  │ ← 95%以上的电子被集电极收集
     ├─────────┤
     │  P      │ ← 基区很薄
     │  ↑↑↑   │ ← 只有少量电子从基极流出
     ├─────────┤
     │  N+     │ ← 发射区，高掺杂
     │  ↑↑↑↑↑ │ ← 大量电子从发射极注入
     └────┬────┘
          │
          ↑
          Ie（发射极电流，最大）
```

**电流关系：**
```
Ie = Ib + Ic

其中：Ic >> Ib（集电极电流远大于基极电流）
```

### 3.2 放大系数 β（hFE）

$$\beta = \frac{I_c}{I_b}$$

- β 通常在 **100~300** 之间
- 意味着：1mA的基极电流，可以控制100-300mA的集电极电流
- 这就是三极管的**放大作用**！

> **举例**：β = 100
> - 基极注入 10μA 电流
> - 集电极就有 1mA 电流
> - 信号被放大了100倍！

## 四、三极管的三种工作状态

### 4.1 截止状态（开关断开）

**条件**：基极没有正向偏置（Vbe < 0.5V）

```
  等效电路：
  C ──┤├── E    （断开，无电流）
  B：无信号
```

- 三极管完全关断
- 集电极电流 ≈ 0
- 相当于开关断开

### 4.2 放大状态（线性放大）

**条件**：发射结正偏，集电结反偏

```
  对于NPN管：
  Vbe ≈ 0.7V（发射结正偏）
  Vce > 0.3V（集电结反偏）
```

- Ic = β × Ib
- 输出电流与输入电流成正比
- 用于**模拟信号放大**（音频放大、信号处理）

```
  输入信号（小）     三极管     输出信号（大）
      ∿∿∿∿     →    放大    →     ∿∿∿∿
    （mV级）                    （V级）
```

### 4.3 饱和状态（开关闭合）

**条件**：发射结和集电结都正偏

```
  对于NPN管：
  Vbe ≈ 0.7V
  Vce ≈ 0.1-0.3V（饱和压降）
```

- Ib足够大，使Ic达到最大值
- 此时 Ic 不再等于 β × Ib（受外部电路限制）
- Vce很小，相当于开关闭合
- 用于**开关电路**（驱动继电器、LED、电机）

```
  三种状态对比：
  
  截止      放大        饱和
  C-E断开   C-E有电流    C-E接近短路
  Ib=0      Ic=β·Ib     Ic达到最大
  Vce=VCC   Vce适中      Vce≈0.2V
  
  ◇─────────◇─────────◇
  关         线性区      开
```

## 五、三极管的关键参数

| 参数 | 符号 | 含义 | 典型值 |
|------|------|------|--------|
| 电流放大倍数 | β / hFE | Ic/Ib | 100~300 |
| 发射结导通电压 | Vbe(on) | 导通时BE间电压 | 0.6-0.7V |
| 饱和压降 | Vce(sat) | 饱和时CE间电压 | 0.1-0.3V |
| 最大集电极电流 | Icm | 能承受的最大Ic | 视型号而定 |
| 最大功耗 | Pcm | 能承受的最大功率 | 视型号而定 |
| 特征频率 | fT | 放大倍数降为1的频率 | 视型号而定 |

### 功耗计算
```
P = Vce × Ic

例：Vce = 5V, Ic = 100mA
P = 5 × 0.1 = 0.5W

注意不要超过Pcm！否则三极管会烧毁
```

## 七、三极管的典型应用电路

### 7.1 LED驱动电路（开关模式）

用小信号控制LED亮灭：

```
         VCC (5V)
          │
          R1 (限流)
          │
         LED
          │
          C
  信号 ──R2── B ──┤  三极管 (NPN)
          │      │
         GND     E
                 │
                GND

工作原理：
- 信号为高电平 → Ib流过 → 三极管饱和 → LED亮
- 信号为低电平 → Ib=0 → 三极管截止 → LED灭
```

**R2计算：**
```
假设 VCC=5V, Vbe=0.7V, 需要Ic=20mA, β=100
Ib = Ic/β = 20mA/100 = 0.2mA（取0.5mA留余量）
R2 = (Vsignal - Vbe) / Ib = (5 - 0.7) / 0.5mA = 8.6kΩ
取标准值 8.2kΩ 或 10kΩ
```

**R1计算：**
```
R1 = (VCC - Vled - Vce(sat)) / Ic
   = (5 - 2 - 0.2) / 20mA = 140Ω
取标准值 150Ω
```

### 7.2 继电器驱动

```
         VCC (12V)
          │
        ┌─┴─┐
        │继电器│
        │线圈 │
        └─┬─┘
          │
          C
  信号 ──R── B ──┤  三极管
          │      │
         GND     E
                 │
                GND

        D ◁|  ← 续流二极管（保护！）
       （并联在继电器两端）
```

> **注意：** 继电器线圈断电时会产生高压反向电动势，必须并联续流二极管保护三极管！

### 7.3 基本共射放大电路

这是最经典的**模拟放大**电路：

```
         VCC
          │
          Rc
          │
          ├──── 输出 (Vo)
          │
  输入 ──C1──┤
  (Vi)      │
           Rb ── B ──┤  三极管
           │         │
          GND        E
                     │
                     Re
                     │
                    GND
                     │
                     Ce ← 旁路电容
```

**工作点设置（直流偏置）：**
```
Ib = (VCC - Vbe) / Rb
Ic = β × Ib
Vce = VCC - Ic × Rc - Ic × Re
```

**电压放大倍数：**
```
Av = -Rc / re    （re = 26mV/Ic，内阻很小）

例：Rc = 4.7kΩ, Ic = 1mA
re = 26mV / 1mA = 26Ω
Av = -4700 / 26 ≈ -180倍

负号表示输出与输入反相！
```

### 7.4 射极跟随器（共集电极电路）

输出从发射极取出，特点是**电压增益≈1，但电流增益大**：

```
         VCC
          │
          R
          │
          ├──── B
          │
  输入 ──┤  三极管
  (Vi)    │
          ├─── 输出 (Vo) ≈ Vi
          │
          Re
          │
         GND
```

**特点：**
- 电压增益 ≈ 1（不放大电压）
- 输入阻抗高（不吸取信号源电流）
- 输出阻抗低（能驱动重负载）
- 用途：**阻抗匹配、缓冲、驱动**

## 八、三极管作为开关的设计要点

### 8.1 确保深度饱和

为了确保三极管可靠导通，设计时让Ib比理论值大2-5倍：

```
Ib_design = (2~5) × Ic / β

这样即使β变化，也能保证饱和
```

### 8.2 开关速度

三极管从截止到饱和（或反过来）需要时间：
- **导通时间 ton**：几ns到几μs
- **关断时间 toff**：通常比导通慢
- 高频开关考虑用**MOSFET**替代

### 8.3 功耗计算

```
截止时：P = Vce(off) × Ic ≈ 0（电流≈0）
饱和时：P = Vce(sat) × Ic ≈ 0（电压≈0）
放大区：P = Vce × Ic （最大！）

开关应用中，三极管大部分时间在截止或饱和，功耗很小
过渡区（放大区）停留时间很短
```

## 九、达林顿管——超级放大

两个三极管级联，β相乘：

```
  ┌─────────────────────┐
  │     C               │
  │     │               │
  │  ┌──┴──┐  ┌──┴──┐  │
  │  │ Q1  │  │ Q2  │  │
  │  │  B──┤  │  B──┤  │
  │  └──┬──┘  └──┬──┘  │
  │     │        │     │
  │     E───C    E     │
  │     ↑        │     │
  │  Q1的E       │     │
  │  接Q2的C     │     │
  └─────┼────────┼─────┘
        B        E
   （输入）    （输出）
```

**特点：**
- 总 β = β1 × β2（可达10000以上！）
- Vbe = 1.2-1.4V（两个管的Vbe相加）
- 开关速度较慢
- 代表型号：TIP120/TIP122（NPN）、TIP125/TIP127（PNP）

## 十、NPN vs PNP 对比

| 特性 | NPN | PNP |
|------|-----|-----|
| 电流方向 | C→E | E→C |
| 导通条件 | B极电压高于E极约0.7V | B极电压低于E极约0.7V |
| 常用程度 | 更常用 | 较少用 |
| 适用场景 | 低边开关 | 高边开关 |
| 电源极性 | 正电源 | 负电源或接地 |

### 高边 vs 低边开关

```
高边开关（用PNP）：          低边开关（用NPN）：
   VCC                         VCC
    │                           │
   PNP                          负载
    │                           │
   负载                         NPN
    │                           │
   GND                         GND

高边：负载一端接地            低边：负载一端接VCC
控制VCC是否给负载供电         控制负载是否接地
```

## 十一、三极管 vs MOSFET

| 特性 | 三极管(BJT) | MOSFET |
|------|------------|--------|
| 控制方式 | 电流控制（需要Ib） | 电压控制（栅极电压） |
| 输入阻抗 | 较低 | 极高（几乎不取电流） |
| 开关速度 | 较慢 | 很快 |
| 导通压降 | Vce(sat) ≈ 0.2V | Rds(on) × Id（很小） |
| 放大区应用 | 更线性 | 跨导放大 |
| 适用场景 | 模拟放大、小功率开关 | 大功率开关、高频 |

## 十二、常见问题

**Q: 三极管和二极管有什么关系？**
A: 三极管可以看成两个背靠背的二极管（BE结和BC结），但因为基区很薄，两个结会相互影响，产生放大作用。单独的两个二极管接在一起是没有放大作用的。

**Q: 为什么三极管能放大信号？能量从哪来？**
A: 三极管不是凭空创造能量！它用小电流（基极）控制大电流（集电极），能量来自电源（VCC）。就像水龙头用小力气控制大水流，能量来自水泵。

**Q: β值为什么会变化？**
A: β受温度、Ic大小、Vce大小影响。温度升高β增大；Ic太大或太小β都会下降。所以设计电路时要留余量。

**Q: 如何判断三极管的三个极？**
A: 用万用表二极管档：
1. 找基极：红表笔接某脚，黑表笔分别接另外两脚，都有0.6-0.7V → 红表笔是基极，管子是NPN
2. 判断C和E：β通常大于β反，据此区分

## 十三、三极管损坏的常见原因

| 原因 | 现象 | 预防措施 |
|------|------|---------|
| 过流 | Ic超过Icm | 串联限流电阻 |
| 过压 | Vce超过Vceo | 选择足够耐压的管子 |
| 过热 | 功耗超过Pcm | 加散热片，计算功耗 |
| 基极过流 | Ib太大烧毁BE结 | 串联基极限流电阻 |
| 反接电源 | 发射结反向击穿 | 注意电源极性 |
