# 场效应管（MOSFET）基础

## 一、什么是MOSFET？

MOSFET = **M**etal **O**xide **S**emiconductor **F**ield **E**ffect **T**ransistor
金属-氧化物-半导体场效应晶体管

> **类比理解：**
> - 三极管像是**水龙头**——需要你持续用力拧（持续的基极电流）
> - MOSFET像是**阀门手柄**——拨过去就行，不需要持续用力（电压控制，栅极几乎不取电流）

---

## 二、MOSFET的结构

### 2.1 两种类型：N沟道 和 P沟道

```
N沟道 MOSFET（更常用）：

  G（栅极 Gate）
  │
  ▼
┌─────────────┐
│  金属栅极    │ ← Gate
├─────────────┤
│  氧化物绝缘层 │ ← SiO₂（很薄）
├─────────────┤
│  P型衬底     │
│  ┌───┐ ┌───┐│
│  │N+ │ │N+ ││ ← 源极和漏极的N+区
│  └─┬─┘ └─┬─┘│
└────┼─────┼──┘
     │     │
     S     D
   源极   漏极
 (Source) (Drain)
```

### 2.2 三个电极

| 电极 | 英文 | 对应三极管 |
|------|------|----------|
| 栅极 G | Gate | 基极 B |
| 漏极 D | Drain | 集电极 C |
| 源极 S | Source | 发射极 E |

## 三、MOSFET的工作原理

### 3.1 核心思想：电场控制

```
断开状态（Vgs = 0）：
  G
  │
  ▼
┌─────────────┐
│    栅极      │
├─────────────┤
│  绝缘层      │
├─────────────┤
│  P型衬底     │
│  N+    N+   │ ← 没有导电沟道
└─────────────┘
  S        D
  （S和D之间不导通）

导通状态（Vgs > Vth）：
  G（加正电压）
  │
  ▼
┌─────────────┐
│    栅极      │
├─────────────┤
│  绝缘层      │
├═════════════┤ ← 感应出N型导电沟道！
│  P型衬底     │
│  N+    N+   │
└─────────────┘
  S        D
  （S和D之间导通）
```

**原理：**
1. 栅极加正电压（对N沟道）
2. 电场穿过绝缘层，吸引P型衬底中的电子到表面
3. 在S和D之间形成一层N型导电沟道
4. 电子可以从S流向D

### 3.2 阈值电压 Vth

- 栅极电压必须超过**阈值电压Vth**才能开启
- 常见Vth：1-4V（逻辑电平MOSFET约1-2V）

```
  Id（漏极电流）
    ↑
    │           ┌───── 线性区
    │          ╱
    │         ╱
    │        ╱  ← 饱和区
    │       ╱
    │──────╱
    │─────╱ Vgs = Vth
    │
    └──────────────→ Vds
```

## 四、MOSFET的三种工作区

### 4.1 截止区（Vgs < Vth）
- 没有导电沟道
- Id ≈ 0
- 相当于开关断开

### 4.2 线性区/可变电阻区（Vgs > Vth, Vds很小）
- 沟道形成，像一个电阻
- Id 随 Vds 线性增加
- **用作开关时工作在这个区域**（导通电阻很小）

```
导通电阻 Rds(on)：
- 典型值：几mΩ到几百mΩ
- 越小越好（导通损耗 = Id² × Rds(on)）
```

### 4.3 饱和区（Vgs > Vth, Vds较大）
- 电流基本恒定
- **用作放大时工作在这个区域**
- Id ≈ K × (Vgs - Vth)²

## 五、MOSFET的关键参数

| 参数 | 符号 | 含义 | 设计意义 |
|------|------|------|---------|
| 阈值电压 | Vth | 开启电压 | 驱动电压必须高于此值 |
| 导通电阻 | Rds(on) | 导通时D-S电阻 | 越小损耗越低 |
| 最大漏极电流 | Id | 能承受的最大电流 | 选型时要大于实际电流 |
| 最大Vds | Vds(max) | D-S最大耐压 | 选型时要大于电源电压 |
| 栅极电荷 | Qg | 栅极需要的总电荷 | 影响开关速度 |
| 最大功耗 | Pd | 能承受的最大功率 | 散热设计依据 |

## 六、N沟道 vs P沟道

| 特性 | N沟道 | P沟道 |
|------|-------|-------|
| 载流子 | 电子 | 空穴 |
| 导通条件 | Vgs > Vth（正电压） | Vgs < -|Vth|（负电压） |
| 导通电阻 | 较小（电子迁移率高） | 较大（约2-3倍于N沟道） |
| 常用程度 | 更常用 | 较少用 |
| 适用场景 | 低边开关 | 高边开关 |

---

## 七、增强型 vs 耗尽型

| 类型 | 默认状态 | 开启条件 | 符号区别 |
|------|---------|---------|---------|
| 增强型 | 断开 | 加电压才导通 | 虚线沟道 |
| 耗尽型 | 导通 | 加电压才断开 | 实线沟道 |

**增强型更常用**（我们日常说的MOSFET通常指增强型）

## 八、典型应用电路

### 8.1 LED驱动（低边开关）

```
         VCC (12V)
          │
          R (限流)
          │
         LED
          │
          D
  信号 ──G ──┤  N-MOSFET
          │
          S
          │
         GND

- 信号高 → Vgs > Vth → MOSFET导通 → LED亮
- 信号低 → Vgs < Vth → MOSFET截止 → LED灭
```

### 8.2 电机驱动（H桥的一部分）

```
         VCC
          │
     ┌────┤
     │    │
    Q1   Q3
   (P)   (P)
     │    │
     ├────┤──── 电机 ────┤
     │    │              │
    Q2   Q4              │
   (N)   (N)             │
     │    │              │
    GND  GND─────────────┘

正转：Q1和Q4导通
反转：Q3和Q2导通
刹车：Q2和Q4导通（短接电机两端）
停止：全部关断
```

### 8.3 电平转换电路

```
  3.3V逻辑 ──G ──┤  N-MOSFET
                  │
  5V ──R── D ────S─── 5V输出
                  │
                 R
                  │
                 GND

3.3V信号控制5V电路的电平转换
```

## 九、MOSFET的驱动

### 9.1 栅极电荷

MOSFET的栅极就像一个**电容**，需要充放电来开关：
- 开启：给栅极电容充电到Vgs > Vth
- 关断：栅极电容放电

```
  栅极电容模型：
  
  G ──┤├── S  （Ciss = Cgs + Cgd）
      │
      ├──┤├── D  （Cgd，米勒电容）
```

### 9.2 驱动电流

```
开关时间 ≈ Qg / Ig

Qg = 栅极总电荷
Ig = 驱动电流

例：Qg = 20nC, Ig = 1A
开关时间 ≈ 20ns
```

### 9.3 自举电容驱动（高端MOSFET）

高端N-MOSFET的源极电位会浮动，需要自举电路：

```
  VCC ──D1── Bootstrap Cap ── 栅极驱动
         │
         │
  高端MOSFET的源极
```

## 十、MOSFET的体二极管

所有MOSFET内部都有一个**寄生体二极管**：

```
        D
        │
   ┌────┴────┐
   │  N+     │
   │   │     │
   │  ┌┴─┐   │
   │  │体 │   │ ← 寄生二极管（从S到D方向）
   │  │二 │   │
   │  │极 │   │
   │  └┬─┘   │
   │   │     │
   │  P型    │
   │  N+     │
   └────┬────┘
        │
        S

对于N-MOSFET：体二极管方向是S→D
```

**注意：**
- 这个二极管在某些应用中是有用的（如电机驱动的续流）
- 在某些应用中是问题（可能造成不期望的导通）

## 十一、MOSFET vs 三极管 对比总结

| 特性 | MOSFET | 三极管(BJT) |
|------|--------|------------|
| 控制方式 | 电压控制 | 电流控制 |
| 输入阻抗 | 极高（10⁹Ω以上） | 较低（kΩ级） |
| 驱动功率 | 几乎为零 | 需要Ib |
| 开关速度 | 很快（ns级） | 较慢（μs级） |
| 导通特性 | Rds(on)恒定 | Vce(sat)≈0.2V |
| 大电流优势 | 导通损耗随I²增大 | 导通压降基本不变 |
| 并联使用 | 容易（自动均流） | 困难（需要均流电阻） |
| 静电敏感 | 很敏感（要防ESD） | 不太敏感 |
| 适用场景 | 开关电源、大功率开关 | 模拟放大、小功率 |

## 十三、防静电注意事项

MOSFET的栅极绝缘层很薄（几十nm），容易被静电击穿：

**防护措施：**
1. 焊接时使用防静电手环
2. 存放时用防静电袋
3. 栅极串联小电阻（10-100Ω）限制充放电电流
4. 栅源之间并联稳压管（保护栅极电压不过高）
5. 未使用时，将栅极与源极短接

## 十四、常见问题

**Q: MOSFET为什么叫"场效应"管？**
A: 因为它是靠电**场**（而不是电流）来控制导电沟道的形**效**应。

**Q: 栅极需要电流吗？**
A: 稳态时不需要（绝缘层隔开），但开关瞬间需要对栅极电容充放电，所以驱动电路要能提供瞬间大电流。

**Q: 为什么N沟道比P沟道常用？**
A: 电子迁移率比空穴高2-3倍，同样大小的管子N沟道的Rds(on)更小，损耗更低。

**Q: Rds(on)会随温度变化吗？**
A: 会！温度升高，Rds(on)增大（正温度系数）。这是好事——多个MOSFET并联时自动均流。
