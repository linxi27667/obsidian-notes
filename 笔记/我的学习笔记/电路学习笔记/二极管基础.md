# 二极管基础

## 一、什么是二极管？

二极管是最简单的半导体器件，你可以把它想象成一个**单向阀门**：
- 水（电流）只能从一个方向流过
- 反方向会被堵住

```
符号表示：
  阳极(A) ──▷|── 阴极(K)
  （正极）        （负极）
  电流方向：从A流向K
```

---

## 二、为什么会有二极管？——从材料说起

### 2.1 导体、绝缘体、半导体

| 类型 | 导电能力 | 举例 |
|------|---------|------|
| 导体 | 很容易导电 | 铜、铝、铁 |
| 绝缘体 | 几乎不导电 | 橡胶、玻璃、塑料 |
| 半导体 | 介于两者之间 | 硅(Si)、锗(Ge) |

**硅**是最重要的半导体材料，地球上含量极丰富（沙子的主要成分）。

### 2.2 纯净硅的结构

硅原子有4个外层电子。在晶体中，每个硅原子与相邻4个硅原子各共享1个电子，形成稳定的共价键结构。

```
    Si --- Si --- Si
    |      |      |
    Si --- Si --- Si
    |      |      |
    Si --- Si --- Si
    
每个Si与4个邻居各共享1个电子
```

纯净硅在常温下导电性很差，因为电子都被束缚在共价键中。

### 2.3 N型半导体——加入"多余电子"的硅

在纯净硅中掺入少量**五价元素**（如磷P、砷As）：
- 磷有5个外层电子
- 4个用来形成共价键，多出1个自由电子
- 这个电子很容易移动，形成导电

```
    Si --- P --- Si
    |     ||     |
    Si    ↑      Si
    （P多出的自由电子）
```

**N = Negative（负的）**，因为主要靠带负电的电子导电。

### 2.4 P型半导体——加入"空穴"的硅

在纯净硅中掺入少量**三价元素**（如硼B、铝Al）：
- 硼只有3个外层电子
- 与4个硅邻居结合时，少了一个电子，形成"空穴"
- 空穴可以看作"正电荷"，旁边的电子可以跳过来填补

```
    Si --- B --- Si
    |     ∘     |
    Si    ↑      Si
    （B留下的空穴，电子可以跳入）
```

**P = Positive（正的）**，因为主要靠空穴（相当于正电荷）导电。

> **类比理解**：
> - 电子 = 椅子上坐着的人
> - 空穴 = 空着的椅子
> - 人可以挪到空椅子上 = 电子跳入空穴
> - 整体效果：空穴在移动（相当于正电荷在移动）

## 三、PN结——二极管的核心

### 3.1 把P型和N型拼在一起

当P型半导体和N型半导体紧密接触时，在交界面发生奇妙的事情：

```
  P型半导体    |    N型半导体
               |
  ⊕ 空穴      |     ⊖ 自由电子
  ⊕ 空穴      |     ⊖ 自由电子
  ⊕ 空穴      |     ⊖ 自由电子
               |
        交界面（PN结）
```

### 3.2 扩散运动——"浓度高的往浓度低的地方跑"

就像墨水滴入清水会扩散一样：
- P区的空穴多，会向N区扩散
- N区的电子多，会向P区扩散

### 3.3 耗尽层的形成

扩散发生后：
1. P区的空穴跑到交界面附近，遇到N区的电子
2. 电子和空穴**复合**（互相填满）
3. 交界面附近：P侧留下负离子（失去空穴），N侧留下正离子（失去电子）
4. 这个区域叫**耗尽层**（或阻挡层），约几微米宽

```
  P区        耗尽层        N区
  ⊕ ⊕ ⊕    ⊖ ⊖ | ⊕ ⊕    ⊖ ⊖ ⊖
  ⊕ ⊕ ⊕    ⊖ ⊖ | ⊕ ⊕    ⊖ ⊖ ⊖
  ⊕ ⊕ ⊕    ⊖ ⊖ | ⊕ ⊕    ⊖ ⊖ ⊖
            ←──→
          内建电场方向（N→P）
```

### 3.4 内建电场——"自动门卫"

耗尽层产生了一个**内建电场**，方向从N指向P：
- 这个电场阻止进一步的扩散
- 当扩散力和电场力平衡时，达到稳定

## 四、二极管的两种工作状态

### 4.1 正向偏置（导通）

给二极管**正向电压**：正极接P，负极接N

```
     电池
    + │ -
    │     │
    │     │
  P型     N型
  ⊕ →    → ⊖
  ⊕ →    → ⊖
  
  外加电压方向：从P到N
```

**发生什么：**
1. 外加电压的正极把P区的空穴推向交界面
2. 外加电压的负极把N区的电子推向交界面
3. 耗尽层变窄
4. 当电压超过**门槛电压**（硅管约0.5-0.7V），耗尽层消失
5. 电流开始流动！

> **门槛电压**：
> - 硅二极管：约 0.7V
> - 锗二极管：约 0.3V
> - 发光二极管(LED)：1.8-3.3V（取决于颜色）

### 4.2 反向偏置（截止）

给二极管**反向电压**：正极接N，负极接P

```
     电池
    + │ -
    │     │
    │     │
  N型     P型
  ⊖ ←    ← ⊕
  ⊖ ←    ← ⊕
  
  外加电压方向：从N到P
```

**发生什么：**
1. 外加电压把P区的空穴拉走，N区的电子拉走
2. 耗尽层变宽
3. 电流被阻断（只有极微小的漏电流，nA级别）

### 4.3 反向击穿——"过头了"

如果反向电压太大（超过**击穿电压**），二极管会被强制导通：
- **齐纳击穿**：电场强到直接把电子从共价键中拉出来（可逆，用于稳压）
- **雪崩击穿**：电子被加速后撞出更多电子（像雪崩一样连锁反应）

> 普通二极管击穿通常是永久损坏！
> 但**稳压二极管（齐纳管）**就是利用这个特性来稳压的。

## 五、二极管的伏安特性曲线

```
  电流 I(mA)
    ↑
    │         正向导通区
    │           ╱
    │          ╱
    │         ╱  ← 正向电流急剧增加
    │        │
    │────────│───────────→ 电压 V(V)
  击穿← │   0│ 0.7
  电流  │    │
    │    │反向截止区（几乎为0）
    │    │
```

**关键参数：**
| 参数 | 含义 |
|------|------|
| 正向压降 VF | 导通时两端的电压（硅管约0.7V） |
| 反向漏电流 IR | 反向时的微小电流（越小越好） |
| 击穿电压 VBR | 反向击穿的临界电压 |
| 最大正向电流 IFM | 能承受的最大正向电流 |

## 六、常见二极管类型

### 6.1 普通整流二极管
- 用途：把交流电变成直流电（整流）
- 代表型号：1N4007（最常用）
- 特点：能承受大电流，速度不快

### 6.2 发光二极管（LED）
- 用途：照明、指示灯、显示屏
- 特点：通电会发光，不同材料发出不同颜色
- 正向压降较大：红光约1.8V，白光约3.0-3.3V

| 颜色 | 材料 | 正向压降 |
|------|------|---------|
| 红色 | GaAsP | 1.8-2.0V |
| 黄色 | GaAsP | 2.0-2.2V |
| 绿色 | GaP | 2.0-3.0V |
| 蓝色 | InGaN | 3.0-3.5V |
| 白光 | InGaN+荧光粉 | 3.0-3.3V |

> **LED必须串联限流电阻！**否则电流过大烧毁。

### 6.3 稳压二极管（齐纳管）
- 用途：稳定电压
- 工作原理：利用反向击穿特性
- 接法：**反向接入电路**（和普通二极管相反！）

```
稳压管接法：
       V_in
        │
        R（限流）
        │
        ├──── V_out（稳定电压）
        │
      ┌─┴─┐
      │ ◁ │ 稳压管（反向）
      └─┬─┘
        │
       GND
```

### 6.4 肖特基二极管
- 用途：高频整流、开关电源
- 特点：正向压降低（约0.3V）、开关速度快
- 原理：金属-半导体结，不是PN结

### 6.5 光电二极管
- 用途：光检测、光通信
- 工作原理：光照产生电子-空穴对，产生电流
- 接法：反向偏置，光照越强电流越大

## 七、二极管的典型应用电路

### 7.1 半波整流

把交流电的负半周切掉，只保留正半周：

```
  输入（交流）：~~正~~负~~正~~负~~
  
       D
  AC ──▷|── +DC
                    ┌──┐
  输出（脉动直流）：│  │  │  │
                   ┘  └──┘  └──
```

### 7.2 全波整流（桥式整流）

用4个二极管，把负半周也翻转成正的：

```
        D1      D2
  AC1 ──▷|──┬──▷|──┬── +DC
             │      │
             │      │
  AC2 ──▷|──┴──▷|──┴── -DC
        D4      D3
        
  输出：~~正~~正~~正~~正~~  （全部为正）
```

### 7.3 续流二极管（保护用）

电感断电时会产生反向高压，二极管提供泄放通路：

```
  VCC
   │
   R
   │
   ├── 继电器线圈 ──┐
   │                │
   │    D ◁|       │
   │    （续流）     │
   │                │
  GND ──────────────┘
  
  断电时，线圈的感应电流通过D释放，保护开关管
```

### 7.4 二极管钳位

把信号的电压限制在一定范围内：

```
        D1（向上钳位到VCC）
  信号 ──┤──▷|── VCC
        │
        D2（向下钳位到GND）
        └──|◁── GND
```

## 八、如何用万用表测试二极管？

### 方法一：二极管档

1. 万用表拨到二极管档（符号是▷|）
2. 红表笔接阳极，黑表笔接阴极
3. 正常显示：0.5-0.8V（硅管）
4. 反接：显示OL（超量程）= 不通

### 方法二：电阻档

1. 万用表拨到电阻档
2. 正向测量：几百Ω到几kΩ
3. 反向测量：无穷大（OL）

> **判断好坏**：正向导通、反向截止 = 好的
> 两个方向都通 = 击穿损坏
> 两个方向都不通 = 开路损坏

## 十、常见问题

**Q: 为什么二极管只能单向导通？**
A: 因为PN结的耗尽层在正向电压下变窄消失，反向电压下变宽阻断。

**Q: LED为什么有颜色？**
A: 不同半导体材料的能带间隙不同，电子跃迁时释放的光子能量（波长/颜色）不同。

**Q: 二极管导通后电压会变吗？**
A: 基本不变。硅管导通后，无论电流怎么变，压降都在0.6-0.7V左右（这就是为什么叫"恒压降模型"）。
