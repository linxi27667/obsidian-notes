# FreeRTOS 队列 （任务间安全的数据传输机制，支持阻塞读写）

> 创建时间: 2026-05-06

## 核心概念

### 队列是什么
- ==FIFO数据缓冲区== - 数据按先进先出顺序传递
- ==任务间通信的主要方式== - 比全局变量更安全，自带同步机制
- ==支持阻塞读写== - 任务可阻塞等待数据或等待空间

### 队列特点
- ==值拷贝机制== - 数据入队时复制，不共享内存
- ==多任务安全== - 多任务同时读写同一队列无需额外锁
- ==可中断安全== - 提供ISR专用API（后缀`FromISR`）
- ==支持多接收者== - 多个任务读同一队列时，最高优先级任务先获取数据

## 一、队列基础

### 1.1 创建队列

```c
QueueHandle_t xQueueCreate(
    UBaseType_t uxQueueLength,
    UBaseType_t uxItemSize
);
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `uxQueueLength` | `UBaseType_t` | 队列容量（最多可容纳元素个数） |
| `uxItemSize` | `UBaseType_t` | 每个元素的大小（字节） |

**返回值**: `QueueHandle_t` — 成功返回队列句柄，失败返回`NULL`

**示例**:
```c
// 创建可存放10个uint32_t的队列
QueueHandle_t xQueue = xQueueCreate(10, sizeof(uint32_t));

// 创建可存放5个结构体的队列
typedef struct {
    uint32_t id;
    float value;
} SensorData_t;

QueueHandle_t xSensorQueue = xQueueCreate(5, sizeof(SensorData_t));
```

### 1.2 发送数据到队列

| 函数 | 行为 |
|------|------|
| `xQueueSendToBack()` | 发送到队尾（入队） |
| `xQueueSendToFront()` | 发送到队头（优先级消息） |
| `xQueueSend()` | 同 `xQueueSendToBack()` |
| `xQueueOverwrite()` | 队列满时覆盖最新数据 |

```c
BaseType_t xQueueSend(
    QueueHandle_t xQueue,
    const void *pvItemToQueue,
    TickType_t xTicksToWait
);
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `xQueue` | `QueueHandle_t` | 目标队列句柄 |
| `pvItemToQueue` | `const void *` | 指向要发送数据的指针 |
| `xTicksToWait` | `TickType_t` | 阻塞等待时间（`portMAX_DELAY` = 无限等待） |

**返回值**: `BaseType_t` — `pdPASS`表示发送成功，`errQUEUE_FULL`表示队列已满且超时

**示例**:
```c
uint32_t data = 42;

// 立即发送，队列满时返回失败
if (xQueueSend(xQueue, &data, 0) == pdPASS) {
    // 发送成功
}

// 阻塞等待最多100个tick
if (xQueueSend(xQueue, &data, pdMS_TO_TICKS(100)) == pdPASS) {
    // 发送成功
}

// 无限等待
xQueueSend(xQueue, &data, portMAX_DELAY);
```

### 1.3 从队列接收数据

```c
BaseType_t xQueueReceive(
    QueueHandle_t xQueue,
    void * const pvBuffer,
    TickType_t xTicksToWait
);
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `xQueue` | `QueueHandle_t` | 源队列句柄 |
| `pvBuffer` | `void *` | 接收数据的缓冲区指针 |
| `xTicksToWait` | `TickType_t` | 阻塞等待时间（队列为空时等待的tick数） |

**返回值**: `BaseType_t` — `pdPASS`表示接收到数据，`pdFAIL`表示队列空且超时

**示例**:
```c
uint32_t received_data;

// 阻塞等待直到有数据
if (xQueueReceive(xQueue, &received_data, portMAX_DELAY) == pdPASS) {
    // 接收到数据：received_data
}

// 非阻塞读取（队列空时立即返回）
xQueueReceive(xQueue, &received_data, 0);
```

## 二、中断安全的队列操作

### 2.1 ISR发送

```c
BaseType_t xQueueSendFromISR(
    QueueHandle_t xQueue,
    const void *pvItemToQueue,
    BaseType_t * const pxHigherPriorityTaskWoken  // 是否有高优先级任务被唤醒
);
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `xQueue` | `QueueHandle_t` | 目标队列句柄 |
| `pvItemToQueue` | `const void *` | 指向要发送数据的指针 |
| `pxHigherPriorityTaskWoken` | `BaseType_t *` | 输出参数，若因发送导致高优先级任务就绪则设为`pdTRUE` |

**返回值**: `BaseType_t` — `pdPASS`表示发送成功，`errQUEUE_FULL`表示队列已满

**示例**:
```c
void EXTI0_IRQHandler(void)
{
    BaseType_t xHigherPriorityTaskWoken = pdFALSE;
    uint32_t event = GPIO_EVENT;

    xQueueSendFromISR(xEventQueue, &event, &xHigherPriorityTaskWoken);

    // 如果有更高优先级任务被唤醒，触发上下文切换
    portYIELD_FROM_ISR(xHigherPriorityTaskWoken);
}
```

### 2.2 ISR接收

```c
BaseType_t xQueueReceiveFromISR(
    QueueHandle_t xQueue,
    void * const pvBuffer,
    BaseType_t * const pxHigherPriorityTaskWoken
);
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `xQueue` | `QueueHandle_t` | 源队列句柄 |
| `pvBuffer` | `void *` | 接收数据的缓冲区指针 |
| `pxHigherPriorityTaskWoken` | `BaseType_t *` | 输出参数，若因接收导致高优先级任务就绪则设为`pdTRUE` |

**返回值**: `BaseType_t` — `pdPASS`表示接收到数据，`pdFAIL`表示队列为空

## 三、完整示例

### 示例1：生产者-消费者模型

```c
static QueueHandle_t xDataQueue;

// 生产者任务
void vProducerTask(void *pvParameters)
{
    uint32_t data = 0;
    while (1) {
        data = read_sensor();
        // 发送到队列，阻塞等待
        xQueueSend(xDataQueue, &data, portMAX_DELAY);
        vTaskDelay(pdMS_TO_TICKS(100));
    }
}

// 消费者任务
void vConsumerTask(void *pvParameters)
{
    uint32_t received;
    while (1) {
        // 阻塞等待数据
        if (xQueueReceive(xDataQueue, &received, portMAX_DELAY) == pdPASS) {
            process_data(received);
        }
    }
}

void main(void)
{
    xDataQueue = xQueueCreate(10, sizeof(uint32_t));

    xTaskCreate(vProducerTask, "Producer", 256, NULL, 1, NULL);
    xTaskCreate(vConsumerTask, "Consumer", 256, NULL, 2, NULL);

    vTaskStartScheduler();
}
```

### 示例2：中断驱动的事件处理

```c
static QueueHandle_t xEventQueue;

// 中断服务例程
void TIM2_IRQHandler(void)
{
    BaseType_t xHigherPriorityTaskWoken = pdFALSE;
    const uint32_t event = TIMER_EXPIRED;

    if (TIM_GetITStatus(TIM2, TIM_IT_Update) != RESET) {
        xQueueSendFromISR(xEventQueue, &event, &xHigherPriorityTaskWoken);
        TIM_ClearITPendingBit(TIM2, TIM_IT_Update);
    }

    portYIELD_FROM_ISR(xHigherPriorityTaskWoken);
}

// 事件处理任务（高优先级）
void vEventTask(void *pvParameters)
{
    uint32_t event;
    while (1) {
        xQueueReceive(xEventQueue, &event, portMAX_DELAY);
        switch (event) {
            case TIMER_EXPIRED:
                handle_timer_event();
                break;
            // 其他事件...
        }
    }
}
```

## 附录：队列 API 速查表

### 队列创建与删除
| 函数                    | 说明     |
| --------------------- | ------ |
| `xQueueCreate()`      | 创建队列   |
| `vQueueDelete()`      | 删除队列   |
| `xQueueCreateMutex()` | 创建互斥队列 |

### 发送操作
| 函数 | 说明 |
|------|------|
| `xQueueSend()` / `xQueueSendToBack()` | 发送到队尾 |
| `xQueueSendToFront()` | 发送到队头 |
| `xQueueOverwrite()` | 覆盖最新数据 |
| `xQueueSendFromISR()` | 中断中发送 |
| `xQueueSendToFrontFromISR()` | 中断中发送到队头 |

### 接收操作
| 函数 | 说明 |
|------|------|
| `xQueueReceive()` | 接收并移除数据 |
| `xQueuePeek()` | 查看但不移除数据 |
| `xQueueReceiveFromISR()` | 中断中接收 |

### 查询操作
| 函数 | 说明 |
|------|------|
| `uxQueueMessagesWaiting()` | 获取队列消息数 |
| `uxQueueSpacesAvailable()` | 获取队列剩余空间 |
| `xQueueIsQueueEmpty()` | 判断队列是否为空 |
| `xQueueIsQueueFull()` | 判断队列是否已满 |
