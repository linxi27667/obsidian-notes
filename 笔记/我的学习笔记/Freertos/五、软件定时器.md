# FreeRTOS 软件定时器 （提供一次性和周期性定时回调功能）

> 创建时间: 2026-05-06

## 核心概念

### 定时器类型
- ==一次性定时器== - 到期触发一次后停止
- ==周期性定时器== - 到期后自动重新加载，重复触发

### 架构特点
- ==RTOS守护任务管理== - 所有定时器由专用的Daemon/Timer任务统一管理
- ==回调函数执行== - 定时器到期时执行回调函数（在Daemon任务上下文中）
- ==回调应尽量简短== - 回调函数不能阻塞，不宜执行耗时操作

## 一、定时器基础

### 1.1 创建定时器

```c
TimerHandle_t xTimerCreate(
    const char * const pcTimerName,
    TickType_t xTimerPeriodInTicks,
    UBaseType_t uxAutoReload,
    void * const pvTimerID,
    TimerCallbackFunction_t pxCallbackFunction
);
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `pcTimerName` | `const char *` | 定时器名称（调试用） |
| `xTimerPeriodInTicks` | `TickType_t` | 周期（tick数，可用`pdMS_TO_TICKS()`转换） |
| `uxAutoReload` | `UBaseType_t` | `pdTRUE`=周期性定时器，`pdFALSE`=一次性定时器 |
| `pvTimerID` | `void *` | 用户自定义ID（回调中通过`pvTimerGetTimerID()`获取） |
| `pxCallbackFunction` | `TimerCallbackFunction_t` | 定时器到期时的回调函数指针 |

**返回值**: `TimerHandle_t` — 成功返回定时器句柄，失败返回`NULL`

### 1.2 定时器命令

| 命令 | 说明 |
|------|------|
| `xTimerStart()` | 启动定时器 |
| `xTimerStop()` | 停止定时器 |
| `xTimerReset()` | 重置定时器（重新开始计时） |
| `xTimerChangePeriod()` | 修改定时器周期 |
| `xTimerDelete()` | 删除定时器 |

**所有命令都支持中断安全版本（后缀`FromISR`）**

### 1.3 回调函数签名

```c
void vTimerCallback(TimerHandle_t xTimer);
```

```c
void vTimerCallback(TimerHandle_t xTimer)
{
    // 获取用户ID
    void *id = pvTimerGetTimerID(xTimer);

    // 处理定时器事件
    handle_timer_event((uint32_t)id);

    // 注意：此处不可调用可能阻塞的API
    // 不可调用 vTaskDelay()、xQueueSend(..., portMAX_DELAY) 等
}
```

## 二、定时器操作详解

### 2.1 启动定时器

```c
BaseType_t xTimerStart(
    TimerHandle_t xTimer,
    TickType_t xTicksToWait
);
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `xTimer` | `TimerHandle_t` | 定时器句柄 |
| `xTicksToWait` | `TickType_t` | 发送命令给守护任务的超时时间（tick数） |

**返回值**: `BaseType_t` — `pdPASS`表示命令成功发送，`pdFAIL`表示命令队列满或无效

**注意**: `xTimerStart()` 不是立即启动，而是发送命令给Timer任务，由Timer任务在下一个tick处理。

### 2.2 重置定时器

```c
// 重置：从当前时刻重新开始计时
xTimerReset(xTimer, 0);
```

**应用场景**：软件看门狗、超时重传

```c
// 看门狗定时器
void vWatchdogTimerCallback(TimerHandle_t xTimer)
{
    // 超时！说明主任务未能及时重置
    system_error_handler();
}

void vMainThread(void *pvParameters)
{
    TimerHandle_t xWatchdog = xTimerCreate(
        "Watchdog",
        pdMS_TO_TICKS(5000),   // 5秒超时
        pdTRUE,                 // 周期性
        NULL,
        vWatchdogTimerCallback
    );
    xTimerStart(xWatchdog, 0);

    while (1) {
        do_main_work();
        // 每次循环重置看门狗
        xTimerReset(xWatchdog, 0);
    }
}
```

### 2.3 修改周期

```c
xTimerChangePeriod(xTimer, pdMS_TO_TICKS(2000), 0);  // 改为2秒周期
```

## 三、完整示例

### 示例1：LED闪烁（周期性定时器）

```c
static TimerHandle_t xLedTimer;

void vLedTimerCallback(TimerHandle_t xTimer)
{
    // 每次到期切换LED状态
    GPIO_ToggleBits(GPIOB, GPIO_Pin_5);
}

void main(void)
{
    // 创建500ms周期的LED闪烁定时器
    xLedTimer = xTimerCreate(
        "LedTimer",
        pdMS_TO_TICKS(500),
        pdTRUE,              // 周期性
        NULL,
        vLedTimerCallback
    );

    xTimerStart(xLedTimer, 0);
    vTaskStartScheduler();
}
```

### 示例2：按键消抖（一次性定时器）

```c
static TimerHandle_t xDebounceTimer;
static TaskHandle_t xKeyTaskHandle;

void vDebounceCallback(TimerHandle_t xTimer)
{
    // 定时器到期，读取按键状态
    if (GPIO_ReadInputDataBit(GPIOA, GPIO_Pin_0) == Bit_RESET) {
        // 按键确实被按下，通知任务
        xTaskNotifyGiveFromISR(xKeyTaskHandle, NULL);
    }
}

void EXTI0_IRQHandler(void)
{
    if (EXTI_GetITStatus(EXTI_Line0) != RESET) {
        // 中断触发时重置消抖定时器
        xTimerResetFromISR(xDebounceTimer, NULL);
        EXTI_ClearITPendingBit(EXTI_Line0);
    }
}

void vKeyTask(void *pvParameters)
{
    while (1) {
        ulTaskNotifyTake(pdTRUE, portMAX_DELAY);
        // 确认按键，执行操作
        handle_key_press();
    }
}

void main(void)
{
    xDebounceTimer = xTimerCreate(
        "Debounce",
        pdMS_TO_TICKS(50),   // 50ms消抖
        pdFALSE,              // 一次性
        NULL,
        vDebounceCallback
    );

    xTaskCreate(vKeyTask, "KeyTask", 256, NULL, 2, &xKeyTaskHandle);
    vTaskStartScheduler();
}
```

### 示例3：多个一次性定时器管理传感器轮询

```c
static TimerHandle_t xSensorTimers[3];
static const char *sensor_names[] = {"Temp", "Humidity", "Pressure"};

void vSensorCallback(TimerHandle_t xTimer)
{
    uint32_t sensor_id = (uint32_t)pvTimerGetTimerID(xTimer);
    read_sensor(sensor_id);
}

void main(void)
{
    for (int i = 0; i < 3; i++) {
        xSensorTimers[i] = xTimerCreate(
            sensor_names[i],
            pdMS_TO_TICKS(1000 + i * 500),  // 不同间隔
            pdTRUE,                          // 周期性
            (void *)i,                       // 传感器ID
            vSensorCallback
        );
        xTimerStart(xSensorTimers[i], 0);
    }

    vTaskStartScheduler();
}
```

## 四、配置要求

使用软件定时器需要在 `FreeRTOSConfig.h` 中启用：

```c
#define configUSE_TIMERS                    1   // 启用定时器功能
#define configTIMER_TASK_PRIORITY           (configMAX_PRIORITIES - 1)  // Timer任务优先级
#define configTIMER_QUEUE_LENGTH            5   // Timer命令队列长度
#define configTIMER_TASK_STACK_DEPTH        128 // Timer任务栈深度
```
