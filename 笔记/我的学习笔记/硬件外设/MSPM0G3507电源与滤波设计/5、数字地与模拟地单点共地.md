# 数字地与模拟地单点共地 （ADC 精准测量的前提，隔绝数字噪音）

## 核心概念

- ==模拟地 ≠ 电源== - 电源是 3.3V（正极），地是公共负极
- ==数字地不是单片机内部独有的== - 两片地都布在 PCB 上，人为分开走线
- ==正极电压全通用== - 3.3V 不分模拟数字，只把负极地线分两路走
- ==单点共地 = 只留一个小区大门互通== - 平时互不打扰，整体电位一致，噪音进不去小区

---

## 一、核心误区纠正

### 1.1 错误理解

- ❌ 模拟地就是电源
- ❌ 数字地是单片机内部的地

### 1.2 正确理解

- 电源是 3.3V、5V（正极供电）
- 地 = 电路公共负极，所有电最后都流回地
- 两片地全都布在 PCB 板子上，是我们人为分开走线，不是芯片天生自带分割

---

## 二、数字地 GND 与模拟地 AGND 划分

### 2.1 谁用什么地

| 地类型 | 给谁用 | 特点 |
|--------|--------|------|
| 数字地 GND | IO、LED、按键、串口、蜂鸣器、普通外设 | 电流忽大忽小，地线自带干扰噪音 |
| 模拟地 AGND | ADC、温度传感器、VREF 基准电压 | 怕干扰，一丁点噪声都不能有 |

### 2.2 供电规则（死记）

所有正极电压（3.3V）全部共用，不分模拟数字！只把负极地线分成两路走，最后只用一个 0 欧电阻连在一起（单点共地）。

---

## 三、单点共地的原理

### 3.1 错误做法：大面积连在一起

- 数字电路一跳跳，地线上会产生微小电压波动
- 这个波动会直接窜进模拟电路
- ADC 读数乱跳、不准、漂移

### 3.2 正确做法：单点共地

- 走线完全分开：数字走数字地，模拟走模拟地
- 只在一个点用 0Ω 电阻/磁珠 连起来
- 全局电压统一，又互不串扰

---

## 四、通俗比喻

### 4.1 楼栋排水系统

| 比喻 | 实际含义 |
|------|----------|
| 整个电路 = 一栋楼 | - |
| 3.3V = 自来水 | 大家一起用 |
| 数字地 = 厨房排污管 | 脏、噪音大 |
| 模拟地 = 卧室安静排水管 | 干净 |
| 单点共地 = 所有排水管最后汇总到一根总下水道 | 互不串味，整体水位一致 |

### 4.2 小区大门

- 数字地：大马路，车多吵闹灰尘大
- 模拟地：安静小区
- 单点接地 = 只留一个小区大门互通
- 平时互不打扰，整体电位一致，噪音进不去小区

---

## 五、实操规则

### 5.1 布线规则

| 规则 | 说明 |
|------|------|
| 所有模拟线路走 AGND | ADC、温度传感器、VREF 等 |
| 其余走 GND | IO、LED、按键、串口等 |
| 全程只放一个 0Ω 把 AGND 和主 GND 连死 | 别处绝不乱连地 |

### 5.2 检查方法

- 数字地和模拟地是否分开走线？
- 是否只在一个点用 0Ω 电阻连接？
- 其他地方有没有意外连接？
