# 防反接电路设计 （硬件安全防线，防止电源接反烧毁板子）

## 核心概念

- ==防反接是硬件设计的安全底线== - 防止用户把电源正负极接反，导致电流倒流烧毁芯片
- ==PMOS 防反接是低压系统最主流方案== - 压降极小、不影响地线、功耗极低，便携式电子产品首选
- ==体二极管是“先遣队”== - 插入电源瞬间，体二极管先导通建立电压，随后 PMOS 沟道完全导通将其短路
- ==NMOS 会导致地漂移== - 接在负极回路，系统与输入地之间有电位差，不适合有外部通信的电路

---

## 一、四种常见防反接方案对比

| 方案 | 关键优点 | 关键缺点 | 适用场景 |
|------|----------|----------|----------|
| **串联二极管** | 成本极低、电路最简单 | 压降大（0.3V~0.7V）、发热大、损耗电能 | 小电流、低成本项目 |
| **并联二极管+保险丝** | 触发后断开保护 | 反接时短路电源，对前级供电设备不友好 | 对前级无要求的简单设备 |
| **PMOS（正极）** | 压降极小、不影响地线、功耗低 | 成本相对高、需注意 Vgs 耐压 | **大电流、有信号通信的精细电路（最推荐）** |
| **NMOS（负极）** | 内阻比 PMOS 更小、成本低 | 导致系统地电位抬高（地漂移） | 独立供电、无外部通信的大功率设备 |
| **整流桥** | 正反接都能正常工作 | 双倍二极管压降、功耗和发热最大 | 玩具、交直流通用设备 |

## 二、PMOS 防反接原理（最推荐方案）

### 2.1 电路结构

```
外部输入 VBUS ──► 漏极 (D) ──┐
                              │
                            [PMOS]
                              │
源极 (S) ──► 系统内部 +5V ──┘
              │
              └── 栅极 (G) ── R1(10kΩ) ──► GND
```

### 2.2 正接时导通过程

1. **体二极管先导通**：插入 USB 瞬间，PMOS 内部体二极管（D→S）导通，电流流向 S 极。S 极电压建立，约 $5V - 0.7V = 4.3V$。
2. **栅源负压形成**：栅极 G 通过电阻接地（0V），源极 S 为 4.3V。$V_{GS} = 0V - 4.3V = -4.3V$。
3. **沟道完全导通**：负压超过开启阈值（通常 -0.5V ~ -2V），PMOS 沟道导通，内阻仅几十毫欧。
4. **短路体二极管**：电流改走低阻抗沟道，压降几乎为零（约 45mV），系统电压升至接近 5V。

### 2.3 反接时阻断过程

1. **体二极管截止**：电源接反，体二极管反向截止，电流无法流向 S 极。
2. **栅源无压差**：S 极为低电平，G 极也为低电平，$V_{GS} \approx 0V$。
3. **PMOS 关闭**：达不到开启阈值，无法形成沟道，电路完全断开，保护后级。

## 三、为什么选 PMOS 而不是 NMOS？

| 对比项 | PMOS（接正极） | NMOS（接负极） |
|--------|----------------|----------------|
| 安装位置 | 电源正极回路 | 电源负极（GND）回路 |
| 地线影响 | **系统地 = 输入地**，无干扰 | **系统地 ≠ 输入地**，产生地漂移 |
| 通信影响 | 串口/USB 信号参考地一致，通信稳定 | 信号参考地不同，可能引发通信干扰 |
| 导通内阻 | 略大（几十毫欧） | 更小（几毫欧到几十毫欧） |
| 成本 | 略高 | 略低 |
| **结论** | ✅ **单片机开发板首选** | ❌ 仅适合无通信的大功率独立设备 |

## 四、设计注意事项

- **R1 下拉电阻不可省**：确保栅极默认接地，防止浮空导致误导通
- **Vgs 耐压检查**：输入电压不能超过 PMOS 的 $V_{GS}$ 最大耐压（AO3401A 为 ±12V）
- **体二极管方向**：必须是从 D 指向 S（输入指向输出），接反则无法启动
- **布局靠近输入端**：PMOS 应紧靠电源输入接口放置

- [ ] 栅极通过 10kΩ 电阻下拉到 GND
- [ ] 体二极管方向正确（D→S）
- [ ] 输入电压不超过 PMOS 的 Vgs 耐压
- [ ] 未使用 NMOS 接负极（避免地漂移影响通信）
- [ ] PMOS 靠近电源输入接口放置
