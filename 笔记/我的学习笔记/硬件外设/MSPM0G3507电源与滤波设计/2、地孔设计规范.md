# 地孔设计规范 （两层板免费性能提升，新手布线不足的万能补救）

## 核心概念

- ==地孔把顶层和底层 GND 铺铜连接成整体== - 大幅降低接地阻抗，所有芯片地电位保持一致
- ==地孔像一堵墙== - 挡住信号线之间的相互干扰，几乎消除所有串扰
- ==地孔提升散热能力== - 芯片底部热量通过地孔快速传导到底层铜皮
- ==地孔越多越好== - 只要不违反禁忌，地孔打得越多越好，没有任何副作用

---

## 一、为什么打满地孔

### 1.1 三大决定性好处

| 好处 | 说明 |
|------|------|
| 降低接地阻抗 | GND 不再是两条分开的铜皮，而是立体的接地平面 |
| 消除串扰干扰 | 地孔像一堵墙，挡住信号线之间相互干扰 |
| 提升散热能力 | 芯片底部热量通过地孔快速传导到底层铜皮 |

### 1.2 新手友好度

对于新手来说：你可能布不好线，可能走线绕了远路，可能电源走线不够粗，但只要你打满了地孔，你的板子稳定性就能超过 80% 没打地孔的老手画的板。

---

## 二、必须密集打地孔的地方

| 位置 | 间距 | 说明 |
|------|------|------|
| 去耦电容 GND 旁 | 一个电容一个地孔 | 最重要 |
| 芯片周围 | 每隔 1.5mm | 形成"地笼" |
| 电源走线两侧 | 每隔 2mm | 5V 和 3.3V 走线两侧 |
| 晶振周围 | 一圈 | 40MHz 晶振完全包围 |
| USB 差分对两侧 | 每隔 1.5mm | D+ 和 D- 两侧 |
| SWD 接口 GND 旁 | 每个 GND 一个 | - |
| 空闲 GND 铺铜 | 均匀分布 | - |

---

## 三、绝对不能打地孔的地方

- ❌ 任何元件焊盘上（会导致焊接不良）
- ❌ 太靠近芯片引脚（距离至少 0.2mm）
- ❌ 打断任何信号线
- ❌ 晶振正下方（会影响晶振的寄生参数）
- ❌ USB 差分对中间（会破坏差分信号的阻抗匹配）

---

## 四、嘉立创免费打样最优参数

| 参数 | 值 |
|------|-----|
| 孔径 | 0.3mm（嘉立创免费支持的最小孔径） |
| 焊盘直径 | 0.6mm |
| 间距 | 1.5-2mm（不要小于 1mm，否则工厂可能会拒收） |

---

## 五、自动打地孔方法

在嘉立创 EDA 中：
1. 画完所有走线，铺完 GND 铜
2. 打开"添加过孔"工具
3. 设置过孔参数为 0.3/0.6mm
4. 按住鼠标左键，在所有空闲的 GND 区域疯狂拖动，能打多少打多少

## 附录：快速检查清单

- [ ] 每个去耦电容 GND 旁有地孔
- [ ] 芯片周围每隔 1.5mm 有地孔
- [ ] 电源走线两侧有地孔
- [ ] 晶振周围有一圈地孔
- [ ] 空闲 GND 区域打满地孔
- [ ] 地孔没有打在元件焊盘上
- [ ] 地孔没有打断信号线
- [ ] 地孔参数符合嘉立创要求（0.3/0.6mm，间距 1.5-2mm）
