# MOS管原理与 AO3401A （P 沟道增强型 MOSFET，低压电源开关之王）

## 核心概念

- ==P 沟道 = 箭头朝外== - 导通时靠正电荷（空穴）形成通道
- ==增强型 = 虚线符号== - 默认断开（常闭），需施加电压才能导通
- ==Vgs 负压导通== - 栅极比源极低一定电压时，吸引空穴形成反型层沟道
- ==AO3401A 是经典型号== - SOT-23 封装，-30V 耐压，-4.2A 电流，内阻 <45mΩ，单片机板子防反接首选

---

## 一、MOS 管符号识别

### 1.1 快速区分 PMOS 和 NMOS

| 特征 | PMOS（你使用的） | NMOS |
|------|------------------|------|
| **箭头方向** | 中间箭头**朝外** ➡️ | 中间箭头**朝内** ⬅️ |
| **沟道线** | 虚线（增强型/常闭） | 虚线（增强型/常闭） |
| **体二极管方向** | 从 D 指向 S（向上） | 从 S 指向 D（向下） |
| **导通条件** | $V_{GS} < 0$（G 比 S 低） | $V_{GS} > 0$（G 比 S 高） |
| **常见接法** | 串联在正极 | 串联在负极 |

### 1.2 物理含义

- **P 沟道**：导通时内部靠正电荷（空穴）连成通道传导电流
- **增强型**：$V_{GS} = 0V$ 时默认断开，主动施加电压才导通（现代电路 99% 使用）

## 二、PMOS 导通原理详解

### 2.1 内部结构

在 N 型硅衬底上扩散两个高浓度 P 型区（源极 S 和漏极 D），中间覆盖绝缘二氧化硅层，上方是金属栅极 G。

### 2.2 导通三阶段

| 阶段 | 过程 | 电压状态 |
|------|------|----------|
| **1. 体二极管先导通** | 插入电源瞬间，体二极管（D→S）导通，S 极建立电压 | $V_S \approx V_{in} - 0.7V$ |
| **2. 负压形成** | G 极接地（0V），S 极有电压，产生负压差 | $V_{GS} = 0V - V_S < 0$ |
| **3. 反型层形成** | 负压电场排斥电子、吸引空穴，在 N 型表面形成 P 型通道 | $|V_{GS}| > V_{GS(th)}$ |
| **4. 低阻导通** | P 沟道连通 S 和 D，内阻仅几十毫欧，短路体二极管 | 压降趋近于 0 |

### 2.3 反型层比喻

把 N 型衬底想象成一条干涸的河道（不导电）。栅极负压就像一场大雨，把河道里填满水（空穴），形成一条能导电的水渠（P 沟道），源极和漏极的水就能顺畅流通。

## 三、AO3401A 核心参数

AO3401A 是电子设计中出镜率极高的 P 沟道增强型 MOSFET，5V/3.3V 系统电源开关首选。

### 3.1 关键参数表

| 参数 | 值 | 说明 |
|------|-----|------|
| **型号** | AO3401A | 简称 3401 |
| **类型** | P 沟道增强型 MOSFET | - |
| **封装** | SOT-23 | 3 引脚贴片封装 |
| **$V_{DS}$（漏源耐压）** | -30V | 输入电压不超过 30V 即可安全工作 |
| **$I_D$（最大连续电流）** | -4.2A | 轻松带动几百 mA 到 1A 的单片机系统 |
| **$R_{DS(on)}$（导通内阻）** | <45mΩ (@$V_{GS}=-4.5V$) | 内阻极小，发热极低 |
| **$V_{GS(th)}$（开启阈值）** | -0.5V ~ -2V | 3.3V/5V 系统轻松驱动 |

### 3.2 功耗计算示例

假设板子全负荷运行，电流 1A：
- **损耗功率**：$P = I^2 \times R = 1^2 \times 0.045 = 0.045W = 45mW$（基本不发热）
- **压降**：$U = I \times R = 1A \times 0.045Ω = 0.045V = 45mV$
- **结果**：5V 输入经过后变为 4.955V，对后级电路完全无影响

对比普通二极管：压降 0.4V~0.7V，损耗大且发热严重。

### 3.3 SOT-23 引脚定义

| 引脚 | 名称 | 电路连接 |
|------|------|----------|
| **Pin 1** | G（栅极） | 接 10kΩ 下拉电阻到 GND |
| **Pin 2** | S（源极） | 接系统内部 +5V |
| **Pin 3** | D（漏极） | 接外部输入 VBUS |

## 四、选型指南

购买或选型时关注以下参数：

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 封装 | SOT-23 | 小巧，适合 5V 几安培以下系统 |
| $V_{DS}$ | -20V 或 -30V | 耐压远超 5V 输入，安全 |
| $I_D$ | -2A ~ -4A | 根据板子总电流选择 |
| $R_{DS(on)}$ | 越小越好（30mΩ~50mΩ） | 内阻越小，发热越低 |
| $V_{GS(th)}$ | -0.5V ~ -2V | 确保低压系统能完全导通 |

## 附录：快速检查清单

- [ ] 确认使用的是 P 沟道增强型 MOSFET
- [ ] 符号箭头朝外，沟道线为虚线
- [ ] 体二极管方向从 D 指向 S
- [ ] AO3401A 引脚对应正确（1-G, 2-S, 3-D）
- [ ] 输入电压不超过 -30V（$V_{DS}$ 耐压）
- [ ] 栅极驱动电压满足开启阈值（$|V_{GS}| > 2V$）
- [ ] 实际电流不超过 -4.2A（$I_D$ 最大电流）
