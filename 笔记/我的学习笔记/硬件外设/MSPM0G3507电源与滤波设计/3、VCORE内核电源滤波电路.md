# VCORE 内核电源滤波电路 （磁珠型 π 型滤波，90% "能识别但跑程序就死机" 的根源）

## 核心概念

- ==VCORE 电路是滤波电路，不是供电电路== - 唯一作用是把内部 LDO 输出的 1.8V 电压里的高频噪声滤掉
- ==磁珠是耗能元件，电感是储能元件== - 磁珠把高频噪声转化为热量消耗掉，电感把噪声反射回电源
- ==PA23 是双功能引脚== - VCORE 内核电源输出和 VREF+ ADC 参考电压输入，只能二选一
- ==0Ω 电阻是"断点调试器"== - 方便测量内核工作电流、快速定位电源问题、兼容外部内核供电

---

## 一、MSPM0G3507 电源架构

### 1.1 完整电源流向

```
外部 5V → AMS1117-3.3V → 芯片 VDD 引脚 → 内部 LDO → 1.8V 从 PA23 输出 → VCORE 滤波电路 → 返回芯片给内核供电
```

### 1.2 为什么需要内部 LDO

| 原因 | 说明 |
|------|------|
| 少画一个 LDO | 只需要一个 3.3V LDO，不需要额外再画一个 1.8V LDO |
| 自动电源管理 | 内部 LDO 会根据 CPU 工作状态自动调节电压和电流 |
| 降低成本 | 一个内部 LDO 的成本几乎为 0 |
| 提高可靠性 | 内部 LDO 和内核集成在一起，走线极短，噪声更小 |

### 1.3 为什么要把 1.8V 引到 PA23 外面

芯片内部只能集成非常小的电容（pF 级），无法有效滤除内核产生的高频噪声。CPU 内核工作在 40MHz，会产生大量的 100MHz 以上的高频噪声。

---

## 二、VCORE 电路元件清单

| 元件 | 参数 | 作用 | 绝对不能做的事 |
|------|------|------|----------------|
| R17 | 0Ω 0603 | 调试电阻，隔离 VCORE 方便测量内核电流 | ❌ 不能去掉，不能换其他阻值 |
| C17 | 100nF 0603 | 高频去耦电容，滤除内核高频噪声 | ❌ 不能去掉，不能换其他容值 |
| C18 | 1μF 0603 | 低频去耦电容，滤除电源低频纹波 | ❌ 不能去掉，不能换其他容值 |
| L1 | 600Ω@100MHz 磁珠 | 高频抑制，防止内核噪声传到 3.3V 网络 | ❌ 不能用 0Ω 电阻代替 |

---

## 三、0Ω 电阻 R17 的 3 个核心作用

### 3.1 测量内核工作电流（低功耗调试必备）

- 正常工作时：焊上 0Ω 电阻，电路正常导通
- 需要测量电流时：焊掉 0Ω 电阻，万用表电流档串入
- 可精确测量不同模式下的内核电流：运行模式、睡眠模式、深度睡眠模式

### 3.2 快速定位电源问题

1. 焊掉 0Ω 电阻
2. 上电测量左边（PA23 引脚）的电压，应该是 1.8V 左右
3. 如果不是 1.8V，说明芯片内部的 LDO 坏了，换芯片
4. 如果是 1.8V，说明内部 LDO 正常，问题出在外部滤波电路
5. 再测量右边（磁珠那边）的电压，应该也是 1.8V 左右
6. 如果不是，说明磁珠或电容有问题

### 3.3 兼容外部内核供电方案

在极端低功耗应用中，可以用外部更高效的 1.8V LDO 直接给内核供电：
1. 焊掉这个 0Ω 电阻
2. 把外部 1.8V 电源接到电阻右边的焊盘上
3. 绕过芯片内部的 LDO，使用外部电源给内核供电

## 四、为什么必须用磁珠，不能用电感或 0Ω

| 元件 | 对直流态度 | 对高频噪声态度 | 后果 |
|------|------------|----------------|------|
| 0Ω 电阻 | 无阻碍 | 无滤波 | 内核噪声直接传到 3.3V，程序跑飞、ADC 漂移 |
| 电感 | 有压降 | 反射噪声 | 噪声反射回内核，内核电源更脏 |
| 磁珠 | 几乎无压降 | 吸收转化为热量 | ✅ 正确选择 |

### 4.1 600Ω@100MHz 参数含义

- 在 100MHz 频率下，磁珠的阻抗是 600Ω
- 频率越高，磁珠的阻抗越大，对高频噪声的抑制能力越强
- 对于直流（0Hz），磁珠的阻抗几乎为 0，不会影响电源的正常供电

## 五、PA23 引脚功能互斥

PA23 是双功能引脚，只能二选一：

| 功能 | 说明 | 默认使用 |
|------|------|----------|
| VCORE 内核电源输出 | 输出 1.8V 给内核供电 | ✅ 99% 应用使用 |
| VREF+ ADC 参考电压输入 | 外部高精度参考电压 | ❌ 极少使用 |

**注意**：两个功能互斥，不能同时使用。第一块开发板只画 VCORE 电路，ADC 参考电压电路空着不焊。

- [ ] C17 是 100nF，C18 是 1μF
- [ ] R17 是 0Ω 电阻，没有去掉
- [ ] 所有元件参数正确，一个都不能少
- [ ] PA23 只画了 VCORE 电路，没有并联 ADC 参考电压电路
