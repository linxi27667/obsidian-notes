# AMS1117 标准电路 （3 个电容缺一不可，少一个自激振荡）

## 核心概念

- ==AMS1117 必须 3 个电容== - 输入电解电容、输出电解电容、输出陶瓷电容，少一个都可能导致 LDO 自激振荡
- ==输出 0.1μF 陶瓷电容最容易漏掉== - 漏掉会导致 AMS1117 产生几百 kHz 的自激振荡
- ==输出电压看起来是 3.3V，但实际上有几百 mV 的纹波== - 会导致芯片工作不稳定，程序莫名其妙跑飞

---

## 一、标准电路

### 1.1 完整电路结构

```
输入 5V → 10μF 电解电容 → AMS1117-3.3V → 10μF 电解电容 + 0.1μF 陶瓷电容 → 输出 3.3V
```

### 1.2 每个电容的作用

| 电容 | 参数 | 作用 | 能不能省 |
|------|------|------|----------|
| 输入电解电容 | 10μF 16V | 防止 LDO 自激振荡，稳定输入电压 | ✅ 必须有 |
| 输出电解电容 | 10μF 16V | 储能，提供瞬间大电流，稳定输出电压 | ✅ 必须有 |
| 输出陶瓷电容 | 0.1μF 50V X7R | 滤除高频噪声 | ✅ 必须有 |

---

## 二、新手最致命错误

### 2.1 错误做法

只加两个电解电容，漏掉输出的 0.1μF 陶瓷电容。

### 2.2 后果

- AMS1117 产生几百 kHz 的自激振荡
- 输出电压看起来是 3.3V，但实际上有几百 mV 的纹波
- 芯片工作不稳定，程序莫名其妙跑飞

---

## 三、电容选型说明

### 3.1 电解电容

- 用于储能和低频滤波
- 容量大，体积大
- 有极性，注意正负极

### 3.2 陶瓷电容

- 用于高频滤波
- 容量小，体积小
- 无极性，X7R 材质温度稳定性好

### 3.3 为什么需要两种电容

- 电解电容滤除低频噪声，提供储能
- 陶瓷电容滤除高频噪声，响应速度快
- 两者配合使用，覆盖全频段噪声

---

## 四、布局建议

- 输入电容紧挨 AMS1117 输入引脚
- 输出电容紧挨 AMS1117 输出引脚
- 0.1μF 陶瓷电容比 10μF 电解电容更靠近输出引脚
- 所有电容的 GND 就近打地孔
