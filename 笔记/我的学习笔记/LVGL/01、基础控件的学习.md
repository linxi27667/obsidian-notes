## 一、对象基础创建

### 1.1 获取当前屏幕

```c
lv_obj_t * scr = lv_screen_active();  // 或 lv_scr_act()
```

| 函数 | 说明 |
|------|------|
| `lv_screen_active()` | 获取当前活动屏幕的指针 |
| `lv_scr_act()` | 简写形式，功能相同 |

**返回值**: `lv_obj_t*` - 当前屏幕对象的指针

---

### 1.2 创建对象

```c
lv_obj_t * obj = lv_obj_create(scr);
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `parent` | `lv_obj_t*` | 父对象指针，设为 `scr` 或 `lv_scr_act()` 表示创建在屏幕上 |

**说明**:
- 创建的对象默认大小为 100x100 像素
- 默认位置为 (0, 0)，相对于父对象的左上角
- 对象创建后会立即显示在屏幕上

**示例**:
```c
// 创建在屏幕上的对象
lv_obj_t * obj = lv_obj_create(lv_scr_act());

// 创建在另一个对象内的子对象
lv_obj_t * parent = lv_obj_create(lv_scr_act());
lv_obj_t * child = lv_obj_create(parent);  // child 是 parent 的子对象
```

## 二、按钮控件 (Button)

### 2.1 创建按钮

```c
lv_obj_t * btn = lv_btn_create(obj);
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `parent` | `lv_obj_t*` | 父对象指针，按钮将作为子对象创建在父对象内 |

---

### 2.2 设置大小

```c
lv_obj_set_size(btn, 40, 40);
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `obj` | `lv_obj_t*` | 要设置大小的对象指针 |
| `w` | `int32_t` | 宽度（像素） |
| `h` | `int32_t` | 高度（像素） |

**替代方法**:
```c
lv_obj_set_width(btn, 40);   // 只设置宽度
lv_obj_set_height(btn, 40);  // 只设置高度
```

**特殊值**:
- `LV_SIZE_CONTENT` - 自动根据内容调整大小

---

### 2.3 设置位置

```c
lv_obj_set_pos(btn, 0, 10);
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `obj` | `lv_obj_t*` | 要设置位置的对象指针 |
| `x` | `int32_t` | X 坐标（相对于父对象内容区的左边缘） |
| `y` | `int32_t` | Y 坐标（相对于父对象内容区的上边缘） |

**替代方法**:
```c
lv_obj_set_x(btn, 0);   // 只设置 X 坐标
lv_obj_set_y(btn, 10);  // 只设置 Y 坐标
```

**百分比坐标**:
```c
lv_obj_set_x(btn, lv_pct(50));  // 设置为父对象宽度的 50%
```

---

### 2.4 按钮事件处理示例

```c
static void btn_event_cb(lv_event_t * e)
{
    lv_event_code_t code = lv_event_get_code(e);
    lv_obj_t * btn = lv_event_get_target(e);
    
    if(code == LV_EVENT_CLICKED) {
        printf("Button clicked!\n");
    }
}

// 使用
lv_obj_t * btn = lv_btn_create(lv_scr_act());
lv_obj_add_event_cb(btn, btn_event_cb, LV_EVENT_CLICKED, NULL);
```

**常用事件类型**:
- `LV_EVENT_CLICKED` - 点击事件
- `LV_EVENT_PRESSED` - 按下事件
- `LV_EVENT_LONG_PRESSED` - 长按事件
- `LV_EVENT_VALUE_CHANGED` - 值改变事件（用于可切换按钮）

## 三、滑块控件 (Slider)

### 3.1 创建滑块

```c
lv_obj_t * slider = lv_slider_create(obj);
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `parent` | `lv_obj_t*` | 父对象指针，滑块将作为子对象创建在父对象内 |

**返回值**: `lv_obj_t*` - 新创建的滑块对象指针

---

### 3.2 设置大小

```c
lv_obj_set_size(slider, 80, 10);
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `obj` | `lv_obj_t*` | 要设置大小的对象指针 |
| `w` | `int32_t` | 宽度（像素）- 滑块的主要方向长度 |
| `h` | `int32_t` | 高度（像素）- 滑块的厚度 |

---

### 3.3 设置位置

```c
lv_obj_set_pos(slider, 10, 80);
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `obj` | `lv_obj_t*` | 要设置位置的对象指针 |
| `x` | `int32_t` | X 坐标（相对于父对象内容区的左边缘） |
| `y` | `int32_t` | Y 坐标（相对于父对象内容区的上边缘） |

---

### 3.4 设置数值范围

```c
lv_slider_set_range(slider, 0, 100);
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `obj` | `lv_obj_t*` | 滑块对象指针 |
| `min` | `int32_t` | 最小值 |
| `max` | `int32_t` | 最大值 |

**说明**:
- 滑块的值必须在 [min, max] 范围内
- 可以设置负数范围，如 `lv_slider_set_range(slider, -100, 100)`

---

### 3.5 设置当前值

```c
lv_slider_set_value(slider, 50, LV_ANIM_ON);
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `obj` | `lv_obj_t*` | 滑块对象指针 |
| `value` | `int32_t` | 要设置的新值 |
| `anim` | `lv_anim_enable_t` | 是否使用动画过渡 |

**动画参数**:
- `LV_ANIM_ON` - 启用动画，值会平滑过渡到新值
- `LV_ANIM_OFF` - 禁用动画，值立即改变

**相关函数**:
```c
// 获取当前值
int32_t val = lv_slider_get_value(slider);

// 获取最小值
int32_t min = lv_slider_get_min_value(slider);

// 获取最大值
int32_t max = lv_slider_get_max_value(slider);
```

---

### 3.6 滑块模式

```c
lv_slider_set_mode(slider, LV_SLIDER_MODE_NORMAL);
```

| 模式 | 说明 |
|------|------|
| `LV_SLIDER_MODE_NORMAL` | 普通模式，单个滑块 |
| `LV_SLIDER_MODE_SYMMETRICAL` | 对称模式，滑块居中，可向两侧拖动 |
| `LV_SLIDER_MODE_RANGE` | 范围模式，有两个滑块，可设置最小和最大值 |

---

### 3.7 滑块事件处理示例

```c
static void slider_event_cb(lv_event_t * e)
{
    lv_event_code_t code = lv_event_get_code(e);
    lv_obj_t * slider = lv_event_get_target(e);
    
    if(code == LV_EVENT_VALUE_CHANGED) {
        int32_t val = lv_slider_get_value(slider);
        printf("Slider value: %d\n", val);
    }
}

// 使用
lv_obj_t * slider = lv_slider_create(lv_scr_act());
lv_obj_add_event_cb(slider, slider_event_cb, LV_EVENT_VALUE_CHANGED, NULL);
lv_slider_set_range(slider, 0, 100);

## 四、对象删除

### 4.1 延迟删除对象

```c
lv_obj_delete_delayed(btn, 1000);
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `obj` | `lv_obj_t*` | 要删除的对象指针 |
| `delay_ms` | `uint32_t` | 延迟时间（毫秒），1000 表示延迟 1 秒删除 |

**说明**:
- 延迟删除可以避免在事件处理中直接删除对象导致的未定义行为
- 延迟时间结束后，对象会被自动从父对象中移除并释放内存

**立即删除**:
```c
lv_obj_delete(btn);  // 立即删除对象
```

## 注意事项与最佳实践

### 内存管理

| 注意点 | 说明 | 建议 |
|--------|------|------|
| **对象生命周期** | LVGL对象分配在堆上，需手动删除 | 页面切换时及时删除旧页面对象 |
| **延迟删除** | 在事件回调中删除对象需使用 `lv_obj_delete_delayed()` | 避免在事件处理中直接删除 |
| **内存泄漏** | 忘记删除对象会导致内存耗尽 | 建立统一的对象管理机制 |
| **子对象继承** | 删除父对象会自动删除所有子对象 | 无需单独删除每个子对象 |

### 常见错误

```c
// ❌ 错误：在事件回调中直接删除对象
static void btn_event_cb(lv_event_t * e) {
    lv_obj_t * btn = lv_event_get_target(e);
    lv_obj_delete(btn);  // 可能导致崩溃！
}

// ✅ 正确：使用延迟删除
static void btn_event_cb(lv_event_t * e) {
    lv_obj_t * btn = lv_event_get_target(e);
    lv_obj_delete_delayed(btn, 100);  // 100ms后安全删除
}

// ❌ 错误：删除后继续使用对象指针
lv_obj_delete(obj);
lv_obj_set_pos(obj, 0, 0);  // 未定义行为！

// ✅ 正确：删除后清空指针
lv_obj_delete(obj);
obj = NULL;  // 避免悬空指针
```

### 控件使用建议

1. **滑块范围设置**：先设置范围，再设置当前值
2. **按钮文字**：文字标签要居中显示
3. **对象命名**：复杂界面建议给对象命名便于查找
4. **样式分离**：将样式定义放在单独函数中，提高可读性
