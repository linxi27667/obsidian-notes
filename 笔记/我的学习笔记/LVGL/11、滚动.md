# LVGL 滚动系统 （内容滚动与视窗管理）

## 核心概念

- **内容区域 (Content Area)** - 控件所有子控件的总边界，可能大于控件本身的尺寸
- **视窗 (View Window)** - 控件实际可见的区域，即控件本身的尺寸
- **可滚动性 (Scrollable)** - 当内容区域大于视窗时，控件自动变为可滚动
- **滚动位置 (Scroll Position)** - 视窗相对于内容区域的偏移量，用 X/Y 坐标表示
- **滚动链 (Scroll Chaining)** - 当子控件滚动到边界时，将滚动事件传递给父控件

## 一、滚动基础

### 1.1 启用/禁用滚动

```c
// 启用滚动（默认大多数控件已启用）
lv_obj_add_flag(widget, LV_OBJ_FLAG_SCROLLABLE);

// 禁用滚动
lv_obj_remove_flag(widget, LV_OBJ_FLAG_SCROLLABLE);
```

| 参数 | 说明 |
|------|------|
| `widget` | 目标控件 |
| `LV_OBJ_FLAG_SCROLLABLE` | 可滚动标志 |

**实际示例：**

```c
// 创建一个固定大小的容器，禁用滚动
lv_obj_t * fixed_container = lv_obj_create(lv_screen_active());
lv_obj_set_size(fixed_container, 300, 200);
lv_obj_remove_flag(fixed_container, LV_OBJ_FLAG_SCROLLABLE);

// 创建一个可滚动的列表
lv_obj_t * scrollable_list = lv_obj_create(lv_screen_active());
lv_obj_set_size(scrollable_list, 300, 400);
lv_obj_add_flag(scrollable_list, LV_OBJ_FLAG_SCROLLABLE);
lv_obj_add_flag(scrollable_list, LV_OBJ_FLAG_SCROLL_MOMENTUM);  // 惯性滚动
```

### 1.2 滚动方向控制

```c
// 设置滚动方向
lv_obj_set_scroll_dir(widget, LV_DIR_VER);  // 仅垂直滚动
```

| 参数 | 说明 |
|------|------|
| `widget` | 目标控件 |
| `direction` | 滚动方向，可选值见下表 |

| 方向常量 | 说明 | 数值 |
|----------|------|------|
| `LV_DIR_TOP` | 仅向上滚动 | 0x01 |
| `LV_DIR_BOTTOM` | 仅向下滚动 | 0x02 |
| `LV_DIR_LEFT` | 仅向左滚动 | 0x04 |
| `LV_DIR_RIGHT` | 仅向右滚动 | 0x08 |
| `LV_DIR_HOR` | 水平滚动（左+右） | 0x0C |
| `LV_DIR_VER` | 垂直滚动（上+下） | 0x03 |
| `LV_DIR_ALL` | 所有方向 | 0x0F |

**组合使用：**

```c
// 仅允许向上和向左滚动
lv_obj_set_scroll_dir(widget, LV_DIR_TOP | LV_DIR_LEFT);

// 仅垂直滚动（常用）
lv_obj_set_scroll_dir(widget, LV_DIR_VER);

// 禁止所有滚动
lv_obj_set_scroll_dir(widget, LV_DIR_NONE);  // 或 0
```

## 二、滚动条 (Scrollbar)

### 2.1 滚动条模式

```c
// 设置滚动条显示模式
lv_obj_set_scrollbar_mode(widget, LV_SCROLLBAR_MODE_AUTO);
```

| 参数 | 说明 |
|------|------|
| `widget` | 目标控件 |
| `mode` | 滚动条模式 |

| 模式 | 说明 | 应用场景 |
|------|------|----------|
| `LV_SCROLLBAR_MODE_OFF` | 从不显示滚动条 | 完全隐藏滚动功能提示 |
| `LV_SCROLLBAR_MODE_ON` | 始终显示滚动条 | 始终提示可滚动 |
| `LV_SCROLLBAR_MODE_ACTIVE` | 滚动时显示 | 默认推荐，节省空间 |
| `LV_SCROLLBAR_MODE_AUTO` | 内容超出时显示 | 自动判断是否需要 |

**实际示例：**

```c
// 创建自动显示滚动条的列表
lv_obj_t * list = lv_list_create(lv_screen_active());
lv_obj_set_scrollbar_mode(list, LV_SCROLLBAR_MODE_AUTO);

// 添加10个列表项
for(int i = 0; i < 10; i++) {
    lv_list_add_btn(list, LV_SYMBOL_FILE, "File");
}

// 创建始终显示滚动条的文本区域
lv_obj_t * textarea = lv_textarea_create(lv_screen_active());
lv_obj_set_scrollbar_mode(textarea, LV_SCROLLBAR_MODE_ON);
```

### 2.2 滚动条样式

```c
// 设置滚动条颜色
static lv_style_t style_scrollbar;
lv_style_init(&style_scrollbar);
lv_style_set_bg_color(&style_scrollbar, lv_palette_main(LV_PALETTE_RED));
lv_style_set_width(&style_scrollbar, 8);  // 滚动条宽度

lv_obj_add_style(widget, &style_scrollbar, LV_PART_SCROLLBAR);
```

| 样式属性 | 说明 | 示例值 |
|----------|------|--------|
| `bg_color` | 滚动条颜色 | `lv_palette_main(LV_PALETTE_RED)` |
| `width` | 滚动条宽度 | 8 (像素) |
| `pad_right` | 右侧间距 | 2 (像素) |

**滚动状态样式：**

```c
// 滚动时滚动条变为蓝色
static lv_style_t style_scrollbar_scrolled;
lv_style_init(&style_scrollbar_scrolled);
lv_style_set_bg_color(&style_scrollbar_scrolled, lv_palette_main(LV_PALETTE_BLUE));

lv_obj_add_style(widget, &style_scrollbar_scrolled, 
    LV_STATE_SCROLLED | LV_PART_SCROLLBAR);
```

## 三、滚动特性

### 3.1 弹性滚动 (Elastic Scroll)

```c
// 启用弹性滚动效果
lv_obj_add_flag(widget, LV_OBJ_FLAG_SCROLL_ELASTIC);
```

**效果说明：**

```
正常滚动：                         弹性滚动：
┌────────────┐                   ┌────────────┐
│ ┌────────┐ │                   │ ┌────────┐ │
│ │ Content│ │                   │ │ Content│ │
│ │   ↑    │ │                   │ │   ↓    │ │  ← 超出边界时
│ │   ↑    │ │                   │ │   ↓    │ │     内容被拉伸
│ │   ↑    │ │                   │ │   ↓    │ │
│ │ (停止) │ │                   │ │ (回弹) │ │  ← 松开后
│ └────────┘ │                   │ └────────┘ │     平滑回弹
└────────────┘                   └────────────┘
  到达边界立即停止                 超出边界后可继续拉动
                                 松开后动画回到边界
```

### 3.2 惯性滚动 (Scroll Momentum)

```c
// 启用惯性滚动
lv_obj_add_flag(widget, LV_OBJ_FLAG_SCROLL_MOMENTUM);
```

**效果说明：**

```
无惯性滚动：                       有惯性滚动：
手指滑动 → 停止 → 内容停止        手指滑动 → 释放 → 内容继续滑动
                               (减速直至停止)
                               
适用场景：                         适用场景：
- 精确控制位置                    - 长列表快速浏览
- 短内容                          - 需要自然手感
```

### 3.3 滚动链 (Scroll Chaining)

```c
// 启用滚动链（默认启用）
lv_obj_add_flag(widget, LV_OBJ_FLAG_SCROLL_CHAIN_VER);  // 垂直链
lv_obj_add_flag(widget, LV_OBJ_FLAG_SCROLL_CHAIN_HOR);  // 水平链

// 禁用滚动链
lv_obj_remove_flag(widget, LV_OBJ_FLAG_SCROLL_CHAIN_VER);
```

**工作原理：**

```
启用滚动链：                       禁用滚动链：

┌──────────────────────┐         ┌──────────────────────┐
│ ┌──────────────────┐ │         │ ┌──────────────────┐ │
│ │ 子控件(可滚动)    │ │         │ │ 子控件(可滚动)    │ │
│ │   到达边界        │ │         │ │   到达边界        │ │
│ │       ↓          │ │         │ │       ×          │ │  ← 停止
│ └───────┬──────────┘ │         │ └──────────────────┘ │
│         │ 滚动传递给父│         │                      │
│         ▼            │         │                      │
│    父控件滚动        │         │    父控件不滚动      │
└──────────────────────┘         └──────────────────────┘

应用场景：                        应用场景：
- 嵌套滚动容器                    - 独立滚动区域
- 页面整体滚动                    - 固定侧边栏
```

### 3.4 吸附对齐 (Snap)

```c
// 启用子控件吸附
lv_obj_add_flag(child_widget, LV_OBJ_FLAG_SNAPPABLE);

// 设置吸附对齐方式
lv_obj_set_scroll_snap_x(container, LV_SCROLL_SNAP_CENTER);  // 水平居中吸附
lv_obj_set_scroll_snap_y(container, LV_SCROLL_SNAP_START);   // 垂直顶部吸附
```

| 吸附模式 | 说明 | 效果 |
|----------|------|------|
| `LV_SCROLL_SNAP_NONE` | 禁用吸附 | 自由停止 |
| `LV_SCROLL_SNAP_START` | 对齐到开始边 | 左/上对齐 |
| `LV_SCROLL_SNAP_END` | 对齐到结束边 | 右/下对齐 |
| `LV_SCROLL_SNAP_CENTER` | 居中对齐 | 中心对齐 |

**实际示例：轮播图效果**

```c
// 创建轮播容器
lv_obj_t * carousel = lv_obj_create(lv_screen_active());
lv_obj_set_size(carousel, 300, 200);
lv_obj_set_flex_flow(carousel, LV_FLEX_FLOW_ROW);
lv_obj_add_flag(carousel, LV_OBJ_FLAG_SCROLL_ONE);  // 每次只滚动一个

// 设置水平居中对齐
lv_obj_set_scroll_snap_x(carousel, LV_SCROLL_SNAP_CENTER);

// 创建多个轮播项
for(int i = 0; i < 5; i++) {
    lv_obj_t * item = lv_obj_create(carousel);
    lv_obj_set_size(item, 280, 180);
    lv_obj_add_flag(item, LV_OBJ_FLAG_SNAPPABLE);  // 启用吸附
    
    lv_obj_t * label = lv_label_create(item);
    lv_label_set_text_fmt(label, "Slide %d", i + 1);
    lv_obj_center(label);
}
```

### 3.5 单次滚动 (Scroll One)

```c
// 启用单次滚动（配合吸附使用）
lv_obj_add_flag(container, LV_OBJ_FLAG_SCROLL_ONE);
```

**效果说明：**

```
普通滚动：                         Scroll One：
自由滚动任意距离                   每次只滚动一个子项
                                  
┌─────┬─────┬─────┐              ┌─────┬─────┬─────┐
│     │     │     │              │     │     │     │
│  1  │  2  │  3  │  ←           │  1  │  2  │  3  │
│     │     │     │              │     │     │     │
└─────┴─────┴─────┘              └─────┴─────┴─────┘
                                   
快速滑动可能停在中间               总是停在下一个完整项
```

## 四、编程控制滚动

### 4.1 滚动到指定位置

```c
// 相对滚动（偏移量）
lv_obj_scroll_by(widget, x, y, LV_ANIM_ON);

// 绝对滚动（目标坐标）
lv_obj_scroll_to(widget, x, y, LV_ANIM_ON);

// 滚动到X/Y坐标
lv_obj_scroll_to_x(widget, x, LV_ANIM_ON);
lv_obj_scroll_to_y(widget, y, LV_ANIM_ON);
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `widget` | `lv_obj_t*` | 目标控件 |
| `x/y` | `int32_t` | 滚动偏移量或目标坐标（像素） |
| `anim_en` | `lv_anim_enable_t` | `LV_ANIM_ON` 或 `LV_ANIM_OFF` |

**坐标说明：**

```
滚动坐标系：

原始位置 (scroll_x=0, scroll_y=0):
┌──────────────────────┐
│ 内容左上角与视窗左上角重合│
│ ┌──────────┐         │
│ │ 内容      │         │
│ │   A      │         │
│ └──────────┘         │
└──────────────────────┘

向下滚动 (scroll_y=100):
┌──────────────────────┐
│ 视窗显示内容的中下部   │
│ ┌──────────┐         │
│ │ 内容      │ ↑ 被滚 │
│ │   B      │   出视窗│
│ │   C      │ ← 可见  │
│ └──────────┘         │
└──────────────────────┘

scroll_x: 正数 = 向右滚动，负数 = 向左滚动
scroll_y: 正数 = 向下滚动，负数 = 向上滚动
```

**实际示例：**

```c
// 示例：点击按钮滚动到列表顶部
lv_obj_t * list = lv_list_create(lv_screen_active());

// ... 添加多个列表项 ...

// 滚动到顶部按钮
lv_obj_t * top_btn = lv_btn_create(lv_screen_active());
lv_obj_add_event_cb(top_btn, 
    lambda void(lv_event_t * e) {
        lv_obj_t * list = lv_event_get_user_data(e);
        lv_obj_scroll_to_y(list, 0, LV_ANIM_ON);  // 滚动到Y=0（顶部）
    }, LV_EVENT_CLICKED, list);
```

### 4.2 获取滚动位置

```c
// 获取当前滚动位置
int32_t scroll_x = lv_obj_get_scroll_x(widget);
int32_t scroll_y = lv_obj_get_scroll_y(widget);

// 获取边界距离
int32_t top = lv_obj_get_scroll_top(widget);     // 顶部滚动距离
int32_t bottom = lv_obj_get_scroll_bottom(widget); // 底部滚动距离
int32_t left = lv_obj_get_scroll_left(widget);   // 左侧滚动距离
int32_t right = lv_obj_get_scroll_right(widget); // 右侧滚动距离
```

**实际示例：滚动进度指示器**

```c
static void scroll_event_cb(lv_event_t * e)
{
    lv_obj_t * cont = lv_event_get_target(e);
    lv_obj_t * progress_bar = lv_event_get_user_data(e);
    
    // 计算滚动百分比
    int32_t scroll_y = lv_obj_get_scroll_y(cont);
    int32_t scroll_bottom = lv_obj_get_scroll_bottom(cont);
    
    if(scroll_bottom > 0) {
        int percentage = (scroll_y * 100) / scroll_bottom;
        lv_bar_set_value(progress_bar, percentage, LV_ANIM_OFF);
    }
}

// 使用
lv_obj_t * scrollable = lv_obj_create(lv_screen_active());
lv_obj_add_event_cb(scrollable, scroll_event_cb, LV_EVENT_SCROLL, progress_bar);
```

### 4.3 滚动到视图

```c
// 将控件滚动到其父容器的可视区域
lv_obj_scroll_to_view(widget, LV_ANIM_ON);

// 递归滚动，确保控件在所有嵌套父容器中可见
lv_obj_scroll_to_view_recursive(widget, LV_ANIM_ON);
```

**应用场景：**

```c
// 场景：表单验证失败，滚动到错误字段
void show_field_error(lv_obj_t * field)
{
    // 高亮错误字段
    lv_obj_set_style_border_color(field, lv_palette_main(LV_PALETTE_RED), 0);
    
    // 滚动到可视区域
    lv_obj_scroll_to_view_recursive(field, LV_ANIM_ON);
}

// 场景：键盘导航时自动滚动聚焦项到视图
void focus_next_item(lv_group_t * g)
{
    lv_obj_t * focused = lv_group_get_focused(g);
    lv_obj_scroll_to_view(focused, LV_ANIM_ON);
}
```

## 五、滚动事件

### 5.1 滚动相关事件

| 事件 | 触发时机 | 参数 |
|------|----------|------|
| `LV_EVENT_SCROLL_BEGIN` | 滚动开始时 | `lv_anim_t*` 或 `NULL` |
| `LV_EVENT_SCROLL` | 滚动位置变化时 | 无 |
| `LV_EVENT_SCROLL_END` | 滚动结束时 | 无 |

**实际示例：**

```c
// 监听滚动开始
lv_obj_add_event_cb(container, 
    lambda void(lv_event_t * e) {
        if(lv_event_get_code(e) == LV_EVENT_SCROLL_BEGIN) {
            LV_LOG_USER("Scroll started");
            // 可以在这里暂停某些操作
        }
    }, LV_EVENT_ALL, NULL);

// 监听滚动结束
lv_obj_add_event_cb(container,
    lambda void(lv_event_t * e) {
        if(lv_event_get_code(e) == LV_EVENT_SCROLL_END) {
            LV_LOG_USER("Scroll ended");
            // 可以在这里加载更多内容
        }
    }, LV_EVENT_ALL, NULL);
```

## 六、Self Size（自尺寸）

### 6.1 概念说明

**Self Size** 用于告诉LVGL控件内容的实际大小，影响滚动范围的计算。

```
默认情况：                         设置Self Size后：

内容大小由子控件决定              内容由你显式指定
┌──────────────────┐             ┌──────────────────┐
│ Child 1          │             │                  │
│ Child 2          │             │                  │
│ ...              │             │  自定义内容区域   │
│                  │             │  500×300         │
│ 实际高度: 动态    │             │                  │
└──────────────────┘             └──────────────────┘
```

### 6.2 设置Self Size

```c
// 在 LV_EVENT_GET_SELF_SIZE 事件中设置
static void my_event_cb(lv_event_t * e)
{
    if(lv_event_get_code(e) == LV_EVENT_GET_SELF_SIZE) {
        lv_point_t * p = lv_event_get_param(e);
        
        // 如果x需要计算
        if(p->x >= 0) {
            p->x = 500;  // 内容宽度500px
        }
        
        // 如果y需要计算
        if(p->y >= 0) {
            p->y = 300;  // 内容高度300px
        }
    }
}

lv_obj_add_event_cb(widget, my_event_cb, LV_EVENT_GET_SELF_SIZE, NULL);
```

## 注意事项与最佳实践

### 滚动性能优化

| 注意点 | 说明 | 建议 |
|--------|------|------|
| **子控件数量** | 过多子控件影响滚动性能 | 使用虚拟列表，只渲染可见项 |
| **复杂样式** | 阴影、渐变等效果影响帧率 | 滚动时简化样式 |
| **图片资源** | 大图片解码耗时 | 预加载或裁剪图片 |
| **滚动事件** | 频繁事件处理影响性能 | 使用节流或防抖 |

### 常见错误

```c
// ❌ 错误：在滚动事件中进行耗时操作
static void scroll_cb(lv_event_t * e) {
    heavy_calculation();  // 阻塞UI线程！
}

// ✅ 正确：延迟处理或使用队列
static void scroll_cb(lv_event_t * e) {
    // 设置标志，在空闲时处理
    need_update = true;
}

// ❌ 错误：手动设置滚动位置超出范围
lv_obj_scroll_to_y(obj, 99999, LV_ANIM_OFF);  // 超出内容边界

// ✅ 正确：检查边界或使用自动滚动
lv_obj_scroll_to_view(child, LV_ANIM_ON);
```
