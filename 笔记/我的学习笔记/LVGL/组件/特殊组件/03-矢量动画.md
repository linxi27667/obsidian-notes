# 矢量动画 (lv_lottie)

> **重要性**: ★☆☆☆☆ (5/10) — 矢量动画播放

## 概述
Lottie 解析、栅格化和播放 Lottie 动画（JSON格式），被描述为 "SVG 和 GIF 的现代替代品"。

## 部件与样式
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 背景（通常透明） |

## 依赖
- **ThorVG** 库（C++编写）
- `lv_conf.h`: `LV_USE_THORVG_INTERNAL` 或 `LV_USE_THORVG_EXTERNAL`
- `LV_USE_VECTOR_GRAPHIC` 需启用

## 核心 API

### 缓冲区
```c
// ARGB8888 格式，大小 = w × h × 4 字节
static uint8_t buf[64 * 64 * 4];
lv_lottie_set_buffer(lottie, 64, 64, buf);

// 或使用 draw buffer
LV_DRAW_BUF_DEFINE_STATIC(draw_buf, 64, 64, LV_COLOR_FORMAT_ARGB8888_PREMULTIPLIED);
lv_lottie_set_draw_buf(lottie, &draw_buf);
```

### 来源
```c
lv_lottie_set_src_data(lottie, data, size);  // 从数组加载
lv_lottie_set_src_file(lottie, "path/file.json");  // 从文件加载
```

### 动画控制
```c
lv_anim_t *anim = lv_lottie_get_anim(lottie);  // 获取动画（默认60FPS无限循环）
```

### 转换工具
```bash
python lvgl/scripts/filetohex.py lottie.json > out.txt
```

## 事件与按键
- 无特殊事件，不处理按键

## 关键要点
- 现代矢量动画（Adobe After Effects 导出）
- 需要 ThorVG（C++编译能力）
- 缓冲区必须 ARGB8888 预乘格式
- 文件路径不使用 LVGL 文件系统接口
