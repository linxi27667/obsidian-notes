# 动画图片 (lv_animimg)

> **重要性**: ★★☆☆☆ (6/10) — 帧动画控件

## 概述
Animation Image 类似普通 Image，但使用多张图片数组作为动画帧。

## 部件与样式
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 背景 + 图像样式 |

## 核心 API

```c
lv_obj_t *animimg = lv_animimg_create(parent);
lv_animimg_set_src(animimg, dsc_array, num);           // 设置帧数组
lv_animimg_set_src_reverse(animimg, dsc_array, num);   // 翻转播放
lv_animimg_set_duration(animimg, 1000);                // 一帧周期（ms）
lv_animimg_set_repeat_count(animimg, LV_ANIM_REPEAT_INFINITE);
lv_animimg_start(animimg);                             // 开始播放
```

### 高级控制
```c
lv_anim_t *anim = lv_animimg_get_anim(animimg);  // 获取内部动画
lv_anim_set_values(anim, start, end);             // 重写动画值
```

## 事件与按键
- 无特殊事件，不处理按键

## 关键要点
- 序列帧动画（旋转图标、加载动画、角色动画）
- 类似 GIF 但使用预编译的 C 数组，性能更好
- 翻转播放可做来回动画效果
