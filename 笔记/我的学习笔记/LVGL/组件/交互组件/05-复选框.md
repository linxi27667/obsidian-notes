# 复选框 (lv_checkbox)

> **重要性**: ★★★★☆ (9/10) — 多选项切换核心控件

## 概述
Checkbox 由勾选框和标签组成，点击时切换勾选状态。

## 部件与样式
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 背景 + 文本 |
| `LV_PART_INDICATOR` | 勾选框（尺寸等于字体高度） |

## 状态控制
```c
lv_obj_add_state(cb, LV_STATE_CHECKED);     // 勾选
lv_obj_remove_state(cb, LV_STATE_CHECKED);  // 取消
lv_obj_has_state(cb, LV_STATE_CHECKED);     // 查询
```

## 文本设置
```c
lv_checkbox_set_text(cb, "New text");        // 动态分配
lv_checkbox_set_text_static(cb, txt);        // 静态缓冲区
```

## RadioButton 模式
```c
lv_obj_set_radio_button(obj, true);  // 启用单选行为（同组互斥）
lv_obj_set_style_radius(obj, LV_RADIUS_CIRCLE, 0);  // 圆形指示器
```

## 事件

| 事件 | 触发 |
|------|------|
| `LV_EVENT_VALUE_CHANGED` | 切换勾选状态时 |

## 按键
- `LV_KEY_RIGHT/UP` — 勾选
- `LV_KEY_LEFT/DOWN` — 取消
- `LV_KEY_ENTER` — 切换

## 关键要点
- 多选场景首选
- 设置 RadioButton 模式可做互斥单选
- 自动加入默认组
