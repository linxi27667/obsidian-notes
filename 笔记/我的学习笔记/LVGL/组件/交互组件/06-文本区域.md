# 文本区域 (lv_textarea)

> **重要性**: ★★★★☆ (9/10) — 文本输入核心控件

## 概述
Text Area = lv_obj + Label + 光标，支持单行模式和密码模式。

## 部件与样式
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 背景 + 文本属性 |
| `LV_PART_SCROLLBAR` | 溢出时滚动条 |
| `LV_PART_SELECTED` | 选中文本样式 |
| `LV_PART_CURSOR` | 光标样式 |
| `LV_PART_TEXTAREA_PLACEHOLDER` | 占位提示文本 |

## 光标样式
- **块光标**: 设置 bg_color + bg_opa
- **条光标**: 留透明 + 设置左边框

## 核心 API

### 文本管理
| 函数 | 说明 |
|------|------|
| `lv_textarea_add_char(ta, 'c')` | 在光标处插入字符 |
| `lv_textarea_add_text(ta, "text")` | 插入字符串（支持CJK） |
| `lv_textarea_set_text(ta, "text")` | 替换全部文本 |
| `lv_textarea_delete_char(ta)` | 删除光标左侧字符 |

### 配置
| 函数 | 说明 |
|------|------|
| `lv_textarea_set_one_line(ta, true)` | 单行模式 |
| `lv_textarea_set_password_mode(ta, true)` | 密码模式 |
| `lv_textarea_set_password_bullet(ta, "x")` | 密码掩码字符 |
| `lv_textarea_set_placeholder_text(ta, "Hint")` | 空时占位提示 |
| `lv_textarea_set_accepted_chars(ta, "0123456789")` | 限制输入字符 |
| `lv_textarea_set_max_length(ta, num)` | 最大字符数 |

## 事件

| 事件 | 说明 |
|------|------|
| `LV_EVENT_INSERT` | 插入前，可修改或取消插入 |
| `LV_EVENT_VALUE_CHANGED` | 内容变化 |
| `LV_EVENT_READY` | 单行模式下按回车 |

## 性能
- 超长文本 (>20k字符): 启用 `LV_LABEL_LONG_TXT_HINT`
- 文本选择功能: 每标签增加 8 字节

## 关键要点
- 文本输入、密码输入、搜索框等场景的首选控件
- 常与 Keyboard 配合使用
- INSERT 事件可做输入过滤/格式化
