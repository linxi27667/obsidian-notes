# 滑块 (lv_slider)

> **重要性**: ★★★★☆ (9/10) — 数值选择核心控件

## 概述
Slider 类似 Bar，增加了可拖拽的旋钮来设置值。

## 部件与样式
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 背景 |
| `LV_PART_INDICATOR` | 当前值指示 |
| `LV_PART_KNOB` | 拖拽旋钮（默认圆形） |

## 默认属性
- 值: 0 | 范围: [0..100] | 方向: 水平

## 模式

| 模式 | 说明 |
|------|------|
| `LV_SLIDER_MODE_NORMAL` | 标准滑块 |
| `LV_SLIDER_SYMMETRICAL` | 从零点对称绘制 |
| `LV_SLIDER_MODE_RANGE` | 范围模式，可设起止 |

## 旋钮专属模式
```c
lv_obj_add_flag(slider, LV_OBJ_FLAG_ADV_HITTEST);  // 仅旋钮可拖拽
lv_obj_set_ext_click_area(slider, value);           // 扩大点击区域
```

## 数据绑定
```c
lv_slider_bind_value(slider, &subject);  // 双向绑定，支持 int/float
```

## 事件

| 事件 | 说明 |
|------|------|
| `LV_EVENT_VALUE_CHANGED` | 拖拽时持续触发 |
| `LV_EVENT_RELEASED` | 释放时触发一次 |

## 按键
- `LV_KEY_UP/RIGHT` — +1
- `LV_KEY_DOWN/LEFT` — -1

## 关键要点
- 音量、亮度、温度等连续数值选择首选控件
- 持续触发 VALUE_CHANGED 事件，注意节流
- Range 模式可做时间区间选择
