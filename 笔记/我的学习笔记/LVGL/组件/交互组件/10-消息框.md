# 消息框 (lv_msgbox)

> **重要性**: ★★★☆☆ (7/10) — 弹窗对话框核心控件

## 概述
Message Box 作为弹窗对话框使用，由内容区、可选头部（标题/关闭按钮）和可选底部（按钮）组成。

## 组成
- 内容/头部/底部: lv_obj
- 按钮: lv_button
- 标题/内容文本: lv_label

## 核心 API

```c
lv_obj_t *msgbox = lv_msgbox_create(parent);  // NULL 父级 = 模态对话框
lv_msgbox_add_title(msgbox, "标题");
lv_msgbox_add_text(msgbox, "内容");
lv_msgbox_add_close_button(msgbox);
lv_obj_t *btn = lv_msgbox_add_footer_button(msgbox, "按钮文本");
lv_msgbox_close(msgbox);              // 立即关闭
lv_msgbox_close_async(msgbox);        // 异步关闭
```

### 获取部分
```c
lv_msgbox_get_content(msgbox);   // 内容容器
lv_msgbox_get_title(msgbox);     // 标题
lv_msgbox_get_header(msgbox);    // 头部
lv_msgbox_get_footer(msgbox);    // 底部
```

## 模态对话框
- 父级为 NULL 时，使用 Top Layer 创建模态对话框
- 模态时阻止对下方控件的点击

## 事件与按键
- 无特殊事件，不处理按键
- 使用按钮自身的事件

## 关键要点
- 确认/取消对话框、设置面板、警告提示
- 模态对话框注意关闭逻辑
- 可添加背景模糊效果提升体验
