# 弧形 (lv_arc)

> **重要性**: ★★★★☆ (8/10) — 仪表盘、旋钮、圆形进度条必备

## 概述
Arc 由背景弧和前景弧组成，前景弧可触摸调节。

## 部件与样式
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 背景 + 弧样式，padding 影响弧大小 |
| `LV_PART_INDICATOR` | 前景弧，padding 相对背景弧 |
| `LV_PART_KNOB` | 弧末端的手柄 |

## 核心 API

### 值与范围
```c
lv_arc_set_value(arc, val);         // 设置值
lv_arc_set_range(arc, min, max);    // 设置范围（默认 0..100）
```

### 角度控制
```c
lv_arc_set_bg_start_angle(arc, angle);   // 背景起始角度
lv_arc_set_bg_end_angle(arc, angle);     // 背景结束角度
lv_arc_set_bg_angles(arc, start, end);   // 同时设置
lv_arc_set_rotation(arc, deg);           // 整体旋转偏移
```

### 模式

| 模式 | 说明 |
|------|------|
| `LV_ARC_MODE_NORMAL` | 顺时针从最小到当前值 |
| `LV_ARC_MODE_REVERSE` | 逆时针从最大到当前值 |
| `LV_ARC_MODE_SYMMETRICAL` | 从中点到当前值 |

### 高级功能
```c
lv_arc_set_change_rate(arc, rate);           // 限制变化速率（度/秒）
lv_arc_set_knob_offset(arc, offset);         // 旋钮偏移角度
lv_arc_set_mode(arc, LV_ARC_MODE_...);       // 设置模式
```

### 跟随对象
```c
lv_arc_align_obj_to_angle(arc, obj, radius_offset);   // 对象跟随角度
lv_arc_rotate_obj_to_angle(arc, obj, radius_offset);  // 对象旋转到角度
```

### 数据绑定
```c
lv_arc_bind_value(arc, &subject);  // 双向绑定，支持 int/float
```

## 事件
- `LV_EVENT_VALUE_CHANGED` — 按下/拖拽到新值时

## 按键
- `LV_KEY_RIGHT/UP` — +1
- `LV_KEY_LEFT/DOWN` — -1

## 禁用交互
```c
lv_obj_remove_style(arc, NULL, LV_PART_KNOB);
lv_obj_remove_flag(arc, LV_OBJ_FLAG_CLICKABLE);
```

## 关键要点
- 音量旋钮、仪表盘、饼图的首选控件
- 角度和值设置互斥，只能选一种方式
- ADV_HITTEST 模式可限制仅在弧线上响应点击
