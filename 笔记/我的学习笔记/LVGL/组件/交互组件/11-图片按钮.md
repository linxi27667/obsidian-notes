# 图片按钮 (lv_imagebutton)

> **重要性**: ★★★☆☆ (7/10) — 图片驱动的按钮

## 概述
Image Button 类似普通按钮，但使用用户定义的图片替代矩形外观。

## 部件与样式
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 图片（背景样式可做矩形底） |

## 状态
```c
LV_IMAGEBUTTON_STATE_RELEASED          // 释放
LV_IMAGEBUTTON_STATE_PRESSED           // 按下
LV_IMAGEBUTTON_STATE_DISABLED          // 禁用
LV_IMAGEBUTTON_STATE_CHECKED_RELEASED  // 已选中-释放
LV_IMAGEBUTTON_STATE_CHECKED_PRESSED   // 已选中-按下
LV_IMAGEBUTTON_STATE_CHECKED_DISABLED  // 已选中-禁用
```

## 核心 API

```c
lv_imagebutton_set_src(imgbtn, state, src_left, src_mid, src_right);
lv_imagebutton_set_state(imgbtn, state);  // 程序设置状态
```

### 图片来源
- 左/中/右三段图片（可 NULL）
- 仅设置 mid 时自动按图片宽度
- 三段都设时需手动设置宽度，中间平铺填充

## 事件
- `LV_EVENT_VALUE_CHANGED` — 启用 CHECKABLE 时切换状态

## 按键
- `LV_KEY_RIGHT/UP` — 选中（CHECKABLE时）
- `LV_KEY_LEFT/DOWN` — 取消
- `LV_KEY_ENTER` — 点击

## 关键要点
- 自定义外观按钮（播放/暂停、图标按钮等）
- 三段式设计可做圆角/渐变按钮
- 不支持符号 (Symbol)
