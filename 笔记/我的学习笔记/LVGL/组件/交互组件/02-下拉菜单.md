# 下拉菜单 (lv_dropdown)

> **重要性**: ★★★★☆ (9/10) — 选项选择核心控件

## 概述
下拉列表允许用户从列表中选择一项。关闭时显示单个值，点击展开列表。

## 部件与样式

### 按钮部分
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 按钮背景 + 文本 |
| `LV_PART_INDICATOR` | 箭头符号（图片或文本） |

### 列表部分
- `LV_PART_MAIN` — 列表本身
- `LV_PART_SCROLLBAR` — 滚动条
- `LV_PART_SELECTED` — 当前选中项

按钮打开时进入 `LV_STATE_CHECKED` 状态。

## 核心 API

| 函数 | 说明 |
|------|------|
| `lv_dropdown_set_options(dd, "A\nB\nC")` | 设置选项（\n 分隔） |
| `lv_dropdown_set_options_static(dd, opts)` | 静态字符串（不复制） |
| `lv_dropdown_add_option(dd, "New", pos)` | 插入新选项 |
| `lv_dropdown_set_selected(dd, id)` | 程序化选中 |
| `lv_dropdown_get_selected(dd)` | 获取选中索引 |
| `lv_dropdown_get_selected_str(dd, buf, size)` | 获取选中文本 |
| `lv_dropdown_set_dir(dd, LV_DIR_*)` | 展开方向 |
| `lv_dropdown_set_symbol(dd, symbol)` | 箭头图标 |
| `lv_dropdown_set_text(dd, "固定文本")` | 固定显示文本 |
| `lv_dropdown_open(dd)` / `lv_dropdown_close(dd)` | 打开/关闭 |

## 数据绑定
```c
lv_dropdown_bind_value(dd, &subject);  // 双向绑定，仅支持整数
```

## 事件

| 事件 | 触发 |
|------|------|
| `LV_EVENT_VALUE_CHANGED` | 新选项选中或列表打开/关闭 |
| `LV_EVENT_CANCEL` | 列表关闭 |
| `LV_EVENT_READY` | 列表打开 |

## 按键
- `LV_KEY_RIGHT/DOWN` — 下一个
- `LV_KEY_LEFT/UP` — 上一个
- `LV_KEY_ENTER` — 确认并关闭

## 关键要点
- 多选项场景替代多个 RadioButton，节省空间
- 列表可自定义展开方向
- 支持键盘/编码器导航
