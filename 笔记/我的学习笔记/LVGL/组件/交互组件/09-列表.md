# 列表 (lv_list)

> **重要性**: ★★★★☆ (8/10) — 菜单/设置项列表核心控件

## 概述
List 是带垂直布局的矩形容器，可添加按钮和文本。

## 部件与样式
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 背景 |
| `LV_PART_SCROLLBAR` | 滚动条 |

内部按钮和文本使用 Button/Label 的样式。

## 核心 API

```c
lv_obj_t *list = lv_list_create(parent);
lv_list_add_button(list, icon, text);   // 添加带图标的按钮
lv_list_add_text(list, text);           // 添加文本（返回 Label 指针）
lv_list_get_button_text(list, btn);     // 获取按钮文本
```

## 事件
- 无特殊事件，按钮正常发送事件

## 按键
- 不处理按键输入

## 使用场景
- 设置菜单、文件列表、导航列表
- 按钮默认全宽，带水平文本滚动

## 关键要点
- 轻量级列表容器，内部按钮可自定义
- 适合嵌入式场景，内存友好
