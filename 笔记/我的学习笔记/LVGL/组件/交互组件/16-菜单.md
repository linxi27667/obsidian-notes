# 菜单 (lv_menu)

> **重要性**: ★★☆☆☆ (6/10) — 多级菜单导航

## 概述
Menu 创建多层级菜单，自动处理层级间的导航。

## 核心 API

```c
lv_obj_t *menu = lv_menu_create(parent);              // 创建菜单
lv_obj_t *page = lv_menu_page_create(menu, "标题");    // 创建页面
lv_menu_set_page(menu, page);                          // 显示页面
lv_menu_set_load_page_event(menu, obj, page);          // 点击对象加载页面
lv_menu_cont_create(page);                             // 创建容器
lv_menu_section_create(page);                          // 创建分组
lv_menu_separator_create(page);                        // 创建分隔线
```

### 头部模式
```c
lv_menu_set_mode_header(menu, LV_MENU_HEADER_TOP_FIXED);          // 固定头部
lv_menu_set_mode_root_back_button(menu, LV_MENU_ROOT_BACK_BTN_ENABLED); // 根返回按钮
```

### 查询
```c
lv_menu_get_cur_main_page(menu);        // 当前主页面
lv_menu_get_cur_sidebar_page(menu);     // 当前侧栏页面
lv_menu_back_button_is_root(menu, btn); // 是否为根级返回
```

## 事件
- `LV_EVENT_VALUE_CHANGED` — 页面切换时
- `LV_EVENT_CLICKED` — 返回按钮点击（带 LV_OBJ_FLAG_EVENT_BUBBLE）

## 关键要点
- 适合设置菜单、文件浏览器等多级导航
- 支持侧栏导航模式
- 可自定义头部位置和返回按钮
