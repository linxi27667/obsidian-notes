# 按钮矩阵 (lv_buttonmatrix)

> **重要性**: ★★★☆☆ (7/10) — 轻量多按钮布局

## 概述
按钮矩阵在行/列中显示多个按钮，按钮不是真实创建而是虚拟绘制。每个按钮仅占用 8 字节。

## 部件与样式
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 背景 |
| `LV_PART_ITEMS` | 按钮 |

## 核心 API

### 按钮映射
```c
const char *map[] = {"btn1", "btn2", "\n", "btn3", ""};  // \n 换行，"" 结尾
lv_buttonmatrix_set_map(btnm, map);
```

### 按钮控制标志

| 标志 | 说明 |
|------|------|
| `LV_BUTTONMATRIX_CTRL_HIDDEN` | 隐藏但保留空间 |
| `LV_BUTTONMATRIX_CTRL_DISABLED` | 禁用 |
| `LV_BUTTONMATRIX_CTRL_CHECKABLE` | 可切换 |
| `LV_BUTTONMATRIX_CTRL_CLICK_TRIG` | 点击触发（非按下） |
| `LV_BUTTONMATRIX_CTRL_RECOLOR` | 文本重着色 |
| `LV_BUTTONMATRIX_CTRL_POPOVER` | 按下时气泡显示 |

### 其他 API
```c
lv_buttonmatrix_set_button_width(btnm, id, width);     // 按钮宽度（1-15）
lv_buttonmatrix_set_one_checked(btnm, true);           // 单选模式
lv_buttonmatrix_get_selected_button(btnm);             // 获取选中按钮ID
lv_buttonmatrix_get_button_text(btnm, id);             // 获取按钮文本
```

## 事件
- `LV_EVENT_VALUE_CHANGED` — 按钮按下/释放/重复

## 按键
- 方向键导航
- `LV_KEY_ENTER` 按下/释放

## 关键要点
- 内存高效: 每按钮 8 字节 vs 正常按钮 ~200 字节
- 数字键盘、计算器、分页控件
- 自定义绘制可实现图标按钮
