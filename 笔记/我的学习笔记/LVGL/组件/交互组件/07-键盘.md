# 键盘 (lv_keyboard)

> **重要性**: ★★★★☆ (9/10) — 触摸设备文本输入必备

## 概述
Keyboard 是特殊的 Button Matrix，带有预定义键位布局，用于向 Text Area 输入文本。

## 部件与样式
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 背景 |
| `LV_PART_ITEMS` | 按键 |

## 模式

| 模式 | 说明 |
|------|------|
| `LV_KEYBOARD_MODE_TEXT_LOWER` | 小写字母 |
| `LV_KEYBOARD_MODE_TEXT_UPPER` | 大写字母（默认） |
| `LV_KEYBOARD_MODE_SPECIAL` | 特殊字符 |
| `LV_KEYBOARD_MODE_NUMBER` | 数字 |
| `LV_KEYBOARD_MODE_USER_1~4` | 自定义布局 |

## 核心 API

```c
lv_keyboard_set_textarea(kb, ta);      // 绑定文本输入区
lv_keyboard_set_mode(kb, mode);        // 切换模式
lv_keyboard_set_popovers(kb, true);    // 按键气泡效果
lv_keyboard_set_map(kb, mode, map, ctrl); // 自定义键位
```

## 特殊键关键字
| 关键字 | 动作 |
|--------|------|
| `LV_SYMBOL_OK` | 发送 `LV_EVENT_READY` |
| `LV_SYMBOL_CLOSE` | 发送 `LV_EVENT_CANCEL` |
| `LV_SYMBOL_BACKSPACE` | 删除 |
| `LV_SYMBOL_LEFT/RIGHT` | 移动光标 |
| `"ABC"` / `"abc"` | 切换大小写布局 |
| `"1#"` | 切换数字布局 |

## 事件

| 事件 | 触发 |
|------|------|
| `LV_EVENT_VALUE_CHANGED` | 按键按下 |
| `LV_EVENT_READY` | 确认键点击 |
| `LV_EVENT_CANCEL` | 关闭键点击 |

## 关键要点
- 与 Text Area 搭配使用实现触摸输入
- 聚焦/失焦事件控制键盘显示隐藏
- 自定义键位可做数字键盘、计算器等
