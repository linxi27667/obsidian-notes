# 开关 (lv_switch)

> **重要性**: ★★★★☆ (9/10) — 二值切换最常用控件

## 概述
Switch 控件像小滑块，用于显示和修改 "开/关" 状态。

## 部件与样式
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 背景 |
| `LV_PART_INDICATOR` | 指示当前状态 |
| `LV_PART_KNOB` | 滑块（默认圆形） |

## 方向
- 默认水平
- width < height 时自动垂直
- 可用 `lv_switch_set_orientation()` 显式设置

## 状态控制
```c
lv_obj_has_state(sw, LV_STATE_CHECKED)  // 当前状态
lv_obj_add_state(sw, LV_STATE_CHECKED)   // 开启
lv_obj_remove_state(sw, LV_STATE_CHECKED) // 关闭
```

## 事件

| 事件 | 触发 |
|------|------|
| `LV_EVENT_VALUE_CHANGED` | 状态切换时 |

## 按键
- `LV_KEY_UP/RIGHT` — 开
- `LV_KEY_DOWN/LEFT` — 关
- `LV_KEY_ENTER` — 切换

## 关键要点
- WiFi开关、蓝牙开关、设置项切换首选
- 比 Checkbox 更直观
- 支持编码器导航
