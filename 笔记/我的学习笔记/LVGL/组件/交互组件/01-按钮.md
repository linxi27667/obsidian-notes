# 按钮 (lv_button)

> **重要性**: ★★★★★ (10/10) — 最基础的交互控件

## 概述
Button 是基础的可点击交互控件。相比 Base Widget 没有新增功能，主要用于语义目的。

## 与 lv_obj 的默认差异
| 特性 | lv_button | lv_obj |
|------|-----------|--------|
| 可滚动 | 否 | 是 |
| 自动加入组 | 是 | 否 |
| 默认尺寸 | `LV_SIZE_CONTENT` | 需手动设置 |

## 部件与样式
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 按钮背景，使用典型背景样式属性 |

## 事件

| 事件 | 触发条件 |
|------|---------|
| `LV_EVENT_CLICKED` | 点击时触发 |
| `LV_EVENT_PRESSED` | 按下时触发 |
| `LV_EVENT_RELEASED` | 释放时触发 |
| `LV_EVENT_VALUE_CHANGED` | 启用 `LV_OBJ_FLAG_CHECKABLE` 时切换状态触发 |

## 按键处理
`LV_KEY_ENTER` → 转换为 `LV_EVENT_PRESSED` / `LV_EVENT_PRESSING` / `LV_EVENT_RELEASED` 等

## 代码示例

### 简单按钮
```c
void event_handler(lv_event_t * e) {
    lv_event_code_t code = lv_event_get_code(e);
    if(code == LV_EVENT_CLICKED) LV_LOG_USER("Clicked");
}

lv_obj_t * btn = lv_button_create(lv_screen_active());
lv_obj_add_event_cb(btn, event_handler, LV_EVENT_ALL, NULL);
```

### 切换按钮 (Toggle)
```c
lv_obj_t * btn = lv_button_create(lv_screen_active());
lv_obj_add_flag(btn, LV_OBJ_FLAG_CHECKABLE);  // 启用切换
```

### 弹性动画按钮
```c
static lv_style_prop_t props[] = {LV_STYLE_TRANSFORM_WIDTH, LV_STYLE_TRANSFORM_HEIGHT, 0};
static lv_style_transition_dsc_t trans;
lv_style_transition_dsc_init(&trans, props, lv_anim_path_overshoot, 250, 100, NULL);
```

## 关键要点
- 本质是语义化的 lv_obj
- 配合 `LV_OBJ_FLAG_CHECKABLE` 可做开关按钮
- 样式过渡动画是提升 UX 的关键
