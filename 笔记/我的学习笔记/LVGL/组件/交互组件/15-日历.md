# 日历 (lv_calendar)

> **重要性**: ★★★☆☆ (7/10) — 日期选择专用控件

## 概述
Calendar 以 7×7 矩阵显示天数，显示星期名称、今天高亮、自定义日期高亮。

## 部件与样式
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 背景 |
| `LV_PART_ITEMS` | 日期和星期名称 |

## 核心 API

```c
lv_calendar_set_today_date(cal, year, month, day);      // 设置今天
lv_calendar_set_month_shown(cal, year, month);           // 显示月份
lv_calendar_set_highlighted_dates(cal, dates, count);   // 高亮日期
lv_calendar_get_pressed_date(cal, &date);                // 获取点击的日期
```

### 日期结构
```c
lv_calendar_date_t date = { .year = 2026, .month = 2, .day = 14 };
```

## 导航头部
```c
lv_calendar_add_header_arrow(cal);       // 箭头导航（上一月/下一月）
lv_calendar_add_header_dropdown(cal);    // 下拉选择年月
```

## 中国日历
```c
lv_calendar_set_chinese_mode(cal, true);  // 启用农历/节气/节日
```

## 事件
- `LV_EVENT_VALUE_CHANGED` — 日期点击时

## 按键
- 方向键导航
- `LV_KEY_ENTER` 确认

## 关键要点
- 日期选择、日程展示
- 支持农历和节气显示
- 高亮日期数组必须是静态/全局的
