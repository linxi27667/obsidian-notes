# 拼音输入法 (lv_ime_pinyin)

> **重要性**: ★☆☆☆☆ (5/10) — 中文拼音输入插件

## 概述
Pinyin IME 为 Keyboard 控件提供中文拼音输入支持，兼容 26 键和 9 键模式。

## 组件
- Pinyin IME 对象（核心插件）
- Keyboard（附加的键盘控件）
- 候选面板（显示候选汉字）
- 字典（内置或自定义）

## 核心 API

```c
lv_obj_t *pinyin_ime = lv_ime_pinyin_create(parent);
lv_ime_pinyin_set_keyboard(pinyin_ime, kb);     // 绑定键盘
lv_ime_pinyin_set_dict(pinyin_ime, dict);       // 设置自定义字典
lv_ime_pinyin_set_mode(pinyin_ime, mode);       // 设置输入模式
lv_obj_t *cand = lv_ime_pinyin_get_cand_panel(pinyin_ime);  // 获取候选面板
```

### 输入模式
| 模式 | 说明 |
|------|------|
| `LV_IME_PINYIN_MODE_K26` | 26键模式（默认） |
| `LV_IME_PINYIN_MODE_K9` | 9键模式 |
| `LV_IME_PINYIN_MODE_K9_NUMBER` | 数字键盘模式 |

## 配置 (lv_conf.h)
```c
#define LV_USE_IME_PINYIN 1
#define LV_IME_PINYIN_USE_DEFAULT_DICT 1
#define LV_IME_PINYIN_USE_K9_MODE 1
```

## 自定义字典格式
```c
lv_pinyin_dict_t dict[] = {
    { "a", "啊阿呵吖" },
    { "ai", "埃挨哎唉" },
    {NULL, NULL}  // 必须以 NULL 结尾
};
```

## 关键要点
- 中文输入场景必备
- 内置字典约1000个常用汉字
- 生产环境建议用自定义字体+字典
- 字典拼音顺序必须遵循汉语拼音音节表
