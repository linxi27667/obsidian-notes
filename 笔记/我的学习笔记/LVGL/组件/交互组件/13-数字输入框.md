# 数字输入框 (lv_spinbox)

> **重要性**: ★★★☆☆ (7/10) — 数值输入专用控件

## 概述
Spinbox 是 Text Area 的扩展，专门用于查看和修改有约束的整数值。

## 部件与样式
与 Text Area 相同。

## 核心 API

```c
lv_spinbox_set_value(spinbox, 1234);             // 设置值
lv_spinbox_set_range(spinbox, -1000, 2500);     // 设置范围
lv_spinbox_set_step(spinbox, 100);               // 步长（10的倍数）
lv_spinbox_set_cursor_pos(spinbox, pos);         // 设置光标位置（0=最低位）
lv_spinbox_increment(spinbox);                   // 增加
lv_spinbox_decrement(spinbox);                   // 减少
```

### 格式
```c
lv_spinbox_set_digit_format(spinbox, digit_count, decimal_pos);
// digit_count=总位数, decimal_pos=小数点前位数(0=无小数点)
```

### 回绕
```c
lv_spinbox_set_rollover(spinbox, true);  // 达到极限时绕回
```

## 数据绑定
```c
lv_spinbox_bind_value(spinbox, &subject);  // 双向绑定，仅支持整数
```

## 事件
- `LV_EVENT_VALUE_CHANGED` — 值变化时

## 按键
- `LV_KEY_LEFT/RIGHT` — 移动光标
- `LV_KEY_UP/DOWN` — 增减数值
- `LV_KEY_ENTER` — 编码器移动数位

## 关键要点
- 温度设置、时间设置、参数调节
- 比 Text Area 更适合纯数字输入
- 回绕模式可做循环计数器
