# 线条 (lv_line)

> **重要性**: ★★★☆☆ (7/10) — 简单线条绘制

## 概述
Line 在点集之间绘制直线。

## 部件与样式
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 背景和线条样式属性 |

## 核心 API

```c
lv_point_precise_t points[] = { {5, 5}, {70, 70}, {120, 10} };
lv_line_set_points(line, points, count);  // 设置点集
lv_line_set_y_invert(line, true);          // 翻转Y轴（默认Y=0在顶部）
```

## 自动尺寸
- 默认宽度/高度为 `LV_SIZE_CONTENT`
- 自动适应所有点的包围盒
- 显式设置尺寸可能遮挡线条

## 事件与按键
- 无特殊事件，不处理按键

## 关键要点
- 简单折线、波形图、路径可视化
- 支持百分比坐标 `lv_pct(x)`
- Y轴翻转适合笛卡尔坐标系
