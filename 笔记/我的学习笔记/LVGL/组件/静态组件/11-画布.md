# 画布 (lv_canvas)

> **重要性**: ★★★☆☆ (7/10) — 自定义绘制万能控件

## 概述
Canvas 继承自 Image，扩展后可绘制任意内容：矩形、文本、图片、线条、弧线等。

## 部件与样式
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 背景和图像样式属性 |

## 缓冲区管理

```c
// 分配缓冲区
uint8_t *buf = lv_malloc(w * h * 4);  // ARGB8888
// 或静态
static uint8_t buf[w * h * 4];

lv_canvas_set_buffer(canvas, buf, w, h, LV_COLOR_FORMAT_...);
```

### 颜色格式
- `LV_COLOR_FORMAT_ARGB8888` — 全功能（推荐）
- `LV_COLOR_FORMAT_RGB565` — 省内存
- `LV_COLOR_FORMAT_I1/2/4/8` — 索引色，需设置调色板

### 索引色
```c
lv_canvas_set_palette(canvas, index, color);  // 设置调色板索引
```

## 像素操作
```c
lv_canvas_set_px(canvas, x, y, color, opa);   // 设置像素
lv_canvas_fill_bg(canvas, color, opa);         // 填充背景
```

## 图层绘制 API

```c
lv_layer_t layer;
lv_canvas_init_layer(canvas, &layer);

lv_draw_rect(&layer, &rect_dsc, &coords);      // 矩形
lv_draw_label(&layer, &label_dsc, &coords);    // 文本
lv_draw_arc(&layer, &arc_dsc);                  // 弧线
lv_draw_image(&layer, &img_dsc, &coords);      // 图片
lv_draw_line(&layer, &line_dsc);                // 线条
lv_draw_triangle(&layer, &tri_dsc);             // 三角形
lv_draw_blur(&layer, &blur_dsc, &area);        // 模糊

lv_canvas_finish_layer(canvas, &layer);
```

## 事件与按键
- 无特殊事件，不处理按键

## 关键要点
- 自定义绘制的终极武器
- 批量像素操作时禁用 invalidation 提升性能
- malloc 缓冲区需在 `LV_EVENT_DELETE` 中释放
- 支持模糊、矢量图形等高级效果
