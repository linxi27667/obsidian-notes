# 旋转器 (lv_roller)

> **重要性**: ★★★☆☆ (7/10) — 列表滚动选择专用控件

## 概述
Roller 允许用户通过滚动从列表中选择项目。选中项居中显示，样式独立。

## 部件与样式
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 背景 + 文本 |
| `LV_PART_SELECTED` | 中间选中项 |

## 关键样式
- `text_line_space` — 项目间距
- `anim_duration` — 自动滚动动画时间

## 核心 API

```c
lv_roller_set_options(roller, "A\nB\nC", LV_ROLLER_MODE_NORMAL);  // 设置选项
lv_roller_set_options(roller, "A\nB\nC", LV_ROLLER_MODE_INFINITE); // 无限循环
lv_roller_set_selected(roller, index, LV_ANIM_ON);     // 程序选中
lv_roller_set_selected_str(roller, "text", LV_ANIM_ON); // 按文本选中
lv_roller_get_selected(roller);                         // 获取索引
lv_roller_set_visible_row_count(roller, count);         // 可见行数
```

## 数据绑定
```c
lv_roller_bind_value(roller, &subject);  // 双向绑定，仅支持整数
```

## 事件
- `LV_EVENT_VALUE_CHANGED` — 新项选中时

## 按键
- `LV_KEY_RIGHT/DOWN` — 下一个
- `LV_KEY_LEFT/UP` — 上一个
- `LV_KEY_ENTER` — 确认

## 关键要点
- 日期/时间选择、参数调节的优雅控件
- INFINITE 模式可做无限循环滚轮
- 比 Dropdown 更省空间
