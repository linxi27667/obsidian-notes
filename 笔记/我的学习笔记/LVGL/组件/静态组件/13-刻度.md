# 刻度 (lv_scale)

> **重要性**: ★★★☆☆ (7/10) — 仪表盘刻度专用控件

## 概述
Scale 显示线性或圆形刻度，支持自定义范围、刻度数、标签和自定义样式区间。

## 部件与样式
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 主线（水平/圆弧） |
| `LV_PART_ITEMS` | 小刻度线 |
| `LV_PART_INDICATOR` | 大刻度线 + 标签 |

## 模式

| 模式 | 说明 |
|------|------|
| `LV_SCALE_MODE_HORIZONTAL_TOP` | 水平，刻度在上 |
| `LV_SCALE_MODE_HORIZONTAL_BOTTOM` | 水平，刻度在下（默认） |
| `LV_SCALE_MODE_VERTICAL_LEFT` | 垂直，刻度在左 |
| `LV_SCALE_MODE_VERTICAL_RIGHT` | 垂直，刻度在右 |
| `LV_SCALE_MODE_ROUND_INNER` | 圆形，刻度在内 |
| `LV_SCALE_MODE_ROUND_OUTER` | 圆形，刻度在外 |

## 核心 API

```c
lv_scale_set_range(scale, min, max);           // 数值范围
lv_scale_set_angle_range(scale, angle);        // 角度范围（默认270°）
lv_scale_set_total_tick_count(scale, count);   // 总刻度数
lv_scale_set_major_tick_every(scale, nth);     // 大刻度间隔
lv_scale_set_label_show(scale, true);          // 显示/隐藏标签
lv_scale_set_text_src(scale, labels);          // 自定义标签
```

### 指针
```c
lv_scale_set_line_needle_value(scale, line, len, val);   // 线条指针
lv_scale_set_image_needle_value(scale, img, val);        // 图片指针
```

### 区间 (Sections)
```c
lv_scale_section_t *sec = lv_scale_add_section(scale);
lv_scale_section_set_range(sec, min, max);
lv_scale_section_set_style(sec, LV_PART_ITEMS, &style);
```

## 事件与按键
- 无特殊事件，不处理按键

## 关键要点
- 仪表盘、温度计、速度表等场景
- 区间可做颜色分区（如温度过高红色）
- 支持自定义标签数组
