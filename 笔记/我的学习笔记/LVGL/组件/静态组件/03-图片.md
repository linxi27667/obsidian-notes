# 图片 (lv_image)

> **重要性**: ★★★★★ (10/10) — 图像展示核心控件

## 概述
Image 控件用于显示图片，支持从 C 数组或外部文件加载，也支持符号 (Symbols)。

## 图片源 (Image Sources)
| 类型 | 说明 |
|------|------|
| C 数组 | 代码中编译的图片数据 |
| 外部文件 | 存储在 SD 卡等外部介质 |
| 符号 | `LV_SYMBOL_...` 宏定义 |

```c
LV_IMAGE_DECLARE(converted_img_var);  // 声明转换后的图片变量
lv_image_set_src(img, src);            // 设置源
```

## 部件与样式
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 背景矩形 + 图片样式属性 |

## 变换属性

| 功能 | API | 说明 |
|------|-----|------|
| 重着色 | `lv_obj_set_style_image_recolor()` | 混合颜色到像素 |
| 偏移 | `lv_image_set_offset_x/y(img, offset)` | 用于纹理图集或跑马灯 |
| 缩放 | `lv_image_set_scale(img, factor)` | 256=原始, 512=2x, 128=0.5x |
| 旋转 | `lv_image_set_rotation(img, angle_x10)` | 度数×10 |
| 旋转中心 | `lv_image_set_pivot(img, x, y)` | (0,0)=左上角 |

## 内部对齐选项
| 值 | 行为 |
|----|------|
| `LV_IMAGE_ALIGN_CENTER` | 居中 |
| `LV_IMAGE_ALIGN_STRETCH` | 拉伸填充 |
| `LV_IMAGE_ALIGN_TILE` | 平铺填充 |
| `LV_IMAGE_ALIGN_CONTAIN` | 保持比例适配 |
| `LV_IMAGE_ALIGN_COVER` | 保持比例裁剪填充 |

## 变换限制
- 索引色图片 (`I1/2/4/8`) 和 Alpha Only 图片 **不能变换**
- 变换需要 ARGB/RGB 格式且存储为 C 数组
- 变换不影响子控件

## 透明度方式
1. **Alpha 通道**: 每个像素附带透明度
2. **索引透明**: 调色板中特定索引表示透明

## 数据绑定
```c
lv_image_bind_src(img, &subject);  // 单向绑定，仅支持指针类型
```

## 事件与按键
- 无特殊事件
- 添加 `LV_OBJ_FLAG_CLICKABLE` 后可点击
- 不处理按键

## 关键要点
- BMP 无压缩，CPU占用少但内存大
- PNG 无损压缩，内存小但需要解压缩
- SVG 矢量格式，无损缩放
- 变换性能: ARGB8888 格式 C 数组最佳
