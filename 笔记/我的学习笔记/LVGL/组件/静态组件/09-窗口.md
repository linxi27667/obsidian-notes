# 窗口 (lv_win)

> **重要性**: ★★★☆☆ (7/10) — 带标题栏的容器

## 概述
Window 由标题栏（标题+按钮）和内容区组成。

## 组件结构
- 背景: lv_obj (flex-column)
- 标题栏: lv_obj (flex-row)
- 标题: lv_label
- 按钮: lv_button
- 内容区: lv_obj

## 核心 API

```c
lv_obj_t *win = lv_win_create(parent);
lv_win_add_title(win, "窗口标题");
lv_win_add_button(win, LV_SYMBOL_CLOSE, 40);  // 添加按钮
lv_win_get_header(win);   // 获取标题栏
lv_win_get_content(win);  // 获取内容容器
```

## 事件与按键
- 无特殊事件，不处理按键
- 按钮正常发送事件

## 关键要点
- 适合需要标题栏和关闭按钮的场景
- 内容区可添加任意子控件
- 标题使用 DOTS 模式（溢出时显示省略号）
