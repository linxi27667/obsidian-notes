# 基础对象 (lv_obj)

> **重要性**: ★★★★★ (10/10) — 所有控件的基类，几乎所有项目都会用到

## 概述
Base Widget (lv_obj) 是 LVGL 中最基础的对象类，所有其他控件都继承自它。提供核心功能如创建、布局、样式、事件处理等。

## 核心功能
- **创建**: `lv_obj_create(parent)` — 在父对象上创建基础容器
- **尺寸**: `lv_obj_set_size(obj, w, h)` / `lv_obj_set_width()` / `lv_obj_set_height()`
- **位置**: `lv_obj_set_pos(obj, x, y)` / `lv_obj_align(obj, align, x, y)`
- **自动尺寸**: `LV_SIZE_CONTENT` — 根据内容自动调整尺寸

## 布局系统
- **Flex 布局**: `lv_obj_set_flex_flow(obj, LV_FLEX_FLOW_ROW/COLUMN)` — 弹性布局
- **Grid 布局**: `lv_obj_set_grid_dsc_array(obj, col_dsc, row_dsc)` — 网格布局
- **Flag 控制**: `lv_obj_add_flag(obj, flag)` / `lv_obj_remove_flag()` — 控制行为标志

## 常用标志 (Flags)
| Flag | 说明 |
|------|------|
| `LV_OBJ_FLAG_CLICKABLE` | 可点击 |
| `LV_OBJ_FLAG_CHECKABLE` | 可切换状态 |
| `LV_OBJ_FLAG_SCROLLABLE` | 可滚动 |
| `LV_OBJ_FLAG_HIDDEN` | 隐藏 |
| `LV_OBJ_FLAG_FLOATING` | 浮动（不受布局影响） |
| `LV_OBJ_FLAG_ADV_HITTEST` | 高级点击检测 |

## 事件系统
- `lv_obj_add_event_cb(obj, callback, event_code, user_data)` — 添加事件回调
- 常用事件: `LV_EVENT_CLICKED`, `LV_EVENT_VALUE_CHANGED`, `LV_EVENT_DRAW_*`

## 样式系统
- `lv_obj_set_style_prop(obj, LV_PART_MAIN, prop, value)` — 设置样式属性
- 样式层: `0` = 本地样式, `LV_PART_MAIN` = 主体, `LV_PART_SCROLLBAR` = 滚动条

## 分组系统
- `lv_group_add_obj(group, obj)` — 加入输入设备组（键盘/编码器导航）

## 关键要点
- 所有控件都是 lv_obj 的子类
- 可以用纯 lv_obj 构建任何自定义控件
- 理解 lv_obj 是掌握 LVGL 的前提
