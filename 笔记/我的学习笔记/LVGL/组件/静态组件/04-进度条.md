# 进度条 (lv_bar)

> **重要性**: ★★★★☆ (9/10) — 进度展示最常用控件

## 概述
Bar 控件由背景和指示器组成，指示器长度相对于背景表示当前值。

## 部件与样式
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 背景，padding 影响指示器大小 |
| `LV_PART_INDICATOR` | 指示器 |

## 方向
- 水平/垂直自动根据 width/height 判断
- 也可用 `lv_bar_set_orientation()` 显式设置

## 值与范围 API

| 函数 | 说明 |
|------|------|
| `lv_bar_set_value(bar, val, LV_ANIM_ON)` | 设置当前值（可带动画） |
| `lv_bar_set_range(bar, min, max)` | 设置范围（默认 0..100） |
| `lv_bar_set_start_value(bar, val, anim)` | 设置起始值（仅范围模式） |

## 模式

| 模式 | 说明 |
|------|------|
| `LV_BAR_MODE_NORMAL` | 标准模式 |
| `LV_BAR_MODE_SYMMETRICAL` | 从零点对称绘制，需负最小正最大 |
| `LV_BAR_MODE_RANGE` | 可配置起止值 |

## 方向特性
- 默认: 左→右（水平），下→上（垂直）
- min > max 时反转方向

## 数据绑定
```c
lv_bar_bind_value(bar, &subject);  // 单向绑定，支持 int/float
```

## 事件与按键
- 无特殊事件
- 不处理按键输入

## 关键要点
- 最常用的进度/电量/音量可视化控件
- 动画过渡 (`LV_ANIM_ON`) 提升体验
- Range 模式可做温度区间等场景
