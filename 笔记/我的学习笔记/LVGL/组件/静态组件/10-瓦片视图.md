# 瓦片视图 (lv_tileview)

> **重要性**: ★★☆☆☆ (6/10) — 全屏页面滑动导航

## 概述
Tile View 是容器控件，元素（tiles）以网格排列，用户通过滑动在 tile 间导航。

## 部件与样式
基于 lv_obj，样式同基础控件。

## 核心 API

```c
// 添加 tile
lv_obj_t *tile = lv_tileview_add_tile(tileview, col, row, dir);
// dir: LV_DIR_LEFT/RIGHT/TOP/BOTTOM/HOR/VER/ALL

// 切换到 tile
lv_tileview_set_tile(tileview, tile, LV_ANIM_ON);
lv_tileview_set_tile_by_index(tileview, col, row, LV_ANIM_ON);

// 获取当前活跃 tile
lv_tileview_get_tile_active(tileview);
```

### 方向控制
每个 tile 可独立设置允许滑动的方向，如 `LV_DIR_BOTTOM` 只允许向下滑入。

## 事件
- `LV_EVENT_VALUE_CHANGED` — 新 tile 显示后触发

## 按键
- 不处理按键

## 关键要点
- 适合全屏页面滑动导航（类似智能手表界面）
- 方向限制可做引导流程
- 支持滚动链 (scroll chaining)
