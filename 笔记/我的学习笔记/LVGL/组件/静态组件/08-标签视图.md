# 标签视图 (lv_tabview)

> **重要性**: ★★★★☆ (8/10) — 多页内容组织核心控件

## 概述
Tab View 将内容组织在多个标签页中，支持点击按钮或滑动切换。

## 组件结构
- 主容器: lv_obj
- 标签按钮: lv_button
- 内容区域: lv_obj

## 核心 API

| 函数 | 说明 |
|------|------|
| `lv_tabview_create(parent)` | 创建 Tab View |
| `lv_tabview_add_tab(tabview, "名称")` | 添加标签页，返回内容容器指针 |
| `lv_tabview_set_active(tabview, id, anim)` | 程序化切换标签 |
| `lv_tabview_set_tab_bar_position(tabview, LV_DIR_*)` | 标签栏位置（上下左右） |
| `lv_tabview_set_tab_bar_size(tabview, size)` | 标签栏尺寸 |
| `lv_tabview_get_content(tabview)` | 获取内容容器 |
| `lv_tabview_get_tab_bar(tabview)` | 获取标签按钮容器 |
| `lv_tabview_get_tab_active(tabview)` | 获取当前标签索引 |
| `lv_tabview_get_tab_count(tabview)` | 获取标签数量 |
| `lv_tabview_set_tab_text(tabview, id, "名称")` | 重命名标签 |

## 事件
- `LV_EVENT_VALUE_CHANGED` — 新标签选中（滑动或点击）

## 按键
- 仅对标签按钮有效

## 关键要点
- 多页面切换导航的首选控件
- 标签栏可放上下左右任意位置
- 内容区域支持滑动切换
- 禁用 `LV_OBJ_FLAG_SCROLLABLE` 可禁止滑动切换
