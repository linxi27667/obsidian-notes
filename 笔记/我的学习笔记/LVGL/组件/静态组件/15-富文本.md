# 富文本 (lv_spangroup)

> **重要性**: ★★☆☆☆ (6/10) — 多样式文本展示

## 概述
Spangroup 用于显示富文本。与 Label 不同，可在单个控件中使用不同字体、颜色和大小的文本。

## 部件与样式
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 唯一部件 |

## 核心 API

```c
// 创建和添加 Span
lv_span_t *span = lv_spangroup_new_span(spangroup);
lv_span_set_text(span, "文本");
lv_style_set_text_color(&span->style, lv_palette_main(LV_PALETTE_RED));
lv_style_set_text_font(&span->style, &lv_font_montserrat_24);
lv_style_set_text_decor(&span->style, LV_TEXT_DECOR_UNDERLINE);

// 查询和删除
lv_spangroup_get_span_count(spangroup);
lv_spangroup_get_child(spangroup, id);  // 支持负索引
lv_spangroup_delete_span(spangroup, span);
```

### 模式（已弃用）
推荐用宽度控制：
- `LV_SIZE_CONTENT` = 根据内容扩展
- 固定宽度 = 换行

### 其他配置
```c
lv_spangroup_set_align(spangroup, LV_TEXT_ALIGN_CENTER);           // 对齐
lv_spangroup_set_overflow(spangroup, LV_SPAN_OVERFLOW_ELLIPSIS);   // 溢出省略号
lv_spangroup_set_indent(spangroup, 20);                            // 首行缩进
lv_spangroup_set_max_lines(spangroup, 10);                         // 最大行数
lv_spangroup_refr_mode(spangroup);                                 // 修改后刷新
```

### 数据绑定
```c
lv_spangroup_bind_span_text(spangroup, span, &subject, "%d");
// 注意：删除 span 前需手动 lv_observer_remove
```

## 事件与按键
- 无特殊事件，不处理按键

## 关键要点
- 混色、混字体大小的富文本展示
- 点击检测: `lv_spangroup_get_span_by_point()` 可定位点击的 span
- 数据绑定时删除 span 需手动移除 observer
