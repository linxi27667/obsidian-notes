# 弧形标签 (lv_arclabel)

> **重要性**: ★★☆☆☆ (6/10) — 弧形文本展示

## 概述
Arc Label 沿弧线显示文本，适用于仪表盘、表盘或自定义弯曲文本界面。

## 部件与样式
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 弧路径 + 沿弧渲染的文本 |

## 核心 API

```c
lv_obj_t *arclabel = lv_arclabel_create(parent);
lv_arclabel_set_text(arclabel, "文本");
lv_arclabel_set_text_static(arclabel, static_text);
lv_arclabel_set_angle_start(arclabel, angle);     // 起始角度
lv_arclabel_set_angle_size(arclabel, size);       // 弧跨度
lv_arclabel_set_dir(arclabel, LV_ARCLABEL_DIR_CLOCKWISE);      // 顺时针
lv_arclabel_set_dir(arclabel, LV_ARCLABEL_DIR_COUNTERCLOCKWISE); // 逆时针
lv_arclabel_set_radius(arclabel, radius);         // 弧半径
lv_arclabel_set_center_offset_x(arclabel, x);     // 中心偏移
lv_arclabel_set_text_vertical_align(arclabel, align);
lv_arclabel_set_text_horizontal_align(arclabel, align);
lv_arclabel_set_recolor(arclabel, true);          // 启用重着色
```

### 角度系统
- 0度 = 3点钟位置（正x轴）
- 角度顺时针增加

### 对齐方式
- `LV_ARCLABEL_TEXT_ALIGN_LEADING` — 弧起始端对齐
- `LV_ARCLABEL_TEXT_ALIGN_CENTER` — 居中
- `LV_ARCLABEL_TEXT_ALIGN_TRAILING` — 弧末端对齐

## 事件与按键
- 不发送特殊事件，默认不可点击

## 关键要点
- 仪表盘刻度标签、弧形标题
- 支持重着色 (`#FA7C45 彩色文本#`)
- 半径可用百分比 `LV_PCT(80)`
