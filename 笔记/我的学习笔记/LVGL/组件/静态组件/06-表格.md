# 表格 (lv_table)

> **重要性**: ★★★★☆ (8/10) — 数据表格展示核心控件

## 概述
Table 由行、列、单元格组成，单元格包含文本。轻量级设计——只存储文本字符串，单元格不是真实控件而是实时绘制的。

## 部件与样式
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 背景 |
| `LV_PART_ITEMS` | 单元格 |

## 核心 API

### 单元格值
```c
lv_table_set_cell_value(table, row, col, "内容");  // 支持换行符 \n
```

### 尺寸
```c
lv_table_set_row_count(table, count);              // 行数
lv_table_set_column_count(table, count);           // 列数
lv_table_set_column_width(table, col, width);      // 列宽
```

### 单元格合并
```c
lv_table_set_cell_ctrl(table, row, col, LV_TABLE_CELL_CTRL_MERGE_RIGHT);
```

### 查询
```c
lv_table_get_selected_cell(table, &row, &col);  // 获取选中单元格
```

## 事件
- `LV_EVENT_VALUE_CHANGED` — 用键盘选择新单元格时

## 按键
- 方向键选择单元格
- `LV_KEY_ENTER` 转换为按下/释放事件

## 关键要点
- 内存高效: 200行表格可轻松处理
- 单元格不是完整控件，只是绘制文本
- 自定义绘制通过 `LV_EVENT_DRAW_TASK_ADDED` 实现表头/斑马线效果
- 支持用户数据绑定，需在 `LV_EVENT_DELETE` 中释放
