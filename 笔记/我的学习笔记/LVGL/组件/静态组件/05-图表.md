# 图表 (lv_chart)

> **重要性**: ★★★★☆ (8/10) — 数据可视化核心控件

## 概述
Chart 用于可视化数据，支持折线图、柱状图、堆叠图、散点图等。

## 图表类型

| 类型 | 说明 |
|------|------|
| `LV_CHART_TYPE_LINE` | 折线图（默认） |
| `LV_CHART_TYPE_CURVE` | 贝塞尔曲线（需矢量图形） |
| `LV_CHART_TYPE_BAR` | 柱状图 |
| `LV_CHART_TYPE_STACKED` | 堆叠柱状图 |
| `LV_CHART_TYPE_SCATTER` | 散点图 |
| `LV_CHART_TYPE_NONE` | 隐藏数据 |

## 部件与样式
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 背景 + 分隔线 |
| `LV_PART_SCROLLBAR` | 缩放后滚动条 |
| `LV_PART_ITEMS` | 数据系列（LINE/BAR） |
| `LV_PART_INDICATOR` | LINE/SCATTER 的数据点 |
| `LV_PART_CURSOR` | 光标 |

## 核心 API

### 数据系列
```c
lv_chart_add_series(chart, color, axis);     // 添加数据系列
lv_chart_set_next_value(chart, series, val); // 添加下一个值
lv_chart_set_all_values(chart, series, val); // 设置所有值
lv_chart_set_point_count(chart, count);      // 设置数据点数（默认10）
```

### 更新模式
- `LV_CHART_UPDATE_MODE_SHIFT` — 左移，新值加到右侧
- `LV_CHART_UPDATE_MODE_CIRCULAR` — 循环更新（ECG风格）

### 其他
```c
lv_chart_set_type(chart, LV_CHART_TYPE_...);         // 设置类型
lv_chart_set_axis_range(chart, axis, min, max);     // Y轴范围
lv_chart_set_div_line_count(chart, hdiv, vdiv);     // 分隔线数
```

### 光标
```c
lv_chart_add_cursor(chart, color, dir);           // 添加光标
lv_chart_set_cursor_pos(chart, cursor, &point);   // 设置光标位置
lv_chart_set_cursor_point(chart, cursor, series, id); // 绑定到数据点
```

## 绘制层次
从后到前：背景 → 数据系列 → 光标

## 事件
- `LV_EVENT_VALUE_CHANGED` — 按下新数据点

## 关键要点
- 折线图和柱状图最常用
- 直接访问数据数组可批量修改 + `lv_chart_refresh()`
- 大点数优化: 点数 > 水平像素时绘制 min/max 竖线
- 缩放: 设置图表尺寸大于父容器，父容器变为滚动窗口
