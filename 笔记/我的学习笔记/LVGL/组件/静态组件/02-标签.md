# 标签 (lv_label)

> **重要性**: ★★★★★ (10/10) — 文本展示必备，几乎所有项目必用

## 概述
Label 是 LVGL 中用于显示文本的控件。

## 设置文本

| 方法 | 说明 |
|------|------|
| `lv_label_set_text(label, "text")` | 动态分配，复制字符串到内部缓冲区 |
| `lv_label_set_text_fmt(label, "Value: %d", val)` | printf 格式化设置 |
| `lv_label_set_text_static(label, "text")` | 零拷贝，缓冲区必须保持有效 |

## 长模式 (Long Modes) — 文本溢出处理

| 模式 | 行为 |
|------|------|
| `LV_LABEL_LONG_MODE_WRAP` | 自动换行，默认模式 |
| `LV_LABEL_LONG_MODE_DOTS` | 末尾显示 "..." |
| `LV_LABEL_LONG_MODE_SCROLL` | 来回滚动 |
| `LV_LABEL_LONG_MODE_SCROLL_CIRCULAR` | 循环滚动 |
| `LV_LABEL_LONG_MODE_CLIP` | 直接裁剪溢出 |

## 文本重着色
```c
lv_label_set_recolor(label, true);  // 启用
lv_label_set_text(label, "写一个 #ff0000 红色# 的字");  // 内嵌颜色
```

## 部件与样式
| 部件 | 说明 |
|------|------|
| `LV_PART_MAIN` | 背景和文本属性 |
| `LV_PART_SCROLLBAR` | 溢出时显示滚动条 |
| `LV_PART_SELECTED` | 选中文本样式 |

## 性能优化
- 超长文本 (>40k字符): 启用 `LV_LABEL_LONG_TXT_HINT` 配置，增加12字节/标签，大幅提升滚动和绘制速度

## 数据绑定
```c
lv_label_bind_text(label, &subject, "%d");  // 单向绑定 (Subject ==> Widget)
```
支持 STRING、POINTER、INTEGER、FLOAT 类型

## 事件与按键
- 不主动发送特殊事件
- 添加 `LV_OBJ_FLAG_CLICKABLE` 后可触发 `LV_EVENT_CLICKED`
- 不处理按键输入

## 关键要点
- 最基础的文本展示控件
- 长模式选择很重要：短文本用 CLIP/WARP，长文本用 SCROLL
- 重着色功能可以做简单富文本
