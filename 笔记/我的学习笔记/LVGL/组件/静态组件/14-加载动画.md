# 加载动画 (lv_spinner)

> **重要性**: ★★★☆☆ (7/10) — 等待加载指示器

## 概述
Spinner 是旋转的弧形弧线加圆环，用于显示正在进行某项操作。

## 部件与样式
与 Arc 相同。

## 核心 API

```c
lv_obj_t *spinner = lv_spinner_create(parent);
lv_spinner_set_anim_params(spinner, duration, angle);  // 设置旋转时间和弧线长度
```

## 事件与按键
- 无特殊事件，不处理按键

## 示例
```c
lv_obj_t *spinner = lv_spinner_create(lv_screen_active());
lv_obj_set_size(spinner, 100, 100);
lv_obj_center(spinner);
lv_spinner_set_anim_params(spinner, 10000, 200);  // 10秒一圈，弧线200度
```

## 关键要点
- 网络请求、数据加载、初始化等场景的等待指示
- 纯展示控件，无需用户交互
- 基于 Arc 实现，可用 Arc 样式自定义外观
