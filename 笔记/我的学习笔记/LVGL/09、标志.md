# LVGL 控件标志 （属性开关与行为控制）

## 核心概念

- **标志位存储** - LVGL使用位域存储标志，节省内存，一个32位整数可存储多个标志
- **开关控制** - 标志用于启用/禁用控件的特定行为或属性
- **运行时修改** - 大部分标志可在运行时动态添加或移除
- **状态检查** - 使用 `has_flag` 检查控件当前标志状态
- **组合使用** - 多个标志可通过位运算组合使用

## 一、标志操作基础

### 1.1 基本操作函数

```c
// 添加标志（启用功能）
lv_obj_add_flag(widget, LV_OBJ_FLAG_XXX);

// 移除标志（禁用功能）
lv_obj_remove_flag(widget, LV_OBJ_FLAG_XXX);

// 检查标志（判断状态）
bool has_flag = lv_obj_has_flag(widget, LV_OBJ_FLAG_XXX);
```

| 函数 | 说明 | 返回值 |
|------|------|--------|
| `lv_obj_add_flag(obj, flag)` | 添加标志 | 无 |
| `lv_obj_remove_flag(obj, flag)` | 移除标志 | 无 |
| `lv_obj_has_flag(obj, flag)` | 检查是否有标志 | `bool` |
| `lv_obj_add/remove/has_flag_any(obj, flags)` | 批量操作多个标志 | - |

### 1.2 组合多个标志

**方式1：多次调用（推荐）**

```c
lv_obj_add_flag(widget, LV_OBJ_FLAG_CLICKABLE);
lv_obj_add_flag(widget, LV_OBJ_FLAG_EVENT_BUBBLE);
```

**方式2：位运算组合（C++需要类型转换）**

```c
// C语言
lv_obj_add_flag(widget, LV_OBJ_FLAG_CLICKABLE | LV_OBJ_FLAG_EVENT_BUBBLE);

// C++ 需要显式转换
lv_obj_add_flag(widget, (lv_obj_flag_t)(LV_OBJ_FLAG_CLICKABLE | LV_OBJ_FLAG_EVENT_BUBBLE));
```

> **注意**：C++ 由于类型检查更严格，位运算组合需要强制转换为 `lv_obj_flag_t`。

## 二、标志分类详解

### 2.1 可见性与交互标志

| 标志 | 说明 | 典型应用 |
|------|------|----------|
| `LV_OBJ_FLAG_HIDDEN` | 隐藏控件（不占用布局空间） | 标签页切换、条件显示 |
| `LV_OBJ_FLAG_CLICKABLE` | 可点击 | 按钮、可点击区域 |
| `LV_OBJ_FLAG_CLICK_FOCUSABLE` | 点击时获得焦点 | 输入框、列表项 |
| `LV_OBJ_FLAG_CHECKABLE` | 可切换选中状态 | 复选框、开关 |
| `LV_OBJ_FLAG_PRESS_LOCK` | 按下后即使移出仍保持按下状态 | 拖拽按钮 |

```c
// 示例：创建一个可点击、可聚焦的输入框
lv_obj_t * textarea = lv_textarea_create(lv_screen_active());
lv_obj_add_flag(textarea, LV_OBJ_FLAG_CLICKABLE);
lv_obj_add_flag(textarea, LV_OBJ_FLAG_CLICK_FOCUSABLE);

// 示例：隐藏和显示控件
lv_obj_add_flag(widget, LV_OBJ_FLAG_HIDDEN);      // 隐藏
lv_obj_remove_flag(widget, LV_OBJ_FLAG_HIDDEN);   // 显示
```

### 2.2 滚动相关标志

| 标志 | 说明 | 效果 |
|------|------|------|
| `LV_OBJ_FLAG_SCROLLABLE` | 可滚动 | 内容超出时显示滚动条 |
| `LV_OBJ_FLAG_SCROLL_ELASTIC` | 弹性滚动 | 边缘回弹效果 |
| `LV_OBJ_FLAG_SCROLL_MOMENTUM` | 惯性滚动 | 快速滑动后继续滚动 |
| `LV_OBJ_FLAG_SCROLL_ONE` | 每次只滚动一个子项 | 轮播图、分页 |
| `LV_OBJ_FLAG_SCROLL_CHAIN_HOR` | 水平滚动传递给父控件 | 嵌套滚动容器 |
| `LV_OBJ_FLAG_SCROLL_CHAIN_VER` | 垂直滚动传递给父控件 | 嵌套滚动容器 |
| `LV_OBJ_FLAG_SCROLL_CHAIN` | 同时启用水平和垂直传递 | 简写形式 |
| `LV_OBJ_FLAG_SCROLL_ON_FOCUS` | 聚焦时自动滚动到可见区域 | 键盘导航 |
| `LV_OBJ_FLAG_SCROLL_WITH_ARROW` | 支持方向键滚动 | 遥控器、键盘 |
| `LV_OBJ_FLAG_SNAPPABLE` | 可吸附对齐 | 轮播、列表对齐 |

```c
// 示例：创建弹性惯性滚动的列表
lv_obj_t * list = lv_obj_create(lv_screen_active());
lv_obj_set_size(list, 200, 300);
lv_obj_add_flag(list, LV_OBJ_FLAG_SCROLLABLE);
lv_obj_add_flag(list, LV_OBJ_FLAG_SCROLL_ELASTIC);
lv_obj_add_flag(list, LV_OBJ_FLAG_SCROLL_MOMENTUM);
lv_obj_add_flag(list, LV_OBJ_FLAG_SNAPPABLE);

// 禁用滚动（保留内容但不可滚动）
lv_obj_remove_flag(list, LV_OBJ_FLAG_SCROLLABLE);
```

### 2.3 事件传递标志

| 标志 | 说明 | 方向 |
|------|------|------|
| `LV_OBJ_FLAG_EVENT_BUBBLE` | 事件冒泡到父控件 | 子 → 父 |
| `LV_OBJ_FLAG_EVENT_TRICKLE` | 事件渗透到子控件 | 父 → 子 |
| `LV_OBJ_FLAG_STATE_TRICKLE` | 状态变化渗透到子控件 | 父 → 子 |
| `LV_OBJ_FLAG_GESTURE_BUBBLE` | 手势冒泡到父控件 | 子 → 父 |

```c
// 示例：事件冒泡 - 父容器统一处理所有按钮点击
lv_obj_t * container = lv_obj_create(lv_screen_active());
lv_obj_add_flag(container, LV_OBJ_FLAG_EVENT_BUBBLE);
lv_obj_add_event_cb(container, button_group_handler, LV_EVENT_CLICKED, NULL);

// 子按钮点击事件会冒泡到容器
for(int i = 0; i < 5; i++) {
    lv_obj_t * btn = lv_button_create(container);
    lv_obj_add_flag(btn, LV_OBJ_FLAG_EVENT_BUBBLE);
}
```

### 2.4 布局与定位标志

| 标志 | 说明 | 应用场景 |
|------|------|----------|
| `LV_OBJ_FLAG_IGNORE_LAYOUT` | 忽略布局定位 | 绝对定位覆盖 |
| `LV_OBJ_FLAG_FLOATING` | 浮动（不随父控件滚动，忽略布局） | 悬浮按钮、提示框 |
| `LV_OBJ_FLAG_FLEX_IN_NEW_TRACK` | Flex布局中开启新行/列 | 强制换行 |
| `LV_OBJ_FLAG_OVERFLOW_VISIBLE` | 子控件可溢出边界显示 | 下拉菜单、弹出层 |
| `LV_OBJ_FLAG_ADV_HITTEST` | 精确点击测试（考虑圆角） | 圆形按钮 |

```c
// 示例：悬浮按钮（固定在屏幕，不随内容滚动）
lv_obj_t * fab = lv_button_create(lv_screen_active());
lv_obj_set_size(fab, 56, 56);
lv_obj_align(fab, LV_ALIGN_BOTTOM_RIGHT, -20, -20);
lv_obj_add_flag(fab, LV_OBJ_FLAG_FLOATING);  // 浮动，不参与布局

// 示例：子控件可溢出（用于下拉菜单）
lv_obj_t * dropdown_container = lv_obj_create(lv_screen_active());
lv_obj_add_flag(dropdown_container, LV_OBJ_FLAG_OVERFLOW_VISIBLE);
```

### 2.5 绘制相关标志

| 标志 | 说明 | 用途 |
|------|------|------|
| `LV_OBJ_FLAG_SEND_DRAW_TASK_EVENTS` | 发送绘制任务事件 | 自定义渲染 |

```c
// 示例：启用绘制事件以自定义渲染
lv_obj_add_flag(widget, LV_OBJ_FLAG_SEND_DRAW_TASK_EVENTS);
lv_obj_add_event_cb(widget, custom_draw_handler, LV_EVENT_DRAW_TASK_ADDED, NULL);
```

### 2.6 自定义标志（用户/控件/布局使用）

| 标志 | 用途 |
|------|------|
| `LV_OBJ_FLAG_LAYOUT_1` / `LV_OBJ_FLAG_LAYOUT_2` | 自定义布局使用 |
| `LV_OBJ_FLAG_WIDGET_1` / `LV_OBJ_FLAG_WIDGET_2` | 自定义控件使用 |
| `LV_OBJ_FLAG_USER_1` / `LV_OBJ_FLAG_USER_2` | 用户代码使用 |

```c
// 示例：使用自定义标志标记控件状态
typedef enum {
    MY_STATE_EDITING = LV_OBJ_FLAG_USER_1,
    MY_STATE_LOCKED = LV_OBJ_FLAG_USER_2
} my_state_t;

// 标记正在编辑
lv_obj_add_flag(textarea, MY_STATE_EDITING);

// 检查状态
if(lv_obj_has_flag(textarea, MY_STATE_EDITING)) {
    // 正在编辑中...
}
```

## 四、完整示例

### 示例1：动态显示/隐藏控件

```c
#include "lvgl.h"

static lv_obj_t * settings_panel;

static void toggle_settings_cb(lv_event_t * e)
{
    // 切换设置面板的显示/隐藏
    if(lv_obj_has_flag(settings_panel, LV_OBJ_FLAG_HIDDEN)) {
        lv_obj_remove_flag(settings_panel, LV_OBJ_FLAG_HIDDEN);
        lv_obj_set_height(settings_panel, 200);  // 展开
    } else {
        lv_obj_add_flag(settings_panel, LV_OBJ_FLAG_HIDDEN);
    }
}

void create_toggle_panel(void)
{
    // 创建切换按钮
    lv_obj_t * toggle_btn = lv_button_create(lv_screen_active());
    lv_obj_align(toggle_btn, LV_ALIGN_TOP_MID, 0, 10);
    lv_obj_add_event_cb(toggle_btn, toggle_settings_cb, LV_EVENT_CLICKED, NULL);
    
    lv_obj_t * label = lv_label_create(toggle_btn);
    lv_label_set_text(label, "Toggle Settings");
    lv_obj_center(label);
    
    // 创建设置面板（初始隐藏）
    settings_panel = lv_obj_create(lv_screen_active());
    lv_obj_set_size(settings_panel, 280, 200);
    lv_obj_align(settings_panel, LV_ALIGN_TOP_MID, 0, 60);
    lv_obj_add_flag(settings_panel, LV_OBJ_FLAG_HIDDEN);  // 初始隐藏
    
    // 添加一些设置项
    lv_obj_t * slider = lv_slider_create(settings_panel);
    lv_obj_set_width(slider, 200);
    lv_obj_center(slider);
}
```

### 示例2：可编辑/只读切换

```c
#include "lvgl.h"

static lv_obj_t * textarea;
static lv_obj_t * edit_btn;

static void toggle_edit_cb(lv_event_t * e)
{
    if(lv_obj_has_flag(textarea, LV_OBJ_FLAG_CLICKABLE)) {
        // 切换到只读模式
        lv_obj_remove_flag(textarea, LV_OBJ_FLAG_CLICKABLE);
        lv_obj_remove_flag(textarea, LV_OBJ_FLAG_CLICK_FOCUSABLE);
        lv_obj_set_state(textarea, LV_STATE_DISABLED, true);
        lv_label_set_text(lv_obj_get_child(edit_btn, 0), "Edit");
    } else {
        // 切换到编辑模式
        lv_obj_add_flag(textarea, LV_OBJ_FLAG_CLICKABLE);
        lv_obj_add_flag(textarea, LV_OBJ_FLAG_CLICK_FOCUSABLE);
        lv_obj_set_state(textarea, LV_STATE_DISABLED, false);
        lv_label_set_text(lv_obj_get_child(edit_btn, 0), "Lock");
        lv_obj_add_state(textarea, LV_STATE_FOCUSED);
    }
}

void create_editable_text(void)
{
    textarea = lv_textarea_create(lv_screen_active());
    lv_obj_set_size(textarea, 200, 100);
    lv_obj_align(textarea, LV_ALIGN_CENTER, 0, -30);
    lv_textarea_set_text(textarea, "Click Edit to modify this text");
    
    // 初始为只读
    lv_obj_remove_flag(textarea, LV_OBJ_FLAG_CLICKABLE);
    lv_obj_set_state(textarea, LV_STATE_DISABLED, true);
    
    edit_btn = lv_button_create(lv_screen_active());
    lv_obj_align(edit_btn, LV_ALIGN_CENTER, 0, 50);
    lv_obj_add_event_cb(edit_btn, toggle_edit_cb, LV_EVENT_CLICKED, NULL);
    
    lv_obj_t * label = lv_label_create(edit_btn);
    lv_label_set_text(label, "Edit");
    lv_obj_center(label);
}
```

### 示例4：使用自定义标志管理状态

```c
#include "lvgl.h"

// 定义自定义标志用途
#define MY_FLAG_IS_SAVED    LV_OBJ_FLAG_USER_1
#define MY_FLAG_IS_MODIFIED LV_OBJ_FLAG_USER_2

static lv_obj_t * text_editor;
static lv_obj_t * save_btn;
static lv_obj_t * status_label;

static void text_changed_cb(lv_event_t * e)
{
    // 文本改变，标记为已修改
    lv_obj_add_flag(text_editor, MY_FLAG_IS_MODIFIED);
    lv_obj_remove_flag(text_editor, MY_FLAG_IS_SAVED);
    
    lv_label_set_text(status_label, "Modified - Unsaved");
    lv_obj_set_style_text_color(status_label, lv_palette_main(LV_PALETTE_RED), 0);
}

static void save_btn_cb(lv_event_t * e)
{
    if(lv_obj_has_flag(text_editor, MY_FLAG_IS_MODIFIED)) {
        // 执行保存操作...
        
        // 更新标志
        lv_obj_add_flag(text_editor, MY_FLAG_IS_SAVED);
        lv_obj_remove_flag(text_editor, MY_FLAG_IS_MODIFIED);
        
        lv_label_set_text(status_label, "Saved");
        lv_obj_set_style_text_color(status_label, lv_palette_main(LV_PALETTE_GREEN), 0);
    }
}

void create_editor_with_state(void)
{
    // 状态标签
    status_label = lv_label_create(lv_screen_active());
    lv_label_set_text(status_label, "New - Unsaved");
    lv_obj_align(status_label, LV_ALIGN_TOP_MID, 0, 10);
    
    // 文本编辑器
    text_editor = lv_textarea_create(lv_screen_active());
    lv_obj_set_size(text_editor, 280, 200);
    lv_obj_align(text_editor, LV_ALIGN_CENTER, 0, 0);
    lv_obj_add_event_cb(text_editor, text_changed_cb, LV_EVENT_VALUE_CHANGED, NULL);
    
    // 保存按钮
    save_btn = lv_button_create(lv_screen_active());
    lv_obj_align(save_btn, LV_ALIGN_BOTTOM_MID, 0, -20);
    lv_obj_add_event_cb(save_btn, save_btn_cb, LV_EVENT_CLICKED, NULL);
    
    lv_obj_t * label = lv_label_create(save_btn);
    lv_label_set_text(label, "Save");
    lv_obj_center(label);
}

### 示例5：从零开始创建自定义按钮（使用基础控件 + 标志）

**场景**：不使用 `lv_button_create()`，而是用基础 `lv_obj` + 标志来创建功能完整的自定义按钮。

**原理**：LVGL的控件本质上都是 `lv_obj`，通过添加不同的标志赋予其特定行为。

```c
#include "lvgl.h"

// 自定义按钮结构体
typedef struct {
    lv_obj_t * obj;           // 基础对象
    lv_obj_t * label;         // 按钮文字
    lv_color_t normal_color;  // 正常颜色
    lv_color_t pressed_color; // 按下颜色
} my_button_t;

static my_button_t my_btn;

// 按钮事件回调
static void my_button_event_cb(lv_event_t * e)
{
    lv_event_code_t code = lv_event_get_code(e);
    lv_obj_t * btn = lv_event_get_target(e);
    
    switch(code) {
        case LV_EVENT_PRESSED:
            // 按下时改变颜色
            lv_obj_set_style_bg_color(btn, my_btn.pressed_color, 0);
            break;
            
        case LV_EVENT_RELEASED:
            // 释放时恢复颜色
            lv_obj_set_style_bg_color(btn, my_btn.normal_color, 0);
            break;
            
        case LV_EVENT_CLICKED:
            // 点击事件
            LV_LOG_USER("Custom button clicked!");
            break;
            
        default:
            break;
    }
}

// 创建自定义按钮
void create_custom_button(void)
{
    // 步骤1：创建基础对象（从零开始）
    my_btn.obj = lv_obj_create(lv_screen_active());
    lv_obj_set_size(my_btn.obj, 120, 50);
    lv_obj_center(my_btn.obj);
    
    // 步骤2：设置基础样式
    my_btn.normal_color = lv_palette_main(LV_PALETTE_BLUE);
    my_btn.pressed_color = lv_palette_darken(LV_PALETTE_BLUE, 2);
    
    lv_obj_set_style_bg_color(my_btn.obj, my_btn.normal_color, 0);
    lv_obj_set_style_radius(my_btn.obj, 10, 0);  // 圆角
    lv_obj_set_style_border_width(my_btn.obj, 0, 0);  // 无边框
    lv_obj_set_style_shadow_width(my_btn.obj, 10, 0);  // 阴影
    lv_obj_set_style_shadow_color(my_btn.obj, lv_color_hex(0x888888), 0);
    
    // 步骤3：添加标志赋予按钮功能
    // 可点击 - 必需，否则无法响应点击
    lv_obj_add_flag(my_btn.obj, LV_OBJ_FLAG_CLICKABLE);
    
    // 按下锁定 - 移出仍保持按下状态
    lv_obj_add_flag(my_btn.obj, LV_OBJ_FLAG_PRESS_LOCK);
    
    // 点击获得焦点
    lv_obj_add_flag(my_btn.obj, LV_OBJ_FLAG_CLICK_FOCUSABLE);
    
    // 步骤4：添加事件回调
    lv_obj_add_event_cb(my_btn.obj, my_button_event_cb, LV_EVENT_PRESSED, NULL);
    lv_obj_add_event_cb(my_btn.obj, my_button_event_cb, LV_EVENT_RELEASED, NULL);
    lv_obj_add_event_cb(my_btn.obj, my_button_event_cb, LV_EVENT_CLICKED, NULL);
    
    // 步骤5：添加文字标签
    my_btn.label = lv_label_create(my_btn.obj);
    lv_label_set_text(my_btn.label, "Custom Btn");
    lv_obj_center(my_btn.label);
    
    // 步骤6：设置布局让标签居中
    lv_obj_set_flex_flow(my_btn.obj, LV_FLEX_FLOW_ROW);
    lv_obj_set_flex_align(my_btn.obj, LV_FLEX_ALIGN_CENTER, LV_FLEX_ALIGN_CENTER, LV_FLEX_ALIGN_CENTER);
}
```

**关键标志解析**：

| 标志 | 作用 | 如果不添加会怎样？ |
|------|------|-------------------|
| `LV_OBJ_FLAG_CLICKABLE` | 使对象可响应点击 | 完全无法点击，所有事件无效 |
| `LV_OBJ_FLAG_PRESS_LOCK` | 按下后保持状态 | 手指移出即释放，无按压效果 |
| `LV_OBJ_FLAG_CLICK_FOCUSABLE` | 点击获得焦点 | 无法通过键盘导航聚焦 |

## 附录：LVGL 标志 API 速查表

### 标志操作函数

| 函数 | 说明 | 返回值 |
|------|------|--------|
| `lv_obj_add_flag(obj, flag)` | 添加标志 | 无 |
| `lv_obj_remove_flag(obj, flag)` | 移除标志 | 无 |
| `lv_obj_has_flag(obj, flag)` | 检查是否有标志 | `bool` |
| `lv_obj_add_flag_any(obj, flags)` | 添加多个标志 | 无 |
| `lv_obj_remove_flag_any(obj, flags)` | 移除多个标志 | 无 |
| `lv_obj_has_flag_any(obj, flags)` | 检查是否有任一标志 | `bool` |
| `lv_obj_has_flag_all(obj, flags)` | 检查是否有所有标志 | `bool` |

### 可见性与交互标志

| 标志 | 说明 |
|------|------|
| `LV_OBJ_FLAG_HIDDEN` | 隐藏控件 |
| `LV_OBJ_FLAG_CLICKABLE` | 可点击 |
| `LV_OBJ_FLAG_CLICK_FOCUSABLE` | 点击获得焦点 |
| `LV_OBJ_FLAG_CHECKABLE` | 可切换选中状态 |
| `LV_OBJ_FLAG_PRESS_LOCK` | 按下后保持状态 |

### 滚动相关标志

| 标志 | 说明 |
|------|------|
