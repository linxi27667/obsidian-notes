# Transitions 过渡动画 （状态切换的平滑过渡：时间+路径+属性）

> 过渡动画为控件状态切换添加动画效果，通过定义时长、缓动曲线和动画属性，实现背景色、大小、透明度等属性的平滑变化。

---

## 核心概念

### 过渡组成要素
- ==**属性列表**== - 需要动画的样式属性（如 `LV_STYLE_BG_COLOR`）
- ==**持续时间**== - 动画完成的总时长（毫秒）
- ==**延迟时间**== - 动画开始前的等待时间（毫秒）
- ==**缓动路径**== - 动画速度曲线（线性、缓入、缓出等）

### 过渡与状态的关系
- ==状态特定过渡== - 不同状态可配置不同的过渡参数
- ==双向动画== - 进入和离开状态的过渡可独立配置
- ==组合效果== - 多个属性可同时动画

### 典型应用场景
- ==按钮反馈== - 按下时背景色平滑过渡
- ==卡片交互== - 悬停时放大+阴影变化
- ==开关切换== - 指示器位置滑动动画
- ==页面转场== - 控件淡入淡出效果

## 一、基础过渡配置

### 1.1 动画属性数组的作用

动画属性数组告诉 LVGL：**哪些样式属性**在状态切换时需要执行过渡动画。

**为什么需要这个数组？**

```c
// ❌ 如果没有属性数组，LVGL不知道要动画哪些属性
// 所有属性都动画？性能太差！
// 没有动画？用户体验不好！

// ✅ 通过数组精确控制：只有指定的属性才会动画
static const lv_style_prop_t trans_props[] = {
    LV_STYLE_BG_COLOR,      // 只有背景颜色会动画
    LV_STYLE_TRANSFORM_SCALE_X, // 只有缩放会动画（v9：ZOOM→SCALE）
    LV_STYLE_TRANSFORM_SCALE_Y, // 垂直缩放（v9新增）
    // LV_STYLE_BORDER_WIDTH  // 边框宽度不会动画（没有添加到数组）
    0  // 结束标记，必须存在！
};
```

**实际效果对比**：

| 场景 | 无属性数组（假设） | 有属性数组（实际） |
|------|------------------|------------------|
| 按钮按下 | 所有属性瞬间变化 | 背景色渐变，缩放动画，其他属性瞬间变化 |
| 性能 | 无法控制 | 只动画必要属性，节省资源 |
| 灵活性 | 无 | 精确控制每个属性的动画 |

### 1.2 过渡描述符结构

过渡动画通过 `lv_style_transition_dsc_t` 描述符配置：

```c
// 1. 定义要动画的属性数组（必须以 0 结尾）
static const lv_style_prop_t trans_props[] = {
    LV_STYLE_BG_COLOR,      // 背景颜色参与动画
    LV_STYLE_BG_OPA,        // 背景透明度参与动画
    LV_STYLE_TRANSFORM_WIDTH, // 变换宽度参与动画
    0  // 结束标记，必须存在！
};

// 2. 创建过渡描述符变量
static lv_style_transition_dsc_t trans;

// 3. 初始化过渡描述符（v9新增第6个参数 user_data）
lv_style_transition_dsc_init(
    &trans,                    // 过渡描述符指针
    trans_props,               // 动画属性数组
    lv_anim_path_ease_out,     // 缓动路径
    300,                       // 持续时间(ms)
    0,                         // 延迟时间(ms)
    NULL                       // 用户数据（v9新增）
);
```

| 参数        | 类型                           | 说明                  |
| ----------- | ------------------------------ | --------------------- |
| `trans`     | `lv_style_transition_dsc_t*`   | 过渡描述符指针        |
| `props`     | `const lv_style_prop_t[]`      | 动画属性数组，以0结尾 |
| `path`      | `lv_anim_path_cb_t`            | 缓动曲线函数          |
| `time`      | `uint32_t`                     | 动画持续时间（毫秒）  |
| `delay`     | `uint32_t`                     | 动画延迟时间（毫秒）  |
| `user_data` | `void*`                        | 用户数据（v9新增）    |

### 1.3 将过渡应用到样式

```c
// 创建样式
static lv_style_t style;
lv_style_init(&style);

// 设置过渡（样式会保存过渡描述符的指针）
lv_style_set_transition(&style, &trans);

// 应用到控件
lv_obj_add_style(btn, &style, 0);
```

> **注意**：过渡描述符必须使用 `static` 或全局变量，因为样式只保存指针！

### 1.4 属性数组使用实例

**场景**：按钮按下时，希望背景色平滑过渡，但边框宽度瞬间变化

```c
// 定义动画属性：只有背景色和缩放会动画
static const lv_style_prop_t btn_props[] = {
    LV_STYLE_BG_COLOR,          // ✅ 会动画（在数组中）
    LV_STYLE_TRANSFORM_SCALE_X, // ✅ 会动画（v9：ZOOM→SCALE）
    LV_STYLE_TRANSFORM_SCALE_Y, // ✅ 垂直缩放（v9新增）
    0  // 结束标记
};

// 初始化过渡（v9新增第6个参数 NULL）
static lv_style_transition_dsc_t trans;
lv_style_transition_dsc_init(&trans, btn_props, 
    lv_anim_path_ease_out, 300, 0, NULL);

// 创建样式
static lv_style_t style_normal, style_pressed;
lv_style_init(&style_normal);
lv_style_init(&style_pressed);

// 正常状态样式
lv_style_set_bg_color(&style_normal, lv_palette_main(LV_PALETTE_BLUE));
// v9：lv_style_set_transform_zoom → lv_style_set_transform_scale_x/y，256=100%
lv_style_set_transform_scale_x(&style_normal, 256);  // 正常大小
lv_style_set_transform_scale_y(&style_normal, 256);
lv_style_set_border_width(&style_normal, 0);         // 无边框
lv_style_set_transition(&style_normal, &trans);

// 按下状态样式
lv_style_set_bg_color(&style_pressed, lv_palette_darken(LV_PALETTE_BLUE, 2));
// v9：缩放值 256=100%，245≈95% (256*0.95)
lv_style_set_transform_scale_x(&style_pressed, 245);  // 缩小到95%
lv_style_set_transform_scale_y(&style_pressed, 245);
lv_style_set_border_width(&style_pressed, 3);         // 显示边框
lv_style_set_border_color(&style_pressed, lv_color_white());

// 应用到按钮
lv_obj_add_style(btn, &style_normal, 0);
lv_obj_add_style(btn, &style_pressed, LV_STATE_PRESSED);
```

**效果分析**：

| 属性 | 是否在数组 | 状态切换时的表现 |
|------|-----------|----------------|
| 背景颜色 | ✅ 是 | **300ms 渐变** 从蓝色到深蓝 |
| 缩放 | ✅ 是 | **300ms 动画** 从100%到95% |
| 边框宽度 | ❌ 否 | **瞬间变化** 从0到3（无动画） |
| 边框颜色 | ❌ 否 | **瞬间变化** 立即变白 |

## 三、缓动路径（动画曲线）

### 3.1 内置缓动函数

| 缓动路径                       | 说明   | 效果示意           |
| -------------------------- | ---- | -------------- |
| `lv_anim_path_linear`      | 线性   | 匀速运动 ▬▬▬       |
| `lv_anim_path_ease_in`     | 缓入   | 由慢到快 ▁▂▃▅▆     |
| `lv_anim_path_ease_out`    | 缓出   | 由快到慢 ▆▅▃▂▁     |
| `lv_anim_path_ease_in_out` | 缓入缓出 | 慢→快→慢 ▁▂▅▂▁    |
| `lv_anim_path_overshoot`   | 过冲   | 超过目标后回弹 ▁▃▅▆▇▃ |
| `lv_anim_path_bounce`      | 弹跳   | 到达后弹跳 ▁▃▅▆▇▂▆▂ |
| `lv_anim_path_step`        | 阶跃   | 瞬间完成 ▁▁▁▁▇     |

### 3.2 选择合适的缓动曲线

```c
// 按钮按下 - 快速响应（v9新增第6个参数 NULL）

    lv_anim_path_ease_out, 100, 0, NULL);

// 按钮释放 - 优雅恢复（v9新增第6个参数 NULL）

    lv_anim_path_ease_in_out, 300, 0, NULL);

// 开关切换 - 弹性反馈（v9新增第6个参数 NULL）
lv_style_transition_dsc_init(&trans_switch, props,
    lv_anim_path_overshoot, 200, 0, NULL);

// 页面进入 - 弹跳效果（v9新增第6个参数 NULL）
lv_style_transition_dsc_init(&trans_page_in, props,
    lv_anim_path_bounce, 500, 100, NULL);
```

## 四、状态特定的过渡

### 4.1 不同状态不同过渡参数

可以为不同状态配置不同的过渡效果：

```c
// 定义属性（v9：ZOOM→SCALE）
static const lv_style_prop_t trans_props[] = {
    LV_STYLE_BG_COLOR,
    LV_STYLE_TRANSFORM_SCALE_X,  // v9：ZOOM→SCALE
    LV_STYLE_TRANSFORM_SCALE_Y,  // v9新增
    0
};

// 默认状态过渡（缓慢恢复，v9新增第6个参数 NULL）
static lv_style_transition_dsc_t trans_default;
lv_style_transition_dsc_init(&trans_default, trans_props,
    lv_anim_path_ease_out, 500, 0, NULL);

// 按下状态过渡（快速响应，v9新增第6个参数 NULL）
static lv_style_transition_dsc_t trans_pressed;
lv_style_transition_dsc_init(&trans_pressed, trans_props,
    lv_anim_path_ease_out, 100, 0, NULL);

// 创建样式
static lv_style_t style_default, style_pressed;
lv_style_init(&style_default);
lv_style_init(&style_pressed);

// 分别设置过渡
lv_style_set_transition(&style_default, &trans_default);
lv_style_set_transition(&style_pressed, &trans_pressed);

// 应用到控件
lv_obj_add_style(btn, &style_default, 0);
lv_obj_add_style(btn, &style_pressed, LV_STATE_PRESSED);
```

**效果**：
- 按下时：100ms 快速变化
- 释放时：500ms 缓慢恢复到默认状态

### 4.2 全局过渡 vs 状态过渡

```c
// 方式1：全局过渡（应用到所有状态变化）
lv_style_set_transition(&style, &trans);  // 添加到基础样式

// 方式2：状态特定过渡（仅应用到特定状态）
lv_style_set_transition(&style_pressed, &trans_fast);  // 仅按下状态
lv_style_set_transition(&style_focused, &trans_slow);  // 仅聚焦状态
```

## 五、完整示例

### 示例1：带过渡动画的按钮

```c
#include "lvgl.h"

static lv_style_t style_btn;
static lv_style_t style_btn_pressed;
static lv_style_transition_dsc_t trans_normal;
static lv_style_transition_dsc_t trans_fast;

void create_animated_button(void)
{
    // 定义动画属性（v9：ZOOM→SCALE）
    static const lv_style_prop_t btn_props[] = {
        LV_STYLE_BG_COLOR,
        LV_STYLE_TRANSFORM_SCALE_X,  // v9：ZOOM→SCALE
        LV_STYLE_TRANSFORM_SCALE_Y,  // v9新增
        0
    };

    // 正常状态过渡（500ms 缓出，v9新增第6个参数 NULL）
    lv_style_transition_dsc_init(&trans_normal, btn_props,
        lv_anim_path_ease_out, 500, 0, NULL);

    // 按下状态过渡（100ms 快速响应，v9新增第6个参数 NULL）
    lv_style_transition_dsc_init(&trans_fast, btn_props,
        lv_anim_path_ease_out, 100, 0, NULL);

    // 正常状态样式
    lv_style_init(&style_btn);
    lv_style_set_bg_color(&style_btn, lv_palette_main(LV_PALETTE_BLUE));
    lv_style_set_text_color(&style_btn, lv_color_white());
    // v9：ZOOM→SCALE，256=100%
    lv_style_set_transform_scale_x(&style_btn, 256);
    lv_style_set_transform_scale_y(&style_btn, 256);
    lv_style_set_radius(&style_btn, 8);
    lv_style_set_size(&style_btn, 120, 50);
    lv_style_set_transition(&style_btn, &trans_normal);

    // 按下状态样式
    lv_style_init(&style_btn_pressed);
    lv_style_set_bg_color(&style_btn_pressed,
        lv_palette_darken(LV_PALETTE_BLUE, 2));
    // v9：缩放值 256=100%，245≈95%
    lv_style_set_transform_scale_x(&style_btn_pressed, 245);
    lv_style_set_transform_scale_y(&style_btn_pressed, 245);
    lv_style_set_transition(&style_btn_pressed, &trans_fast);
    
    // 创建按钮
    lv_obj_t *btn = lv_btn_create(lv_scr_act());
    lv_obj_add_style(btn, &style_btn, 0);
    lv_obj_add_style(btn, &style_btn_pressed, LV_STATE_PRESSED);
    lv_obj_center(btn);
    
    lv_obj_t *label = lv_label_create(btn);
    lv_label_set_text(label, "Press Me");
}
```

### 示例2：卡片悬停效果

```c
#include "lvgl.h"

static lv_style_t style_card;
static lv_style_t style_card_hovered;
static lv_style_transition_dsc_t trans_card;

void create_hover_card(void)
{
    // 卡片动画属性（v9：ZOOM→SCALE）
    static const lv_style_prop_t card_props[] = {
        LV_STYLE_BG_COLOR,
        LV_STYLE_SHADOW_OPA, //阴影透明度
        LV_STYLE_TRANSFORM_SCALE_X,  // v9：ZOOM→SCALE
        LV_STYLE_TRANSFORM_SCALE_Y,  // v9新增
        LV_STYLE_SHADOW_WIDTH,
        0
    };

    // 统一的过渡效果（300ms 缓入缓出，v9新增第6个参数 NULL）
    lv_style_transition_dsc_init(&trans_card, card_props,
        lv_anim_path_ease_in_out, 300, 0, NULL);  // v9新增第6个参数 NULL

    // 默认状态
    lv_style_init(&style_card);
    lv_style_set_bg_color(&style_card, lv_color_white());
    lv_style_set_shadow_color(&style_card, lv_color_hex(0xCCCCCC));
    lv_style_set_shadow_width(&style_card, 5);
    lv_style_set_shadow_opa(&style_card, LV_OPA_30);
    // v9：ZOOM→SCALE，256=100%
    lv_style_set_transform_scale_x(&style_card, 256);
    lv_style_set_transform_scale_y(&style_card, 256);
    lv_style_set_radius(&style_card, 12);
    lv_style_set_size(&style_card, 200, 150);
    lv_style_set_transition(&style_card, &trans_card);

    // 悬停状态
```

**动画效果预览：**

**完整代码实现：**

```c
#include "lvgl.h"

void create_animated_button(void)
{
    // ========== 1. 定义要动画的属性 ==========
    // 只有这些属性会在状态切换时产生动画效果
    static const lv_style_prop_t anim_props[] = {
        LV_STYLE_BG_COLOR,           // 背景颜色渐变
        LV_STYLE_TRANSFORM_SCALE_X,  // 水平缩放 (v9: ZOOM→SCALE)
        LV_STYLE_TRANSFORM_SCALE_Y,  // 垂直缩放
        LV_STYLE_SHADOW_WIDTH,       // 阴影宽度变化
        LV_STYLE_SHADOW_OPA,         // 阴影透明度
        0  // 结束标记，必须存在！
    };
    
    // ========== 2. 创建过渡描述符 ==========
    // static是必须的！样式只保存指针，不是复制数据
    static lv_style_transition_dsc_t trans;
    lv_style_transition_dsc_init(
        &trans,                      // 过渡描述符
        anim_props,                  // 动画属性数组
        lv_anim_path_ease_out,       // 缓动曲线：先快后慢
        300,                         // 持续时间：300毫秒
        0,                           // 延迟：立即开始
        NULL                         // 用户数据（v9新增）
    );
    
    // ========== 3. 创建【默认状态】样式 ==========
    static lv_style_t style_normal;
    lv_style_init(&style_normal);
    lv_style_set_bg_color(&style_normal, lv_palette_main(LV_PALETTE_BLUE));
    lv_style_set_transform_scale_x(&style_normal, 256);  // 256 = 100%
    lv_style_set_transform_scale_y(&style_normal, 256);
    lv_style_set_shadow_width(&style_normal, 10);
    lv_style_set_shadow_opa(&style_normal, LV_OPA_30);
    lv_style_set_radius(&style_normal, 10);
    // 将过渡应用到默认样式
    lv_style_set_transition(&style_normal, &trans);
    
    // ========== 4. 创建【按下状态】样式 ==========
    static lv_style_t style_pressed;
    lv_style_init(&style_pressed);
    lv_style_set_bg_color(&style_pressed, lv_palette_darken(LV_PALETTE_BLUE, 2));
    lv_style_set_transform_scale_x(&style_pressed, 243);  // 256*0.95 ≈ 243 (缩小5%)
    lv_style_set_transform_scale_y(&style_pressed, 243);
    lv_style_set_shadow_width(&style_pressed, 20);
    lv_style_set_shadow_opa(&style_pressed, LV_OPA_50);
    // 按下状态的过渡（释放时的动画）
    lv_style_set_transition(&style_pressed, &trans);
    
    // ========== 5. 创建按钮并应用样式 ==========
    lv_obj_t * btn = lv_btn_create(lv_scr_act());
    lv_obj_set_size(btn, 150, 60);
    lv_obj_center(btn);
    
    // 应用默认状态样式（选择器为0，即 LV_PART_MAIN | LV_STATE_DEFAULT）
    lv_obj_add_style(btn, &style_normal, 0);
    
    // 应用按下状态样式
    lv_obj_add_style(btn, &style_pressed, LV_STATE_PRESSED);
    
    // 添加文字
    lv_obj_t * label = lv_label_create(btn);
    lv_label_set_text(label, "Press Me");
    lv_obj_center(label);
    
    // 添加点击事件（用于测试）
    lv_obj_add_event_cb(btn, 
        lambda void(lv_event_t * e) {
            if(lv_event_get_code(e) == LV_EVENT_CLICKED) {
                LV_LOG_USER("Button clicked!");
            }
        }, LV_EVENT_ALL, NULL);
}
```

**关键理解点：**

| 要点 | 说明 |
|------|------|
| **属性数组精确控制** | 只有数组中的属性会动画，其他属性瞬间变化 |
| **static变量必需** | 过渡描述符和样式必须使用static或全局变量 |
| **双向动画** | 按下和释放都有过渡效果（从A→B和B→A） |
| **缩放值** | v9中256=100%，243≈95%（缩小5%） |
| **缓动曲线** | ease_out让动画自然减速，更符合物理直觉 |

**常见问题：**

**Q: 为什么我的动画没有效果？**
A: 检查：1) 属性是否在数组中；2) 样式是否应用了过渡；3) static变量是否被释放；4) 两个状态的样式值是否确实不同。

**Q: 如何只让按下有动画，释放瞬间恢复？**
A: 只为默认状态样式设置过渡，按下状态不设置过渡。

**Q: 不同状态可以使用不同过渡参数吗？**
A: 可以！默认状态和按下状态可以使用不同的过渡描述符。

## 附录：过渡动画 API 速查表

### 过渡描述符操作
| 函数 | 说明 |
|------|------|
| `lv_style_transition_dsc_init(&trans, props, path, time, delay, user_data)` | 初始化过渡描述符（v9新增user_data） |
| `lv_style_set_transition(&style, &trans)` | 将过渡应用到样式 |

### 缓动路径函数
| 函数 | 说明 |
|------|------|
| `lv_anim_path_linear` | 线性匀速 |
| `lv_anim_path_ease_in` | 缓入（加速） |
| `lv_anim_path_ease_out` | 缓出（减速） |
| `lv_anim_path_ease_in_out` | 缓入缓出 |
| `lv_anim_path_overshoot` | 过冲回弹 |
| `lv_anim_path_bounce` | 弹跳效果 |
| `lv_anim_path_step` | 瞬间完成 |

### 常用可动画属性
| 属性常量 | 说明 |
|----------|------|
| `LV_STYLE_BG_COLOR` | 背景颜色 |
| `LV_STYLE_BG_OPA` | 背景透明度 |
| `LV_STYLE_BORDER_COLOR` | 边框颜色 |
| `LV_STYLE_BORDER_WIDTH` | 边框宽度 |
| `LV_STYLE_TEXT_COLOR` | 文字颜色 |
| `LV_STYLE_WIDTH` / `LV_STYLE_HEIGHT` | 宽高 |
| `LV_STYLE_TRANSFORM_SCALE_X` | 水平缩放（v9：ZOOM→SCALE，256=100%） |
| `LV_STYLE_TRANSFORM_SCALE_Y` | 垂直缩放（v9新增） |
| `LV_STYLE_TRANSFORM_ROTATION` | 旋转角度（v9：ANGLE→ROTATION） |
| `LV_STYLE_OPA` | 整体透明度 |
| `LV_STYLE_SHADOW_WIDTH` | 阴影宽度 |
| `LV_STYLE_SHADOW_OPA` | 阴影透明度 |

### 注意事项
- 过渡描述符必须使用 `static` 或全局变量
- 属性数组必须以 `0` 结尾
- 不同状态可配置不同的过渡参数
- 过渡只影响样式属性变化，不影响布局
