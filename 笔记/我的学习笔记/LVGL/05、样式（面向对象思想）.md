# Styles 样式系统 （CSS灵感：级联+状态+部件分离）

> LVGL 样式系统灵感来源于 CSS，通过 `lv_style_t` 变量存储属性（边框、颜色、字体等），支持级联、状态切换和部件分离，实现灵活的控件外观控制。

---

## 核心概念

### 样式本质
- ==`lv_style_t` 变量== - 存储外观属性的容器，类似 CSS 中的 class
- ==级联（Cascading）== - 多个样式可叠加，后添加的样式优先级更高
- ==继承（Inheritance）== - 部分属性（如文字颜色）可从父控件继承

### 状态系统
- ==状态（State）== - 控件的不同交互状态（默认、按下、聚焦等）
- ==状态优先级== - 数值越大优先级越高（`LV_STATE_PRESSED` > `LV_STATE_FOCUSED` > `LV_STATE_DEFAULT`）
- ==组合状态== - 支持 `LV_STATE_FOCUSED | LV_STATE_PRESSED` 组合

### 部件分离
- ==部件（Part）== - 控件的组成部分可独立设置样式
- ==主部件== - `LV_PART_MAIN` 控件主体
- ==子部件== - 如滑块的 `LV_PART_INDICATOR` 和 `LV_PART_KNOB`

### 样式类型
- ==全局样式== - 可复用的 `lv_style_t` 对象，多个控件共享
- ==本地样式== - 控件专属样式，类似 CSS 内联样式 `<div style="...">`

## 一、状态（States）

### 1.1 内置状态常量

| 状态常量 | 十六进制值 | 说明 |
|----------|-----------|------|
| `LV_STATE_DEFAULT` | 0x0000 | 默认/正常/释放状态 |
| `LV_STATE_CHECKED` | 0x0001 | 切换/勾选状态 |
| `LV_STATE_FOCUSED` | 0x0002 | 通过键盘/编码器聚焦，或触摸/鼠标点击 |
| `LV_STATE_FOCUS_KEY` | 0x0004 | 仅通过键盘/编码器聚焦（非触摸） |
| `LV_STATE_EDITED` | 0x0008 | 编码器编辑中 |
| `LV_STATE_HOVERED` | 0x0010 | 鼠标悬停 |
| `LV_STATE_PRESSED` | 0x0020 | 按下状态 |
| `LV_STATE_SCROLLED` | 0x0040 | 滚动中 |
| `LV_STATE_DISABLED` | 0x0080 | 禁用状态 |
| `LV_STATE_USER_1` | 0x1000 | 自定义状态1 |
| `LV_STATE_USER_2` | 0x2000 | 自定义状态2 |
| `LV_STATE_USER_3` | 0x4000 | 自定义状态3 |
| `LV_STATE_USER_4` | 0x8000 | 自定义状态4 |

**组合状态示例**：
```c
// 控件同时处于聚焦和按下状态
lv_obj_set_style_bg_color(btn, lv_color_red(), 
    LV_STATE_FOCUSED | LV_STATE_PRESSED);
```

### 1.2 状态优先级机制

状态值越大优先级越高，决定属性应用顺序： 

```c
// 定义示例
LV_STATE_DEFAULT:  white   // 0x0000
LV_STATE_PRESSED:  gray    // 0x0020
LV_STATE_FOCUSED:  red     // 0x0002
```

**优先级规则**：
1. **完全匹配优先** - 如果当前状态精确定义了属性，直接使用
2. **高优先级覆盖** - 数值大的状态覆盖数值小的状态
3. **默认状态兜底** - 无匹配时使用 `LV_STATE_DEFAULT` 的属性
4. **组合状态** - `LV_STATE_PRESSED | LV_STATE_FOCUSED` = 0x0022，优先级高于单独状态

**实际案例**：
```c
// 场景：按钮在聚焦+按下状态下的颜色选择
lv_obj_set_style_bg_color(btn, lv_color_white(), LV_STATE_DEFAULT);    // 白色
lv_obj_set_style_bg_color(btn, lv_color_gray(),  LV_STATE_PRESSED);    // 灰色
lv_obj_set_style_bg_color(btn, lv_color_red(),   LV_STATE_FOCUSED);    // 红色
lv_obj_set_style_bg_color(btn, lv_color_rose(),  
    LV_STATE_PRESSED | LV_STATE_FOCUSED);  // 玫瑰色

// 结果：
// - 默认状态 → 白色
// - 按下状态 → 灰色（0x0020 > 0x0000）
// - 聚焦状态 → 红色（0x0002 > 0x0000）
// - 按下+聚焦 → 玫瑰色（0x0022 > 0x0020）
```

### 1.3 快捷写法

由于 `LV_PART_MAIN` 和 `LV_STATE_DEFAULT` 都为 0，可以简写：

```c
// 完整写法
lv_obj_add_style(btn, &style, LV_PART_MAIN | LV_STATE_DEFAULT);

// 快捷写法（等效）
lv_obj_add_style(btn, &style, 0);
```

#### 核心概念图解

**样式系统工作原理：**

```
┌─────────────────────────────────────────────────────────────────┐
│                     样式选择器决策流程                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   用户设置样式：                                                 │
│   lv_obj_set_style_bg_color(btn, red, selector)                 │
│                            │                                    │
│                            ▼                                    │
│   ┌──────────────────────────────────────────────┐             │
│   │           选择器 = 部件 | 状态                 │             │
│   │                                                │             │
│   │   LV_PART_MAIN (0x000000)                      │             │
│   │        |                                       │             │
│   │        + 状态 (0x0000 ~ 0xFFFF)                 │             │
│   │        |                                       │             │
│   │   = 最终选择器值                                │             │
│   └──────────────────────────────────────────────┘             │
│                            │                                    │
│                            ▼                                    │
│   匹配检查：                                                    │
│   ┌─────────────┐  ┌─────────────┐  ┌─────────────┐            │
│   │ 当前部件？  │→│ 当前状态？  │→│ 精确匹配？   │            │
│   │ LV_PART_MAIN│  │ PRESSED     │  │ 使用此样式   │            │
│   └─────────────┘  └─────────────┘  └─────────────┘            │
│                            │                                    │
│                    否 ↓    │    ↓ 是                           │
│               ┌────────────┴────────────┐                      │
│               │ 查找下一个可能匹配的状态 │                      │
│               │ (按优先级从高到低)       │                      │
│               └─────────────────────────┘                      │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**选择器值计算示例：**

```
选择器 = 部件值 | 状态值

示例1：主部件 + 默认状态
LV_PART_MAIN    = 0x000000
LV_STATE_DEFAULT = 0x0000
选择器 = 0x000000 | 0x0000 = 0x000000 (即 0)

示例2：主部件 + 按下状态  
LV_PART_MAIN    = 0x000000
LV_STATE_PRESSED = 0x0020
选择器 = 0x000000 | 0x0020 = 0x002000

示例3：旋钮部件 + 聚焦状态
LV_PART_KNOB    = 0x030000
LV_STATE_FOCUSED = 0x0002
选择器 = 0x030000 | 0x0002 = 0x030002
```

**样式优先级规则：**

```
优先级从高到低：

1. 本地样式（直接设置）> 全局样式（添加到对象）
   lv_obj_set_style_XXX()  >  lv_obj_add_style()

2. 后添加的样式 > 先添加的样式
   lv_obj_add_style(obj, &style2) 覆盖 style1

3. 特定状态 > 默认状态
   LV_STATE_PRESSED 样式覆盖 LV_STATE_DEFAULT

4. 组合状态 > 单个状态
   LV_STATE_PRESSED | FOCUSED > 单独 PRESSED
```

## 二、部件（Parts）

### 2.1 内置部件常量

| 部件常量 | 十六进制值 | 说明 |
|----------|-----------|------|
| `LV_PART_MAIN` | 0x000000 | 主部件（背景矩形） |
| `LV_PART_SCROLLBAR` | 0x010000 | 滚动条 |
| `LV_PART_INDICATOR` | 0x020000 | 指示器（滑块、开关、复选框勾选） |
| `LV_PART_KNOB` | 0x030000 | 旋钮/手柄（用于调整值） |
| `LV_PART_SELECTED` | 0x040000 | 选中项标记 |
| `LV_PART_ITEMS` | 0x050000 | 多个相似元素（表格单元格等） |
| `LV_PART_CURSOR` | 0x060000 | 光标（文本区、图表） |
| `LV_PART_CUSTOM_FIRST` | 0x080000 | 自定义部件起始值 |
| `LV_PART_ANY` | 0x0F0000 | 特殊值，匹配所有部件 |

### 2.2 滑块部件示例

滑块（Slider）包含三个独立部件：

```c
lv_obj_t *slider = lv_slider_create(lv_scr_act());

// 主部件（背景轨道）
lv_obj_set_style_bg_color(slider, lv_color_gray(), LV_PART_MAIN);

// 指示器部件（已滑动部分）
lv_obj_set_style_bg_color(slider, lv_palette_main(LV_PALETTE_BLUE), 
    LV_PART_INDICATOR);

// 旋钮部件（可拖动手柄）
lv_obj_set_style_bg_color(slider, lv_palette_main(LV_PALETTE_BLUE), 
    LV_PART_KNOB);
lv_obj_set_style_radius(slider, LV_RADIUS_CIRCLE, LV_PART_KNOB);
```

## 三、级联与继承

### 3.1 样式级联（Cascading）

类似 CSS 多 class 叠加，后添加的样式优先级更高：

```c
// 创建基础按钮样式（灰色）
static lv_style_t style_btn;
lv_style_init(&style_btn);
lv_style_set_bg_color(&style_btn, lv_color_hex(0x888888));

// 创建红色按钮样式（仅修改背景色）
static lv_style_t style_btn_red;
lv_style_init(&style_btn_red);
lv_style_set_bg_color(&style_btn_red, lv_palette_main(LV_PALETTE_RED));

// 应用样式：先基础，后红色
lv_obj_add_style(btn, &style_btn, 0);      // 先添加
lv_obj_add_style(btn, &style_btn_red, 0);  // 后添加，覆盖背景色
// 结果：按钮显示红色背景
```

**级联优先级规则**：
1. 后添加的样式 > 先添加的样式
2. 但状态优先级仍然有效

```c
// 复杂案例：基础样式定义了多状态颜色
lv_style_set_bg_color(&style_btn, lv_color_dark_gray(), LV_STATE_DEFAULT);
lv_style_set_bg_color(&style_btn, lv_color_light_gray(), LV_STATE_PRESSED);

// 红色样式只定义了默认状态
lv_style_set_bg_color(&style_btn_red, lv_color_red(), LV_STATE_DEFAULT);

// 应用结果：
// - 默认状态：红色（精确匹配，且后添加）
// - 按下状态：浅灰色（唯一精确匹配，虽然先添加）
```

### 3.2 属性继承

部分属性（主要是文本相关）可从父控件继承：

```c
// 父容器设置文字颜色
lv_obj_t *parent = lv_obj_create(lv_scr_act());
lv_obj_set_style_text_color(parent, lv_color_blue(), 0);

// 子标签未设置文字颜色，将继承父容器的蓝色
lv_obj_t *label = lv_label_create(parent);
lv_label_set_text(label, "继承蓝色文字");

// 子标签也可覆盖继承
lv_obj_set_style_text_color(label, lv_color_red(), 0);  // 现在显示红色
```

**可继承属性**：`text_color`、`text_font`、`text_letter_space` 等文本属性

**继承链**：子 → 父 → 祖父 → ... → 找到定义或到达根

## 四、样式表（Style Sheets）

### 4.1 初始化样式

样式变量需为静态、全局或动态分配，不能是函数局部变量：

```c
// ✅ 正确：全局静态样式
static lv_style_t style_btn;

void init_ui(void)
{
    lv_style_init(&style_btn);
    lv_style_set_bg_color(&style_btn, lv_palette_main(LV_PALETTE_BLUE));
    lv_style_set_border_width(&style_btn, 2);
}

// ❌ 错误：局部变量函数退出后失效
void wrong_example(void)
{
    lv_style_t style_local;  // 错误！
    lv_style_init(&style_local);
}
```

### 4.2 设置样式属性

通用格式：`lv_style_set_<property_name>(&style, <value>)`

```c
static lv_style_t style_btn;
lv_style_init(&style_btn);

// 背景相关
lv_style_set_bg_color(&style_btn, lv_color_hex(0x115588));
lv_style_set_bg_opa(&style_btn, LV_OPA_50);

// 边框相关
lv_style_set_border_width(&style_btn, 2);
lv_style_set_border_color(&style_btn, lv_color_black());

// 大小和圆角
lv_style_set_width(&style_btn, 100);
lv_style_set_height(&style_btn, 50);
lv_style_set_radius(&style_btn, 10);

// 内边距
lv_style_set_pad_all(&style_btn, 10);
```

### 4.3 移除和获取属性

```c
// 移除属性
lv_style_remove_prop(&style, LV_STYLE_BG_COLOR);

// 获取属性值
lv_style_value_t v;
lv_result_t res = lv_style_get_prop(&style, LV_STYLE_BG_COLOR, &v);
if(res == LV_RESULT_OK) {
    // v.color 包含颜色值
    lv_color_t color = v.color;
}

// lv_style_value_t 包含三个字段，根据属性类型使用：
// v.num   - 整数、布尔、透明度
// v.color - 颜色值
// v.ptr   - 指针（字体、图片等）
```

### 4.4 重置样式

```c
// 释放样式数据
lv_style_reset(&style);
```

### 4.5 常量样式（节省 RAM）

如果样式运行时不变，可定义为常量节省内存：

```c
const lv_style_const_prop_t style1_props[] = {
    LV_STYLE_CONST_WIDTH(50),
    LV_STYLE_CONST_HEIGHT(50),
    LV_STYLE_CONST_BG_COLOR(LV_COLOR_MAKE(0x11, 0x55, 0x88)),
    LV_STYLE_CONST_PROPS_END
};

LV_STYLE_CONST_INIT(style1, style1_props);

// 使用方式与普通样式相同，但不能添加新属性
lv_obj_add_style(btn, &style1, 0);
```

## 五、为控件添加样式

### 5.1 添加样式

```c
// 基本用法
lv_obj_add_style(widget, &style, selector);

// selector = 部件 | 状态

// 示例1：默认状态主部件（最常用）
lv_obj_add_style(btn, &style_btn, 0);

// 示例2：按下状态主部件（可省略 LV_PART_MAIN）
lv_obj_add_style(btn, &style_pressed, LV_STATE_PRESSED);

// 示例3：滚动条默认状态（可省略 LV_STATE_DEFAULT）
lv_obj_add_style(list, &style_scrollbar, LV_PART_SCROLLBAR);

// 示例4：滚动条滚动时状态
lv_obj_add_style(list, &style_scrollbar_active, 
    LV_PART_SCROLLBAR | LV_STATE_SCROLLED);

// 示例5：指示器按下+选中状态
lv_obj_add_style(switch, &style_indicator_on,
    LV_PART_INDICATOR | LV_STATE_PRESSED | LV_STATE_CHECKED);
```

### 5.2 移除样式

```c
// 移除所有样式
lv_obj_remove_style_all(widget);

// 移除特定样式（需精确匹配 selector）
lv_obj_remove_style(widget, &style, LV_STATE_PRESSED);

// 移除所有匹配 selector 的样式（style 传 NULL）
lv_obj_remove_style(widget, NULL, LV_STATE_PRESSED);

// 从任意状态移除
lv_obj_remove_style(widget, &style, LV_STATE_ANY);

// 从任意部件移除
lv_obj_remove_style(widget, &style, LV_PART_ANY);
```

### 5.3 替换样式

```c
// 将 old_style 替换为 new_style（selector 必须匹配）
lv_bool_t replaced = lv_obj_replace_style(widget, 
    &old_style, &new_style, 0);

// 示例：动态切换主题
lv_obj_add_style(btn, &style_light, 0);
// ... 用户切换到深色模式
lv_obj_replace_style(btn, &style_light, &style_dark, 0);
```

### 5.4 通知样式变更

当已分配给控件的样式发生变化时，需要通知控件刷新：

```c
// 方式1：简单重绘（适用于颜色、透明度变化）
lv_obj_invalidate(widget);  // 重绘单个控件
lv_obj_invalidate(lv_screen_active());  // 重绘整个屏幕

// 方式2：刷新特定属性（复杂属性变更）
lv_obj_refresh_style(widget, LV_PART_ANY, LV_STYLE_PROP_ANY);

// 方式3：全局通知（检查所有使用该样式的控件）
lv_obj_report_style_change(&style);  // 通知特定样式变更
lv_obj_report_style_change(NULL);    // 通知所有样式变更
```

## 六、本地样式（Local Styles）

### 6.1 本地样式概述

本地样式类似 CSS 内联样式，控件专属、自动管理内存：

```c
// CSS 对比：
// <div style="color: red; background: blue;">

// LVGL 本地样式：
lv_obj_set_style_text_color(label, lv_color_red(), 0);
lv_obj_set_style_bg_color(label, lv_palette_main(LV_PALETTE_BLUE), 0);
```

**特点**：
- 优先级高于普通样式
- 自动分配/释放内存
- 不能共享给其他控件
- 支持状态和部件

### 6.2 使用本地样式

```c
lv_obj_t *slider = lv_slider_create(lv_scr_act());

// 指示器部件聚焦时变红
lv_obj_set_style_bg_color(slider, lv_color_red(), 
    LV_PART_INDICATOR | LV_STATE_FOCUSED);

// 旋钮始终圆形
lv_obj_set_style_radius(slider, LV_RADIUS_CIRCLE, LV_PART_KNOB);
```

#### 实际示例图解：自定义按钮的完整样式系统

**场景：创建一个具有多种状态和过渡效果的自定义按钮**

```
┌─────────────────────────────────────────────────────────────────┐
│                    自定义按钮样式架构                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   按钮状态流转：                                                  │
│                                                                 │
│   ┌──────────┐    按下     ┌──────────┐    释放     ┌──────────┐ │
│   │  DEFAULT │ ─────────→ │ PRESSED  │ ─────────→ │  DEFAULT │ │
│   │  (白色)  │            │  (红色)  │            │  (白色)  │ │
│   └──────────┘            └──────────┘            └──────────┘ │
│        │                                               ↑        │
│        │ 聚焦                                          │        │
│        ▼                                               │        │
│   ┌──────────┐            ┌──────────┐                 │        │
│   │ FOCUSED  │ ←───────── │ DISABLED │ ────────────────┘        │
│   │ (蓝色边框)│   禁用      │  (灰色)  │                          │
│   └──────────┘            └──────────┘                          │
│                                                                 │
│   样式层次（从内到外）：                                          │
│   ┌─────────────────────────────────────────────┐              │
│   │  LV_PART_MAIN (背景层)                       │              │
│   │  ┌─────────────────────────────────────┐    │              │
│   │  │ 背景色、边框、阴影、圆角              │    │              │
│   │  └─────────────────────────────────────┘    │              │
│   └─────────────────────────────────────────────┘              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**完整代码实现：**

```c
// 创建自定义按钮
lv_obj_t * custom_btn = lv_btn_create(lv_scr_act());
lv_obj_set_size(custom_btn, 150, 60);

// ========== 1. 默认状态样式 ==========
// 主部件 - 背景
lv_obj_set_style_bg_color(custom_btn, lv_color_white(), LV_PART_MAIN | LV_STATE_DEFAULT);
lv_obj_set_style_border_color(custom_btn, lv_color_hex(0xCCCCCC), LV_PART_MAIN | LV_STATE_DEFAULT);
lv_obj_set_style_border_width(custom_btn, 2, LV_PART_MAIN | LV_STATE_DEFAULT);
lv_obj_set_style_radius(custom_btn, 10, LV_PART_MAIN | LV_STATE_DEFAULT);

// ========== 2. 按下状态样式 ==========
// 按下时背景变红，边框加粗
lv_obj_set_style_bg_color(custom_btn, lv_palette_main(LV_PALETTE_RED), 
    LV_PART_MAIN | LV_STATE_PRESSED);
lv_obj_set_style_border_color(custom_btn, lv_palette_darken(LV_PALETTE_RED, 2), 
    LV_PART_MAIN | LV_STATE_PRESSED);
lv_obj_set_style_border_width(custom_btn, 4, LV_PART_MAIN | LV_STATE_PRESSED);

// ========== 3. 聚焦状态样式 ==========
// 聚焦时蓝色边框
lv_obj_set_style_border_color(custom_btn, lv_palette_main(LV_PALETTE_BLUE), 
    LV_PART_MAIN | LV_STATE_FOCUSED);
lv_obj_set_style_shadow_width(custom_btn, 10, LV_PART_MAIN | LV_STATE_FOCUSED);
lv_obj_set_style_shadow_color(custom_btn, lv_palette_main(LV_PALETTE_BLUE), 
    LV_PART_MAIN | LV_STATE_FOCUSED);

// ========== 4. 禁用状态样式 ==========
// 禁用时灰色，透明度降低
lv_obj_set_style_bg_color(custom_btn, lv_color_hex(0xE0E0E0), 
    LV_PART_MAIN | LV_STATE_DISABLED);
lv_obj_set_style_text_color(custom_btn, lv_color_hex(0x999999), 
    LV_PART_MAIN | LV_STATE_DISABLED);
lv_obj_set_style_bg_opa(custom_btn, LV_OPA_50, LV_PART_MAIN | LV_STATE_DISABLED);

// ========== 5. 添加文字 ==========
lv_obj_t * label = lv_label_create(custom_btn);
lv_label_set_text(label, "Click Me");
lv_obj_center(label);

// ========== 6. 启用状态切换（让按钮可以进入PRESSED状态） ==========
lv_obj_add_flag(custom_btn, LV_OBJ_FLAG_CHECKABLE);
```

**效果预览：**

```
默认状态:              按下状态:              聚焦状态:
┌────────────┐        ┌════════════┐        ┌────────────┐
│            │   →    │            │   →    │  ╔══════╗  │
│  Click Me  │        │  Click Me  │        │  ║Click ║  │
│            │        │            │        │  ║ Me   ║  │
└────────────┘        └════════════┘        │  ╚══════╝  │
  白色背景              红色背景+粗边框         蓝色边框+阴影

禁用状态:
┌────────────┐
│░░░░░░░░░░░░│  ← 灰色半透明
│░ Click Me ░│
│░░░░░░░░░░░░│
└────────────┘
```

**关键理解点：**

1. **状态切换是自动的** - LVGL根据用户交互自动切换控件状态
2. **样式自动匹配** - 当前状态会自动匹配对应的选择器
3. **过渡平滑** - 结合过渡动画，状态切换会有平滑效果
4. **优先级处理** - 组合状态（如聚焦+按下）会优先于单个状态

---

### 7.1 核心概念

- **过渡描述符** - `lv_style_transition_dsc_t` 定义动画参数
- **缓动路径** - 控制动画速度曲线（linear、ease_out、bounce等）
- **可动画属性** - 背景色、透明度、缩放、旋转等
- **状态特定** - 不同状态可配置不同的过渡效果

### 7.2 快速入门

```c
// 1. 定义动画属性（v9：ZOOM→SCALE）
static const lv_style_prop_t props[] = {
    LV_STYLE_BG_COLOR,
    LV_STYLE_TRANSFORM_SCALE_X,  // v9：ZOOM→SCALE
    LV_STYLE_TRANSFORM_SCALE_Y,  // v9新增
    0  // 结束标记
};

// 2. 初始化过渡描述符（v9新增第6个参数 NULL）
static lv_style_transition_dsc_t trans;
lv_style_transition_dsc_init(&trans, props,
    lv_anim_path_ease_out, 300, 0, NULL);

// 3. 应用过渡
lv_style_set_transition(&style, &trans);
```

### 7.3 典型应用

| 场景 | 效果 |
|------|------|
| 按钮按下 | 背景色 + 缩放动画 |
| 卡片悬停 | 阴影 + 放大效果 |
| 弹窗显示 | 淡入 + 缩放动画 |
| 进度条 | 宽度平滑过渡 |

## 八、主题（Themes）

主题是样式的集合，自动应用到新创建的控件。

> 📚 **详细内容见**：[[07、主题|Themes 主题系统]]

### 8.1 核心概念

- **样式集合** - 预定义的控件样式，按类型组织
- **自动应用** - 新控件自动继承主题样式
- **全局配色** - 主色调、次色调、字体统一配置
- **亮暗切换** - 支持亮色/暗色主题切换

### 8.2 快速入门

```c
// 1. 初始化主题（在创建控件前！）
lv_theme_t *th = lv_theme_default_init(
    display,                    // 目标显示器
    LV_COLOR_PALETTE_BLUE,      // 主色调
    LV_COLOR_PALETTE_CYAN,      // 次色调
    false,                      // false=亮色, true=暗色
    &lv_font_montserrat_10,     // 小字体
    &lv_font_montserrat_14,     // 正常字体
    &lv_font_montserrat_18      // 大字体
);

// 2. 应用主题
lv_display_set_theme(display, th);
```

### 8.3 典型应用

| 场景 | 说明 |
|------|------|
| 应用配色 | 一次性设置整个应用配色方案 |
| 亮暗切换 | 实时切换亮色/暗色主题 |
| 品牌定制 | 根据品牌色定制控件外观 |
| 样式继承 | 主题 → 控件默认样式 → 自定义样式 |

### 示例1：多状态按钮

```c
#include "lvgl.h"

static lv_style_t style_btn;
static lv_style_t style_btn_pressed;
static lv_style_t style_btn_disabled;

void create_multi_state_button(void)
{
    // 初始化默认状态样式
    lv_style_init(&style_btn);
    lv_style_set_bg_color(&style_btn, lv_palette_main(LV_PALETTE_BLUE));
    lv_style_set_text_color(&style_btn, lv_color_white());
    lv_style_set_border_width(&style_btn, 0);
    lv_style_set_radius(&style_btn, 8);
    lv_style_set_pad_hor(&style_btn, 20);
    lv_style_set_pad_ver(&style_btn, 10);
    
    // 初始化按下状态样式（仅修改背景色）
    lv_style_init(&style_btn_pressed);
    lv_style_set_bg_color(&style_btn_pressed, 
        lv_palette_darken(LV_PALETTE_BLUE, 2));
    
    // 初始化禁用状态样式
    lv_style_init(&style_btn_disabled);
    lv_style_set_bg_color(&style_btn_disabled, 
        lv_palette_main(LV_PALETTE_GREY));
    lv_style_set_text_color(&style_btn_disabled, 
        lv_color_hex(0xAAAAAA));
    
    // 创建按钮并应用样式
    lv_obj_t *btn = lv_btn_create(lv_scr_act());
    lv_obj_add_style(btn, &style_btn, 0);
    lv_obj_add_style(btn, &style_btn_pressed, LV_STATE_PRESSED);
    lv_obj_add_style(btn, &style_btn_disabled, LV_STATE_DISABLED);
    
    lv_obj_t *label = lv_label_create(btn);
    lv_label_set_text(label, "Click Me");
    
    // 测试禁用状态
    // lv_obj_add_state(btn, LV_STATE_DISABLED);
}
```

### 示例3：带过渡动画的卡片

```c
#include "lvgl.h"

static lv_style_t style_card;
static lv_style_t style_card_pressed;
static lv_style_transition_dsc_t trans;

void create_animated_card(void)
{
    // 定义过渡动画属性（v9：ZOOM→SCALE）
    static const lv_style_prop_t trans_props[] = {
        LV_STYLE_BG_COLOR,
        LV_STYLE_TRANSFORM_SCALE_X,  // v9：ZOOM→SCALE
        LV_STYLE_TRANSFORM_SCALE_Y,  // v9新增
        0
    };

    // 初始化过渡描述符（v9新增第6个参数 NULL）
    lv_style_transition_dsc_init(&trans, trans_props,
        lv_anim_path_ease_out, 200, 0, NULL);

    // 默认状态样式
    lv_style_init(&style_card);
    lv_style_set_bg_color(&style_card, lv_color_white());
    lv_style_set_shadow_color(&style_card, lv_color_hex(0xCCCCCC));
    lv_style_set_shadow_width(&style_card, 10);
    lv_style_set_radius(&style_card, 12);
    lv_style_set_size(&style_card, 150, 100);
    lv_style_set_transition(&style_card, &trans);

    // 按下状态样式
    lv_style_init(&style_card_pressed);
    lv_style_set_bg_color(&style_card_pressed,
        lv_palette_lighten(LV_PALETTE_BLUE, 5));
    // v9：ZOOM→SCALE，269≈105% (256*1.05)
    lv_style_set_transform_scale_x(&style_card_pressed, 269);  // 放大5%
    lv_style_set_transform_scale_y(&style_card_pressed, 269);
    
    // 创建卡片
    lv_obj_t *card = lv_obj_create(lv_scr_act());
    lv_obj_add_style(card, &style_card, 0);
    lv_obj_add_style(card, &style_card_pressed, LV_STATE_PRESSED);
    lv_obj_center(card);
    
    // 添加标签
    lv_obj_t *label = lv_label_create(card);
    lv_label_set_text(label, "Press Me");
    lv_obj_center(label);
}
```
