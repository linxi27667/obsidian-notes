# Themes 主题系统 （样式的集合与全局配色方案）

> 主题是预定义样式的集合，自动应用到所有新创建的控件。通过主题可以统一应用配色方案、字体设置和基础样式，实现整个应用的一致性设计。

---

## 核心概念

### 主题的本质
- ==**样式集合**== - 主题是一组预定义的样式，按控件类型组织
- ==**自动应用**== - 新创建的控件自动继承主题中的样式
- ==**全局配色**== - 定义主色调、次色调、背景色等全局颜色
- ==**字体层级**== - 配置小/中/大三级字体

### 主题与样式
- ==**层级关系**== - 主题 → 控件默认样式 → 用户自定义样式
- ==**优先级**== - 后添加的样式覆盖前面的样式
- ==**局部覆盖**== - 可以覆盖主题中的特定样式

### 典型应用场景
- ==**应用配色**== - 一次性设置整个应用的配色方案
- ==**亮暗切换**== - 快速切换亮色/暗色主题
- ==**品牌定制**== - 根据品牌色定制控件外观
- ==**字体管理**== - 统一设置应用字体

## 一、内置主题

### 1.1 默认主题

LVGL 提供默认主题 `lv_theme_default`，包含所有标准控件的样式：

```c
// 初始化默认主题
lv_theme_t *th = lv_theme_default_init(
    display,                        // 目标显示器
    LV_COLOR_PALETTE_BLUE,          // 主色调（按钮、滑块等）
    LV_COLOR_PALETTE_CYAN,          // 次色调（开关、复选框等）
    false,                          // 暗色主题? false=亮色 true=暗色
    &lv_font_montserrat_10,         // 小字体
    &lv_font_montserrat_14,         // 正常字体
    &lv_font_montserrat_18          // 大字体
);

// 应用主题到显示器
lv_display_set_theme(display, th);
```

### 1.2 主题参数说明

| 参数 | 说明 | 典型值 |
|------|------|--------|
| `display` | 目标显示器 | `lv_display_get_default()` |
| `primary` | 主色调 | `LV_COLOR_PALETTE_BLUE` |
| `secondary` | 次色调 | `LV_COLOR_PALETTE_CYAN` |
| `dark` | 暗色模式 | `false`=亮色, `true`=暗色 |
| `font_small` | 小字体 | `&lv_font_montserrat_10` |
| `font_normal` | 正常字体 | `&lv_font_montserrat_14` |
| `font_large` | 大字体 | `&lv_font_montserrat_18` |

## 二、主题应用时机

### 2.1 最佳实践：在创建任何控件前设置主题

```c
void lvgl_init_theme(void)
{
    // 1. 初始化 LVGL（必须先完成）
    lv_init();
    
    // 2. 创建显示器
    lv_display_t *disp = lv_display_create(800, 480);
    lv_display_set_flush_cb(disp, my_flush_cb);
    
    // 3. 创建并应用主题（在创建控件之前！）
    lv_theme_t *th = lv_theme_default_init(
        disp,
        LV_COLOR_PALETTE_BLUE,      // 蓝色主色调
        LV_COLOR_PALETTE_CYAN,      // 青色次色调
        false,                      // 亮色主题
        &lv_font_montserrat_10,
        &lv_font_montserrat_14,
        &lv_font_montserrat_18
    );
    lv_display_set_theme(disp, th);
    
    // 4. 现在可以安全地创建控件了
    // 所有控件会自动继承主题样式
    lv_obj_t *btn = lv_btn_create(lv_scr_act());
    // 按钮会自动使用主题中的蓝色配色
}
```

### 2.2 延迟设置主题的问题

```c
// ❌ 错误：先创建控件，后设置主题
lv_obj_t *btn = lv_btn_create(lv_scr_act());  // 没有主题样式
lv_theme_t *th = lv_theme_default_init(...); // 创建主题
lv_display_set_theme(disp, th);              // 主题不会应用到已创建的按钮！

// ✅ 正确：先设置主题，后创建控件
lv_theme_t *th = lv_theme_default_init(...);
lv_display_set_theme(disp, th);
lv_obj_t *btn = lv_btn_create(lv_scr_act());  // 自动继承主题样式
```

## 三、主题继承与扩展

### 3.1 主题继承链

LVGL 支持主题继承，形成样式链：

```
默认主题 → 自定义主题 → 应用主题
   ↓           ↓            ↓
基础样式    品牌定制     特殊控件
```

**继承规则**：
- 子主题没有的样式，使用父主题的样式
- 子主题有的样式，覆盖父主题的样式
- 可以链式继承多个主题

### 3.2 扩展现有主题

```c
// 创建自定义主题，继承默认主题
lv_theme_t *custom_theme = lv_theme_custom_init(
    display,
    LV_COLOR_PALETTE_RED,       // 自定义主色调为红色
    LV_COLOR_PALETTE_ORANGE,    // 次色调为橙色
    false,
    &lv_font_montserrat_10,
    &lv_font_montserrat_14,
    &lv_font_montserrat_18
);

// 设置父主题（继承默认主题的其他样式）
lv_theme_set_parent(custom_theme, lv_theme_get_from_obj(lv_screen_active()));

// 应用自定义主题
lv_display_set_theme(display, custom_theme);
```

### 3.3 获取当前主题

```c
// 从任意控件获取其所在显示器的主题
lv_obj_t *btn = lv_btn_create(lv_scr_act());
lv_theme_t *th = lv_theme_get_from_obj(btn);

// 检查主题类型
if(th == lv_theme_default_get()) {
    LV_LOG_USER("Using default theme");
}
```

## 四、自定义控件样式

### 4.1 覆盖主题样式

即使应用了主题，也可以为特定控件添加自定义样式：

```c
// 创建按钮（自动使用主题样式：蓝色背景）
lv_obj_t *btn = lv_btn_create(lv_scr_act());

// 覆盖主题的默认颜色
lv_obj_set_style_bg_color(btn, lv_palette_main(LV_PALETTE_RED), 0);

// 按钮现在显示红色，但其他样式（圆角、边框等）仍来自主题
```

**样式优先级**（从高到低）：
1. 控件本地样式（`lv_obj_set_style_xxx`）
2. 用户添加的样式（`lv_obj_add_style`）
3. 主题样式
4. 默认样式

### 4.2 状态特定的主题覆盖

```c
// 默认状态使用主题颜色
// 按下状态使用自定义颜色
lv_obj_set_style_bg_color(btn, lv_palette_main(LV_PALETTE_GREEN), LV_STATE_PRESSED);

// 禁用状态自定义样式
lv_obj_set_style_bg_color(btn, lv_palette_main(LV_PALETTE_GREY), LV_STATE_DISABLED);
lv_obj_set_style_opa(btn, LV_OPA_40, LV_STATE_DISABLED);
```

## 五、亮暗主题切换

### 5.1 实时切换主题

```c
// 当前显示器
lv_display_t *disp = lv_display_get_default();

// 切换为暗色主题
lv_theme_t *dark_theme = lv_theme_default_init(
    disp,
    LV_COLOR_PALETTE_BLUE,
    LV_COLOR_PALETTE_CYAN,
    true,  // 暗色模式
    &lv_font_montserrat_10,
    &lv_font_montserrat_14,
    &lv_font_montserrat_18
);
lv_display_set_theme(disp, dark_theme);

// 通知所有控件主题已变更
lv_obj_report_style_change(NULL);
```

### 5.2 主题切换函数封装

```c
static lv_theme_t *light_theme = NULL;
static lv_theme_t *dark_theme = NULL;
static bool is_dark = false;

void init_themes(lv_display_t *disp)
{
    // 创建亮色主题
    light_theme = lv_theme_default_init(
        disp, LV_COLOR_PALETTE_BLUE, LV_COLOR_PALETTE_CYAN, false,
        &lv_font_montserrat_10, &lv_font_montserrat_14, &lv_font_montserrat_18
    );
    
    // 创建暗色主题
    dark_theme = lv_theme_default_init(
        disp, LV_COLOR_PALETTE_BLUE, LV_COLOR_PALETTE_CYAN, true,
        &lv_font_montserrat_10, &lv_font_montserrat_14, &lv_font_montserrat_18
    );
}

void toggle_theme(void)
{
    lv_display_t *disp = lv_display_get_default();
    
    is_dark = !is_dark;
    lv_display_set_theme(disp, is_dark ? dark_theme : light_theme);
    
    // 通知所有控件更新样式
    lv_obj_report_style_change(NULL);
    
    LV_LOG_USER("Switched to %s theme", is_dark ? "dark" : "light");
}
```

## 注意事项与最佳实践

### 主题应用时机

| 注意点 | 说明 | 建议 |
|--------|------|------|
| **应用顺序** | 主题必须在创建控件之前应用 | 在 `lv_init()` 后尽早设置主题 |
| **多显示器** | 每个显示器可设置不同主题 | 使用 `lv_display_set_theme(disp, theme)` |
| **动态切换** | 支持运行时切换主题 | 切换后需要刷新或重新创建控件 |
| **继承关系** | 新主题继承旧主题未覆盖的样式 | 注意样式继承链 |

### 主题初始化正确做法

```c
// ❌ 错误：在创建控件后设置主题
lv_obj_t *btn = lv_btn_create(lv_scr_act());
lv_theme_t *th = lv_theme_default_init(...);  // 太晚！
lv_display_set_theme(disp, th);

// ✅ 正确：先设置主题，再创建控件
lv_theme_t *th = lv_theme_default_init(...);
lv_display_set_theme(disp, th);
lv_obj_t *btn = lv_btn_create(lv_scr_act());  // 自动应用主题样式
```

### 样式覆盖优先级

```c
// 优先级：用户样式 > 主题样式 > 默认样式

// 1. 主题设置背景色为蓝色
// 2. 用户代码覆盖为红色 - 最终显示红色

```

### 字体注意事项

| 注意点 | 说明 |
|--------|------|
| **字体大小** | 确保字体文件包含所需字符 |
| **内存占用** | 大字体文件占用大量Flash |
| **多语言** | 使用支持目标语言的字体 |
| **fallback** | 设置备用字体处理缺失字符 |

### 常见问题

**Q: 设置了主题但控件没有变化？**
A: 检查：1) 主题是否在控件创建前设置；2) 显示器是否正确；3) 控件类型是否在主题中定义。

**Q: 如何为特定控件禁用主题？**
A: 使用 `lv_obj_remove_style_all(obj)` 移除所有样式后手动设置。

**Q: 亮暗主题切换时闪烁？**
A: 使用双缓冲并在切换时禁用刷新，或采用渐进式切换。

**Q: 自定义控件如何支持主题？**
A: 在 `apply_cb` 中处理自定义控件类型，为主题添加对应样式。

```c
#include "lvgl.h"

// 创建深色主题
void setup_dark_theme(void)
{
    // 获取默认显示器
    lv_display_t * disp = lv_display_get_default();
    
    // 初始化主题：紫色主色，青色次色，true=暗色模式
    lv_theme_t * th = lv_theme_default_init(
        disp,
        LV_COLOR_PALETTE_PURPLE,    // 主色调（按钮、滑块）
        LV_COLOR_PALETTE_TEAL,      // 次色调（开关、复选框）
        true,                       // 暗色模式
        &lv_font_montserrat_12,     // 小字体
        &lv_font_montserrat_16,     // 正常字体
        &lv_font_montserrat_22      // 大字体
    );
    
    // 应用主题到显示器
    lv_display_set_theme(disp, th);
    
    // 可选：进一步自定义背景色
    lv_obj_set_style_bg_color(lv_screen_active(), 
        lv_color_hex(0x121212), 0);
}

// 创建示例界面
void create_dark_ui(void)
{
    // 主容器
    lv_obj_t * cont = lv_obj_create(lv_screen_active());
    lv_obj_set_size(cont, 400, 300);
    lv_obj_center(cont);
    lv_obj_set_style_bg_color(cont, lv_color_hex(0x1E1E1E), 0);
    
    // 标题
    lv_obj_t * title = lv_label_create(cont);
    lv_label_set_text(title, "Dark Theme Demo");
    lv_obj_set_style_text_font(title, &lv_font_montserrat_22, 0);
    lv_obj_align(title, LV_ALIGN_TOP_MID, 0, 20);

    lv_obj_t * btn1 = lv_btn_create(cont);
    lv_obj_set_size(btn1, 120, 50);
    lv_obj_align(btn1, LV_ALIGN_CENTER, -70, 20);
    lv_obj_t * label1 = lv_label_create(btn1);
    lv_label_set_text(label1, "Primary");
    lv_obj_center(label1);
    
    // 次按钮
    lv_obj_t * btn2 = lv_btn_create(cont);
    lv_obj_set_size(btn2, 120, 50);
    lv_obj_align(btn2, LV_ALIGN_CENTER, 70, 20);
    // 应用次要样式
    lv_obj_set_style_bg_color(btn2, lv_palette_main(LV_PALETTE_TEAL), 0);
    lv_obj_t * label2 = lv_label_create(btn2);
    lv_label_set_text(label2, "Secondary");
    lv_obj_center(label2);
    
    // 滑块
    lv_obj_t * slider = lv_slider_create(cont);
    lv_obj_set_width(slider, 300);
    lv_obj_align(slider, LV_ALIGN_BOTTOM_MID, 0, -30);
```

**常见问题：**

**Q: 如何动态切换亮暗主题？**
A: 重新初始化主题并调用 `lv_obj_invalidate(lv_screen_active())` 刷新所有控件。

**Q: 主题切换后为什么有些控件没变化？**
A: 这些控件可能有本地样式覆盖，需要清除本地样式或重新创建。

### 主题初始化
| 函数 | 说明 |
|------|------|
| `lv_theme_default_init(disp, primary, secondary, dark, font_small, font_normal, font_large)` | 初始化默认主题 |

| 函数 | 说明 |
|------|------|
| `lv_theme_set_parent(new, base)` | 设置主题的父主题 |
| `lv_theme_get_from_obj(obj)` | 从控件获取所在主题 |
| `lv_theme_default_get()` | 获取默认主题实例 |

### 主题更新
| 函数 | 说明 |
|------|------|
| `lv_obj_report_style_change(style)` | 通知所有控件样式变更（NULL=所有样式） |
| `lv_obj_refresh_style(obj, part, prop)` | 刷新控件特定样式 |

### 主题切换最佳实践

```c
// 1. 切换主题
lv_display_set_theme(disp, new_theme);

// 2. 通知所有控件更新
lv_obj_report_style_change(NULL);

// 3. 如需强制刷新，可调用
lv_obj_invalidate(lv_scr_act());
```

```c
// 主色调用法
```

### 注意事项

1. **设置时机**：在创建任何控件前设置主题
2. **静态存储**：主题对象需要静态或全局存储
3. **样式优先级**：本地样式 > 添加样式 > 主题样式
4. **切换成本**：切换主题会触发所有控件重绘，有一定性能开销
