# LVGL 布局系统 （自动排列与自适应布局）

## 核心概念

- **布局管理** - LVGL通过布局自动管理子控件的位置和大小，覆盖手动设置的x、y坐标
- **Flex布局** - 一维布局，子控件按行或列排列，支持换行和弹性伸缩
- **Grid布局** - 二维布局，子控件放置在固定的行和列交叉单元格中
- **自适应尺寸** - 使用 `LV_SIZE_CONTENT` 让容器根据内容自动调整大小
- **布局标志** - 通过标志控制控件在布局中的行为（如忽略布局、浮动等）

## 一、布局基础

### 1.1 启用布局

```c
// 启用Flex布局
lv_obj_set_layout(container, LV_LAYOUT_FLEX);

// 启用Grid布局
lv_obj_set_layout(container, LV_LAYOUT_GRID);
```

| 布局类型 | 说明 | 适用场景 |
|----------|------|----------|
| `LV_LAYOUT_FLEX` | 一维弹性布局 | 列表、按钮组、导航栏 |
| `LV_LAYOUT_GRID` | 二维网格布局 | 表单、仪表盘、复杂界面 |
| `LV_LAYOUT_NONE` | 无布局（手动定位） | 绝对定位、特殊需求 |

### 1.2 布局相关标志

| 标志 | 说明 | 应用场景 |
|------|------|----------|
| `LV_OBJ_FLAG_HIDDEN` | 隐藏控件，布局计算时忽略 | 动态显示/隐藏 |
| `LV_OBJ_FLAG_IGNORE_LAYOUT` | 完全忽略布局，手动定位 | 绝对定位覆盖 |
| `LV_OBJ_FLAG_FLOATING` | 忽略布局且不参与内容尺寸计算 | 悬浮按钮、弹出层 |
| `LV_OBJ_FLAG_FLEX_IN_NEW_TRACK` | 强制换到新行/列（仅Flex） | 强制换行 |

```c
// 示例：悬浮按钮不参与布局
lv_obj_t * fab = lv_button_create(container);
lv_obj_add_flag(fab, LV_OBJ_FLAG_FLOATING);
lv_obj_align(fab, LV_ALIGN_BOTTOM_RIGHT, -20, -20);

// 示例：强制换行
lv_obj_t * new_section = lv_label_create(flex_container);
lv_obj_add_flag(new_section, LV_OBJ_FLAG_FLEX_IN_NEW_TRACK);
```

## 二、Flex 弹性布局

### 2.1 Flex 基础概念

```
Flex布局示意图：

┌─────────────────────────────────────────┐
│  Flex Container (父容器)                 │
│  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐       │
│  │Item1│ │Item2│ │Item3│ │Item4│       │
│  └─────┘ └─────┘ └─────┘ └─────┘       │
│  ←───────── Main Direction ──────────→  │
│  (主轴：ROW时为水平，COLUMN时为垂直)       │
└─────────────────────────────────────────┘
         ↑
    Cross Direction (交叉轴)
```

| 术语                  | 说明            |
| ------------------- | ------------- |
| **Track**           | 行或列           |
| **Main Direction**  | 主轴，子控件排列的主要方向 |
| **Cross Direction** | 交叉轴，垂直于主轴的方向  |
| **Wrap**            | 空间不足时换到新行/列   |
| **Grow**            | 子控件伸展填充剩余空间   |
| **Gap**             | 子控件之间的间距      |

### 2.2 Flex 流向 (Flex Flow)

```c
// 设置Flex流向
lv_obj_set_flex_flow(container, LV_FLEX_FLOW_ROW);
```

| 流向 | 说明 | 示意图 |
|------|------|--------|
| `LV_FLEX_FLOW_ROW` | 水平排列，不换行 | `→ → →` |
| `LV_FLEX_FLOW_COLUMN` | 垂直排列，不换行 | `↓`<br>`↓`<br>`↓` |
| `LV_FLEX_FLOW_ROW_WRAP` | 水平排列，自动换行 | `→ →`<br>`→ →` |
| `LV_FLEX_FLOW_COLUMN_WRAP` | 垂直排列，自动换列 | `↓ ↓ ↓`<br>`↓ ↓` |
| `LV_FLEX_FLOW_ROW_REVERSE` | 水平反向排列 | `← ← ←` |
| `LV_FLEX_FLOW_COLUMN_REVERSE` | 垂直反向排列 | `↑`<br>`↑`<br>`↑` |

```c
// 示例：水平排列的按钮组
lv_obj_t * btn_group = lv_obj_create(lv_screen_active());
lv_obj_set_size(btn_group, 400, 60);
lv_obj_set_flex_flow(btn_group, LV_FLEX_FLOW_ROW);

for(int i = 0; i < 5; i++) {
    lv_obj_t * btn = lv_button_create(btn_group);
    lv_obj_t * label = lv_label_create(btn);
    lv_label_set_text_fmt(label, "Btn %d", i + 1);
    lv_obj_center(label);
}
```

### 2.3 Flex 对齐 (Flex Align)
**第一个对齐参数** 整体结构 是左对齐还是右对齐如果列排布 上对齐和下对齐
**第二个对齐参数** 单个元素 是左对齐还是右对齐如果列排布上对齐和下对齐
**第三个对齐参数** 轨道布局的起始位置 是左对齐还是右对齐如果列排布 上对齐和下对齐
```c
// 设置对齐方式
lv_obj_set_flex_align(
    container,
    LV_FLEX_ALIGN_CENTER,      // main_place: 主轴对齐
    LV_FLEX_ALIGN_CENTER,      // cross_place: 交叉轴对齐
    LV_FLEX_ALIGN_START        // track_cross_place: 多行时行间距对齐
);
```

**主轴对齐 (main_place)**：

| 对齐方式 | 效果 |
|----------|------|
| `LV_FLEX_ALIGN_START` | 左对齐（ROW）/ 顶部对齐（COLUMN） |
| `LV_FLEX_ALIGN_END` | 右对齐 / 底部对齐 |
| `LV_FLEX_ALIGN_CENTER` | 居中对齐 |
| `LV_FLEX_ALIGN_SPACE_EVENLY` | 均匀分布（含边缘） |
| `LV_FLEX_ALIGN_SPACE_AROUND` | 均匀分布（边缘减半） |
| `LV_FLEX_ALIGN_SPACE_BETWEEN` | 两端对齐，中间均匀 |

```c
// 示例：按钮均匀分布
lv_obj_set_flex_align(btn_group,
    LV_FLEX_ALIGN_SPACE_EVENLY,  // 主轴均匀分布
    LV_FLEX_ALIGN_CENTER,         // 交叉轴居中
    LV_FLEX_ALIGN_START           // 行间距从顶部开始
);
```

### 2.4 Flex 伸展 (Flex Grow)

#### 什么是 Flex Grow？

**Flex Grow** 决定了子控件如何**分配父容器中剩余的空间**。

```
┌─────────────────────────────────────────────────────────┐
│  父容器总宽度: 500px                                      │
│  ┌──────────┐ ┌────────────────────┐ ┌──────────┐      │
│  │  固定宽   │ │     伸展区域        │ │  固定宽   │      │
│  │  100px   │ │   (剩余300px)       │ │  100px   │      │
│  │  grow=0  │ │    grow=1           │ │  grow=0  │      │
│  └──────────┘ └────────────────────┘ └──────────┘      │
│            ↑                                          │
│    剩余空间 = 500 - 100 - 100 = 300px                  │
│    grow=1 的子控件获得全部剩余空间                      │
└─────────────────────────────────────────────────────────┘
```

#### 核心概念

| 概念 | 说明 |
|------|------|
| **剩余空间** | 父容器大小 - 所有固定大小子控件的总和 |
| **grow值** | 决定子控件分得剩余空间的比例 |
| **grow=0** | 子控件保持固定大小，不参与分配剩余空间 |
| **grow≥1** | 子控件按比例分配剩余空间 |

#### 计算方法

```
┌─────────────────────────────────────────────────────────────┐
│  示例：父容器 500px，3个子控件                                │
│                                                             │
│  子控件A: 固定100px, grow=0  → 最终宽度 = 100px              │
│  子控件B: 固定50px, grow=1   → 最终宽度 = 50 + 剩余份额      │
│  子控件C: 固定50px, grow=2   → 最终宽度 = 50 + 剩余份额×2    │
│                                                             │
│  计算步骤：                                                   │
│  1. 总固定宽度 = 100 + 50 + 50 = 200px                      │
│  2. 剩余空间 = 500 - 200 = 300px                            │
│  3. 总grow值 = 0 + 1 + 2 = 3                                │
│  4. 每份grow = 300 ÷ 3 = 100px                              │
│                                                             │
│  最终结果：                                                   │
│  A: 100px (固定，不伸展)                                     │
│  B: 50 + 100×1 = 150px                                      │
│  C: 50 + 100×2 = 250px                                      │
│                                                             │
│  验证：100 + 150 + 250 = 500px ✓                            │
└─────────────────────────────────────────────────────────────┘
```

#### 实际示例图解

**示例1：搜索栏（输入框伸展，按钮固定）**

```
┌────────────────────────────────────────────────────────┐
│                    搜索栏 (400px)                       │
│  ┌────────────────────────────────────┐ ┌──────────┐  │
│  │          输入框                     │ │   按钮   │  │
│  │        grow = 1                     │ │ grow = 0 │  │
│  │     占据所有剩余空间                 │ │ 固定80px │  │
│  │        320px                        │ │          │  │
│  └────────────────────────────────────┘ └──────────┘  │
│                                                         │
│  计算：400px(总) - 80px(按钮) = 320px(输入框)            │
└────────────────────────────────────────────────────────┘
```

```c
lv_obj_t * search_bar = lv_obj_create(lv_screen_active());
lv_obj_set_size(search_bar, 400, 50);
lv_obj_set_flex_flow(search_bar, LV_FLEX_FLOW_ROW);

lv_obj_t * input = lv_textarea_create(search_bar);
lv_obj_set_flex_grow(input, 1);  // 输入框伸展，占据所有剩余空间

lv_obj_t * btn = lv_button_create(search_bar);
lv_obj_set_size(btn, 80, 40);    // 按钮固定大小，grow=0（默认）
```

**示例2：多个伸展控件按比例分配**

```
┌────────────────────────────────────────────────────────────┐
│              工具栏 (600px)                                 │
│  ┌──────┐ ┌──────────────┐ ┌────────────────────┐ ┌──────┐ │
│  │ 图标1 │ │    标题栏     │ │      进度条         │ │ 图标2 │ │
│  │40px  │ │   grow=1      │ │     grow=2         │ │40px  │ │
│  │固定  │ │   占1份       │ │     占2份          │ │固定  │ │
│  └──────┘ │   140px       │ │     280px          │ └──────┘ │
│           └──────────────┘ └────────────────────┘          │
│                                                             │
│  计算：                                                      │
│  固定部分：40 + 40 = 80px                                   │
│  剩余空间：600 - 80 = 520px                                 │
│  总grow：1 + 2 = 3                                          │
│  每份：520 ÷ 3 ≈ 173px                                      │
│  标题栏：173px × 1 = 173px ≈ 140px (受其他约束影响)          │
└────────────────────────────────────────────────────────────┘
```

```c
lv_obj_t * toolbar = lv_obj_create(lv_screen_active());
lv_obj_set_size(toolbar, 600, 50);
lv_obj_set_flex_flow(toolbar, LV_FLEX_FLOW_ROW);

// 左侧图标（固定）
lv_obj_t * icon1 = lv_obj_create(toolbar);
lv_obj_set_size(icon1, 40, 40);
// grow=0 默认，不伸展

// 标题栏（伸展1份）
lv_obj_t * title = lv_label_create(toolbar);
lv_label_set_text(title, "Title");
lv_obj_set_flex_grow(title, 1);

// 进度条（伸展2份，占更多空间）
lv_obj_t * progress = lv_bar_create(toolbar);
lv_obj_set_flex_grow(progress, 2);

// 右侧图标（固定）
lv_obj_t * icon2 = lv_obj_create(toolbar);
lv_obj_set_size(icon2, 40, 40);
```

**示例3：两侧固定，中间自适应**

```
┌─────────────────────────────────────────────────────────────┐
│                 三栏布局 (500px)                             │
│  ┌──────────┐ ┌──────────────────────────┐ ┌──────────┐    │
│  │          │ │                          │ │          │    │
│  │  侧边栏   │ │        主内容区           │ │  侧边栏   │    │
│  │  固定    │ │        grow=1            │ │   固定   │    │
│  │  100px   │ │       占据剩余            │ │  100px   │    │
│  │          │ │       300px              │ │          │    │
│  └──────────┘ └──────────────────────────┘ └──────────┘    │
│                                                             │
│  常用于：侧边导航 + 主内容区 + 辅助信息栏                      │
└─────────────────────────────────────────────────────────────┘
```

```c
lv_obj_t * layout = lv_obj_create(lv_screen_active());
lv_obj_set_size(layout, 500, 300);
lv_obj_set_flex_flow(layout, LV_FLEX_FLOW_ROW);

// 左侧边栏（固定宽度）
lv_obj_t * left_sidebar = lv_obj_create(layout);
lv_obj_set_size(left_sidebar, 100, lv_pct(100));

// 主内容区（伸展）
lv_obj_t * main_content = lv_obj_create(layout);
lv_obj_set_flex_grow(main_content, 1);
lv_obj_set_height(main_content, lv_pct(100));

// 右侧边栏（固定宽度）
lv_obj_t * right_sidebar = lv_obj_create(layout);
lv_obj_set_size(right_sidebar, 100, lv_pct(100));
```

#### 常见用法总结

| 场景 | grow设置 | 效果 |
|------|----------|------|
| **一侧固定，另一侧自适应** | 固定侧grow=0，自适应侧grow=1 | 经典搜索栏布局 |
| **均匀分配剩余空间** | 所有子控件grow=1 | 等宽布局 |
| **按比例分配** | 按需求设置grow=1,2,3... | 主内容占多，侧边占少 |
| **全部固定** | 所有grow=0 | 传统固定布局 |

#### 注意事项

```c
// ❌ 错误：设置了grow又设置固定大小（冲突）
lv_obj_set_flex_grow(obj, 1);
lv_obj_set_width(obj, 100);  // 会被grow覆盖！

// ✅ 正确：grow用于控制尺寸时，不需要设置固定大小
lv_obj_set_flex_grow(obj, 1);
// 宽度会根据剩余空间自动计算

// ❌ 错误：所有子控件都grow=0，父容器有剩余空间
// 剩余空间会空着，不会自动分配

// ✅ 正确：至少有一个子控件设置grow≥1来填充空间
lv_obj_set_flex_grow(at_least_one_child, 1);
```

### 2.5 Flex 间距 (Gap)

```c
// 设置子控件间距
lv_obj_set_style_pad_row(container, 10, 0);     // 行间距
lv_obj_set_style_pad_column(container, 20, 0);  // 列间距
```

```c
// 示例：带间距的卡片列表
lv_obj_t * card_list = lv_obj_create(lv_screen_active());

}
```

**FR单位计算详解：**

```
场景：容器宽度600px，列定义 {100, LV_GRID_FR(1), LV_GRID_FR(3), LV_GRID_FR(2)}

┌──────────────────────────────────────────────────────────────┐
│                                                              │
│  步骤1：计算固定列宽度                                        │
│  固定列：100px                                               │
│                                                              │
│  步骤2：计算剩余空间                                          │
│  剩余 = 600 - 100 = 500px                                    │
│                                                              │
│  步骤3：计算FR分配                                            │
│  总FR = 1 + 3 + 2 = 6                                        │
│  每份 = 500 ÷ 6 ≈ 83.33px                                    │
│                                                              │
│  步骤4：计算各列宽度                                          │
│  Col 0: 100px (固定)                                         │
│  Col 1: 83.33 × 1 = 83px                                     │
```

## 三、Grid 网格布局

### 3.1 Grid 基础概念

```
Grid布局示意图：

┌─────────┬─────────┬─────────┐
│  (0,0)  │  (1,0)  │  (2,0)  │  ← Row 0
│  100px  │  1fr    │  2fr    │
├─────────┼─────────┼─────────┤
│  (0,1)  │  (1,1)  │  (2,1)  │  ← Row 1
│         │  span   │  span   │
│         │  2 cols │  1 col  │
├─────────┼─────────┼─────────┤
│  (0,2)  │  (1,2)  │  (2,2)  │  ← Row 2
└─────────┴─────────┴─────────┘
   Col 0     Col 1     Col 2
  100px     1fr       2fr
```

| 术语 | 说明 |
|------|------|
| **Track** | 行或列 |
| **FR单位** | 分数单位，按比例分配剩余空间 |
| **Cell** | 单元格，行和列的交叉区域 |
| **Span** | 跨越多行或多列 |
| **column_pos** | 列索引，从0开始 |
| **row_pos** | 行索引，从0开始 |
| **column_span** | 跨越列数，最小为1 |
| **row_span** | 跨越行数，最小为1 |

### 3.2 定义行列尺寸

```c
// 定义列：100px固定 | 剩余空间的1份 | 剩余空间的2份
static int32_t column_dsc[] = {100, LV_GRID_FR(1), LV_GRID_FR(2), LV_GRID_TEMPLATE_LAST};

// 定义行：3行，每行50px
static int32_t row_dsc[] = {50, 50, 50, LV_GRID_TEMPLATE_LAST};

// 应用到容器
lv_obj_set_grid_dsc_array(container, column_dsc, row_dsc);
```

| 尺寸类型 | 写法 | 说明 |
|----------|------|------|
| 固定像素 | `100` | 固定100像素 |
| 内容自适应 | `LV_GRID_CONTENT` | 根据最大子控件自动调整 |
| 比例分配 | `LV_GRID_FR(1)` | 按FR值比例分配剩余空间 |

```c
    lv_obj_t * header = lv_obj_create(main);
    lv_obj_set_size(header, lv_pct(100), 40);
    lv_obj_set_flex_flow(header, LV_FLEX_FLOW_ROW);
    lv_obj_set_flex_align(header, LV_FLEX_ALIGN_SPACE_BETWEEN, LV_FLEX_ALIGN_CENTER, LV_FLEX_ALIGN_START);
    
    lv_obj_t * title = lv_label_create(header);
    lv_label_set_text(title, "Dashboard");
    lv_obj_set_style_text_font(title, &lv_font_montserrat_18, 0);
    
    lv_obj_t * time_label = lv_label_create(header);
    lv_label_set_text(time_label, "12:00");
    
    // 内容区 - Grid布局
    lv_obj_t * content = lv_obj_create(main);
    lv_obj_set_size(content, lv_pct(100), lv_pct(100));
    lv_obj_set_flex_grow(content, 1);  // 占据所有剩余空间
    
    static int32_t col_dsc[] = {LV_GRID_FR(1), LV_GRID_FR(1), LV_GRID_TEMPLATE_LAST};
    static int32_t row_dsc[] = {LV_GRID_FR(1), LV_GRID_FR(1), LV_GRID_TEMPLATE_LAST};
    lv_obj_set_grid_dsc_array(content, col_dsc, row_dsc);
    lv_obj_set_style_pad_all(content, 10, 0);
    lv_obj_set_style_pad_row(content, 10, 0);
    lv_obj_set_style_pad_column(content, 10, 0);
    
    // 4个数据卡片
    const char * card_titles[] = {"Temperature", "Humidity", "Pressure", "Battery"};
    const char * card_values[] = {"25.3°C", "68%", "1013hPa", "85%"};
    lv_color_t card_colors[] = {
        lv_palette_main(LV_PALETTE_RED),
        lv_palette_main(LV_PALETTE_BLUE),
        lv_palette_main(LV_PALETTE_GREEN),
        lv_palette_main(LV_PALETTE_ORANGE)
    };
    
    for(int i = 0; i < 4; i++) {
        lv_obj_t * card = lv_obj_create(content);
        lv_obj_set_grid_cell(card, LV_GRID_ALIGN_STRETCH, i % 2, 1, LV_GRID_ALIGN_STRETCH, i / 2, 1);
        lv_obj_set_style_bg_color(card, card_colors[i], 0);
        lv_obj_set_style_bg_opa(card, LV_OPA_20, 0);
        lv_obj_set_style_radius(card, 15, 0);
        lv_obj_set_style_border_width(card, 2, 0);
        lv_obj_set_style_border_color(card, card_colors[i], 0);
        
        // 卡片内部垂直Flex
        lv_obj_set_flex_flow(card, LV_FLEX_FLOW_COLUMN);
        lv_obj_set_flex_align(card, LV_FLEX_ALIGN_CENTER, LV_FLEX_ALIGN_CENTER, LV_FLEX_ALIGN_CENTER);
        
        lv_obj_t * title = lv_label_create(card);
        lv_label_set_text(title, card_titles[i]);
        lv_obj_set_style_text_color(title, card_colors[i], 0);
        
        lv_obj_t * value = lv_label_create(card);
        lv_label_set_text(value, card_values[i]);
        lv_obj_set_style_text_font(value, &lv_font_montserrat_24, 0);
        lv_obj_set_style_text_color(value, card_colors[i], 0);
    }
```

## 附录：LVGL 布局 API 速查表

### Flex API

| 函数 | 说明 |
|------|------|
| `lv_obj_set_layout(obj, LV_LAYOUT_FLEX)` | 启用Flex布局 |
| `lv_obj_set_flex_flow(obj, flow)` | 设置流向 |
| `lv_obj_set_flex_align(obj, main, cross, track)` | 设置对齐 |
| `lv_obj_set_flex_grow(child, value)` | 设置伸展值 |
| `lv_obj_set_style_pad_row(obj, value, selector)` | 设置行间距 |
| `lv_obj_set_style_pad_column(obj, value, selector)` | 设置列间距 |

### Flex 流向

| 常量 | 说明 |
|------|------|
| `LV_FLEX_FLOW_ROW` | 水平排列 |
| `LV_FLEX_FLOW_COLUMN` | 垂直排列 |
| `LV_FLEX_FLOW_ROW_WRAP` | 水平排列+换行 |
| `LV_FLEX_FLOW_COLUMN_WRAP` | 垂直排列+换列 |
| `LV_FLEX_FLOW_ROW_REVERSE` | 水平反向 |
| `LV_FLEX_FLOW_COLUMN_REVERSE` | 垂直反向 |

### Flex 对齐

| 常量 | 说明 |
|------|------|
| `LV_FLEX_ALIGN_START` | 起始对齐 |
| `LV_FLEX_ALIGN_END` | 末尾对齐 |
| `LV_FLEX_ALIGN_CENTER` | 居中对齐 |
| `LV_FLEX_ALIGN_SPACE_EVENLY` | 均匀分布（含边缘） |
| `LV_FLEX_ALIGN_SPACE_AROUND` | 均匀分布（边缘减半） |
| `LV_FLEX_ALIGN_SPACE_BETWEEN` | 两端对齐 |

### Grid API

| 函数 | 说明 |
|------|------|
| `lv_obj_set_layout(obj, LV_LAYOUT_GRID)` | 启用Grid布局 |
| `lv_obj_set_grid_dsc_array(obj, col_dsc, row_dsc)` | 设置行列描述 |
| `lv_obj_set_grid_cell(child, col_align, col_pos, col_span, row_align, row_pos, row_span)` | 放置子控件 |
| `lv_obj_set_grid_align(obj, col_align, row_align)` | 设置对齐 |

### Grid 尺寸类型

| 常量 | 说明 | 示例 |
|------|------|------|
| `100` | 固定像素 | `100` = 100px |
| `LV_GRID_CONTENT` | 内容自适应 | 根据子控件自动调整 |
| `LV_GRID_FR(1)` | 比例分配 | `LV_GRID_FR(2)` = 2份 |
| `LV_GRID_TEMPLATE_LAST` | 描述数组结束标记 | 必须放在最后 |

### Grid 对齐

| 常量 | 说明 |
|------|------|
| `LV_GRID_ALIGN_START` | 起始对齐 |
| `LV_GRID_ALIGN_END` | 末尾对齐 |
| `LV_GRID_ALIGN_CENTER` | 居中对齐 |
| `LV_GRID_ALIGN_STRETCH` | 拉伸填满 |
| `LV_GRID_ALIGN_SPACE_EVENLY` | 均匀分布 |

### 布局选择指南

| 场景 | 推荐布局 | 原因 |
|------|----------|------|
| 按钮组、导航栏 | Flex ROW | 一维水平排列 |
| 列表、卡片堆叠 | Flex COLUMN | 一维垂直排列 |
| 表单、登录界面 | Grid | 标签+输入框对齐 |
| 计算器、仪表盘 | Grid | 严格的行列对齐 |
| 复杂响应式界面 | Flex + Grid 组合 | 灵活且精确 |
