# Color (lv_color) （颜色系统：RGB/HSV/调色板统一处理）

> 颜色模块处理所有与颜色相关的功能，如更改颜色深度、从十六进制代码创建颜色、在不同颜色深度之间转换、混合颜色等。

---

## 核心概念

### 颜色存储格式
- ==`lv_color_t`== - LVGL统一使用的颜色类型，内部以 **RGB888** 格式存储
- ==颜色深度无关== - 无论 `LV_COLOR_DEPTH` 设置为何值，`lv_color_t` 都使用 RGB888

### 颜色创建方式
- ==RGB模式== - 通过红绿蓝三通道直接创建
- ==HSV模式== - 通过色相/饱和度/明度创建，适合渐变动画
- ==调色板模式== - 使用 Material Design 预设颜色，快速构建UI配色

### 透明度系统
- ==`lv_opa_t`== - 透明度类型，本质为 `uint8_t`（0-255）
- ==混合计算== - `lv_color_mix()` 支持按比率混合两种颜色

## 一、RGB颜色创建

### 1.1 通过RGB通道创建

**功能说明**：使用红(0-255)、绿(0-255)、蓝(0-255)三通道值创建颜色

```c
// 运行时创建颜色
lv_color_t c = lv_color_make(red, green, blue);

// 编译时常量初始化
lv_color_t c = LV_COLOR_MAKE(red, green, blue);
```

| 参数 | 类型 | 范围 | 说明 |
|------|------|------|------|
| `red` | `uint8_t` | 0-255 | 红色通道值 |
| `green` | `uint8_t` | 0-255 | 绿色通道值 |
| `blue` | `uint8_t` | 0-255 | 蓝色通道值 |

**返回值**: `lv_color_t` - 创建的颜色值

**示例**：
```c
// 创建纯红色
lv_color_t red = lv_color_make(255, 0, 0);

// 创建白色（常量初始化）
static const lv_color_t white = LV_COLOR_MAKE(255, 255, 255);
```

### 1.2 通过十六进制创建

**功能说明**：从十六进制颜色代码创建颜色

```c
// 6位十六进制: 0x000000 ~ 0xFFFFFF
lv_color_t c = lv_color_hex(0x123456);

// 3位简写: 0x123 等同于 0x112233
lv_color_t c = lv_color_hex3(0x123);
```

| 参数 | 类型 | 格式 | 说明 |
|------|------|------|------|
| `hex` | `uint32_t` | 0xRRGGBB | 6位RGB十六进制值 |
| `hex3` | `uint16_t` | 0xRGB | 3位简写，自动展开为6位 |

**示例**：
```c
// 天蓝色 #87CEEB
lv_color_t sky_blue = lv_color_hex(0x87CEEB);

// 红色 #FF0000 的简写
lv_color_t red = lv_color_hex3(0xF00);  // 等同于 0xFF0000
```

### 1.3 内置颜色快捷函数

```c
lv_color_t white = lv_color_white();   // 0xFFFFFF
lv_color_t black = lv_color_black();   // 0x000000
```

## 二、HSV颜色创建

### 2.1 HSV转RGB

**功能说明**：将色相/饱和度/明度转换为RGB颜色，适合动态颜色计算

```c
// h: 0~359, s: 0~100, v: 0~100
lv_color_t c = lv_color_hsv_to_rgb(h, s, v);
```

| 参数 | 类型 | 范围 | 说明 |
|------|------|------|------|
| `h` | `uint16_t` | 0-359 | 色相(Hue)，0=红，120=绿，240=蓝 |
| `s` | `uint8_t` | 0-100 | 饱和度(Saturation)，0=灰度，100=纯色 |
| `v` | `uint8_t` | 0-100 | 明度(Value)，0=黑色，100=最亮 |

**返回值**: `lv_color_t` - 转换后的RGB颜色

**应用场景**：彩虹渐变、颜色动画、主题色动态调整

```c
// 创建彩虹色循环
for(int h = 0; h < 360; h += 30) {
    lv_color_t rainbow = lv_color_hsv_to_rgb(h, 100, 100);
    // 使用 rainbow 设置控件颜色...
}
```

### 2.2 RGB转HSV

```c
// 从RGB通道转换
lv_color_hsv_t c_hsv = lv_color_rgb_to_hsv(r, g, b);

// 从 lv_color_t 变量转换
lv_color_hsv_t c_hsv = lv_color_to_hsv(color);
```

| 参数/返回值 | 类型 | 说明 |
|-------------|------|------|
| `r, g, b` | `uint8_t` | RGB通道值(0-255) |
| `color` | `lv_color_t` | 源颜色 |
| `返回值` | `lv_color_hsv_t` | HSV结构体，含 h, s, v 字段 |

## 三、Material Design 调色板

### 3.1 可用的调色板颜色

LVGL内置 [Material Design 调色板](https://vuetifyjs.com/en/styles/colors/#material-colors)，每种颜色包含：
- 1个主色(nominal)
- 5个浅色变体(lighten 1-5)
- 4个深色变体(darken 1-4)

**可用颜色常量**：

| 常量 | 颜色名称 | 常量 | 颜色名称 |
|------|----------|------|----------|
| `LV_PALETTE_RED` | 红色 | `LV_PALETTE_GREEN` | 绿色 |
| `LV_PALETTE_PINK` | 粉色 | `LV_PALETTE_LIGHT_GREEN` | 浅绿 |
| `LV_PALETTE_PURPLE` | 紫色 | `LV_PALETTE_LIME` | 柠檬黄 |
| `LV_PALETTE_DEEP_PURPLE` | 深紫 | `LV_PALETTE_YELLOW` | 黄色 |
| `LV_PALETTE_INDIGO` | 靛蓝 | `LV_PALETTE_AMBER` | 琥珀色 |
| `LV_PALETTE_BLUE` | 蓝色 | `LV_PALETTE_ORANGE` | 橙色 |
| `LV_PALETTE_LIGHT_BLUE` | 浅蓝 | `LV_PALETTE_DEEP_ORANGE` | 深橙 |
| `LV_PALETTE_CYAN` | 青色 | `LV_PALETTE_BROWN` | 棕色 |
| `LV_PALETTE_TEAL` | 蓝绿 | `LV_PALETTE_BLUE_GREY` | 蓝灰 |
| - | - | `LV_PALETTE_GREY` | 灰色 |

### 3.2 获取调色板颜色

```c
// 获取主色
lv_color_t main = lv_palette_main(LV_PALETTE_BLUE);

// 获取浅色变体 (v = 1~5，数字越大越浅)
lv_color_t light = lv_palette_lighten(LV_PALETTE_BLUE, 3);

// 获取深色变体 (v = 1~4，数字越大越深)
lv_color_t dark = lv_palette_darken(LV_PALETTE_BLUE, 2);
```

| 函数 | 参数 | 返回值 | 说明 |
|------|------|--------|------|
| `lv_palette_main()` | `lv_palette_t` | `lv_color_t` | 获取主色 |
| `lv_palette_lighten()` | 调色板, `v`(1-5) | `lv_color_t` | 获取浅色变体 |
| `lv_palette_darken()` | 调色板, `v`(1-4) | `lv_color_t` | 获取深色变体 |

**示例**：构建主色+强调色的配色方案
```c
// 主题色：蓝色系
lv_color_t primary = lv_palette_main(LV_PALETTE_BLUE);
lv_color_t primary_light = lv_palette_lighten(LV_PALETTE_BLUE, 2);
lv_color_t primary_dark = lv_palette_darken(LV_PALETTE_BLUE, 2);

// 强调色：琥珀色
lv_color_t accent = lv_palette_main(LV_PALETTE_AMBER);
```

## 四、颜色修改与混合

### 4.1 颜色变亮/变暗

```c
// 变亮: 0=无变化, 255=纯白
lv_color_t lighter = lv_color_lighten(c, lvl);

// 变暗: 0=无变化, 255=纯黑
lv_color_t darker = lv_color_darken(c, lvl);

// 同时调整亮度: 0=黑, 128=原色, 255=白
lv_color_t adjusted = lv_color_change_lightness(c, lvl);
```

| 参数 | 类型 | 范围 | 说明 |
|------|------|------|------|
| `c` | `lv_color_t` | - | 源颜色 |
| `lvl` | `lv_opa_t` | 0-255 | 调整程度 |

### 4.2 颜色混合

```c
// 按比率混合两种颜色
lv_color_t mixed = lv_color_mix(c1, c2, ratio);
```

| 参数 | 类型 | 范围 | 说明 |
|------|------|------|------|
| `c1` | `lv_color_t` | - | 第一种颜色 |
| `c2` | `lv_color_t` | - | 第二种颜色 |
| `ratio` | `uint8_t` | 0-255 | 混合比率：0=全c2, 255=全c1, 128=各半 |

**返回值**: `lv_color_t` - 混合后的颜色

**示例**：
```c
lv_color_t red = lv_color_make(255, 0, 0);
lv_color_t blue = lv_color_make(0, 0, 255);

// 紫色 (红蓝各半混合)
lv_color_t purple = lv_color_mix(red, blue, 128);

// 偏红的紫 (红占75%)
lv_color_t reddish = lv_color_mix(red, blue, 192);
```

## 六、在控件中应用颜色

### 6.1 样式颜色属性概览

LVGL 控件通过 **样式（Style）** 系统设置颜色，主要颜色属性包括：

| 样式属性 | 作用对象 | 说明 |
|----------|----------|------|
| `bg_color` | 背景 | 控件背景填充色 |
| `bg_grad_color` | 背景渐变 | 渐变结束颜色（配合 `bg_main_color`） |
| `text_color` | 文字 | 标签、按钮文字颜色 |
| `border_color` | 边框 | 控件边框颜色 |
| `shadow_color` | 阴影 | 控件阴影颜色 |
| `outline_color` | 外轮廓 | 聚焦时外圈颜色 |
| `arc_color` | 圆弧 | 弧形进度条颜色 |
| `image_recolor` | 图片重着色 | 对图片进行颜色叠加 |

### 6.2 局部样式设置（一次性）

使用 `lv_obj_set_style_xxx()` 函数为单个控件设置颜色：

```c
lv_obj_t *btn = lv_btn_create(lv_scr_act());

// 设置背景颜色
lv_obj_set_style_bg_color(btn, lv_palette_main(LV_PALETTE_BLUE), 0);

// 设置文字颜色
lv_obj_set_style_text_color(btn, lv_color_white(), 0);

// 设置边框颜色
lv_obj_set_style_border_color(btn, lv_palette_darken(LV_PALETTE_BLUE, 2), 0);
lv_obj_set_style_border_width(btn, 2, 0);

// 设置圆角
lv_obj_set_style_radius(btn, 10, 0);
```

| 参数 | 说明 |
|------|------|
| 第1个参数 | 目标控件对象 |
| 第2个参数 | 颜色值（`lv_color_t`） |
| 第3个参数 | 选择器（`0` 表示默认状态） |

### 6.3 状态样式（不同状态不同颜色）

控件在不同状态下可以显示不同颜色：

```c
lv_obj_t *btn = lv_btn_create(lv_scr_act());

// 默认状态：蓝色背景，白色文字
lv_obj_set_style_bg_color(btn, lv_palette_main(LV_PALETTE_BLUE), LV_STATE_DEFAULT);
lv_obj_set_style_text_color(btn, lv_color_white(), LV_STATE_DEFAULT);

// 按下状态：深蓝背景
lv_obj_set_style_bg_color(btn, lv_palette_darken(LV_PALETTE_BLUE, 2), LV_STATE_PRESSED);

// 聚焦状态：添加橙色边框
lv_obj_set_style_border_color(btn, lv_palette_main(LV_PALETTE_ORANGE), LV_STATE_FOCUSED);
lv_obj_set_style_border_width(btn, 3, LV_STATE_FOCUSED);

// 禁用状态：灰色背景
lv_obj_set_style_bg_color(btn, lv_palette_main(LV_PALETTE_GREY), LV_STATE_DISABLED);
lv_obj_set_style_text_color(btn, lv_color_hex(0xAAAAAA), LV_STATE_DISABLED);
```

**常用状态常量**：

| 状态常量 | 说明 |
|----------|------|
| `LV_STATE_DEFAULT` | 默认状态 |
| `LV_STATE_CHECKED` | 选中状态 |
| `LV_STATE_FOCUSED` | 聚焦状态 |
| `LV_STATE_PRESSED` | 按下状态 |
| `LV_STATE_DISABLED` | 禁用状态 |
| `LV_STATE_HOVERED` | 悬停状态（鼠标） |

### 6.4 全局样式（复用样式）

创建可复用的样式对象，应用到多个控件：

```c
// 定义全局样式变量
static lv_style_t style_btn_primary;
static lv_style_t style_btn_pressed;

void init_styles(void)
{
    // 初始化主按钮样式（蓝色主题）
    lv_style_init(&style_btn_primary);
    lv_style_set_bg_color(&style_btn_primary, lv_palette_main(LV_PALETTE_BLUE));
    lv_style_set_text_color(&style_btn_primary, lv_color_white());
    lv_style_set_border_color(&style_btn_primary, lv_palette_darken(LV_PALETTE_BLUE, 2));
    lv_style_set_border_width(&style_btn_primary, 2);
    lv_style_set_radius(&style_btn_primary, 8);
    
    // 初始化按下状态样式
    lv_style_init(&style_btn_pressed);
    lv_style_set_bg_color(&style_btn_pressed, lv_palette_darken(LV_PALETTE_BLUE, 3));
}

void create_buttons(void)
{
    // 创建按钮1并应用样式
    lv_obj_t *btn1 = lv_btn_create(lv_scr_act());
    lv_obj_add_style(btn1, &style_btn_primary, 0);
    lv_obj_add_style(btn1, &style_btn_pressed, LV_STATE_PRESSED);
    
    // 创建按钮2复用相同样式
    lv_obj_t *btn2 = lv_btn_create(lv_scr_act());
    lv_obj_add_style(btn2, &style_btn_primary, 0);
    lv_obj_add_style(btn2, &style_btn_pressed, LV_STATE_PRESSED);
}
```

## 七、完整示例

### 示例1：创建渐变色背景

```c
#include "lvgl.h"

void create_gradient_bg(lv_obj_t *parent)
{
    // 使用 HSV 创建从蓝到紫的渐变背景
    lv_obj_t *bg = lv_obj_create(parent);
    lv_obj_set_size(bg, 200, 100);
    lv_obj_set_style_bg_color(bg, lv_color_hsv_to_rgb(240, 80, 90), 0); // 蓝色
    
    // 使用调色板快速设置边框颜色
    lv_obj_set_style_border_color(bg, lv_palette_main(LV_PALETTE_PURPLE), 0);
    lv_obj_set_style_border_width(bg, 3, 0);
}
```

### 示例2：动态主题色切换

```c
#include "lvgl.h"

static lv_color_t current_primary;
static lv_color_t current_secondary;

void set_theme_colors(lv_palette_t primary, lv_palette_t secondary)
{
    current_primary = lv_palette_main(primary);
    current_secondary = lv_palette_main(secondary);
    
    // 应用到全局样式
    lv_style_t *style = lv_theme_get_style_primary();
    lv_style_set_bg_color(style, current_primary);
}

void apply_button_style(lv_obj_t *btn)
{
    // 正常状态：主色
    lv_obj_set_style_bg_color(btn, current_primary, LV_STATE_DEFAULT);
    
    // 按下状态：主色加深
    lv_obj_set_style_bg_color(btn, 
        lv_color_darken(current_primary, LV_OPA_20), 
        LV_STATE_PRESSED);
    
    // 禁用状态：主色变灰
    lv_obj_set_style_bg_color(btn, 
        lv_color_mix(lv_palette_main(LV_PALETTE_GREY), current_primary, LV_OPA_50),
        LV_STATE_DISABLED);
}
```

### 示例3：十六进制颜色解析器

```c
#include "lvgl.h"

// 从字符串解析十六进制颜色 (支持 #RGB 和 #RRGGBB)
lv_color_t parse_hex_color(const char *hex_str)
{
    if(hex_str[0] != '#') {
        return lv_color_black();
    }
    
    uint32_t hex_val = strtol(hex_str + 1, NULL, 16);
    
    if(strlen(hex_str) == 4) {  // #RGB format
        return lv_color_hex3(hex_val);
    } else {  // #RRGGBB format
        return lv_color_hex(hex_val);
    }
}

void example_usage(void)
{
    lv_color_t c1 = parse_hex_color("#FF5733");  // 6位格式
    lv_color_t c2 = parse_hex_color("#F53");     // 3位简写
}
```

## 注意事项与最佳实践

### 颜色深度注意事项

| 注意点 | 说明 | 建议 |
|--------|------|------|
| **颜色深度设置** | `LV_COLOR_DEPTH` 可在 1/8/16/32 位之间选择 | 根据硬件资源选择合适的深度 |
| **16位色深** | 默认推荐，平衡质量和内存 | RGB565 格式，注意色带问题 |
| **32位色深** | 最高质量，但占用内存大 | 仅在高端硬件上使用 |
| **8位色深** | 低内存占用，质量较差 | 简单UI或低端设备使用 |

### 颜色创建常见错误

```c
// ❌ 错误：直接使用十六进制数值
lv_color_t c = 0xFF5733;  // 编译错误！

// ✅ 正确：使用颜色创建函数
lv_color_t c = lv_color_hex(0xFF5733);

// ❌ 错误：RGB值超出范围
lv_color_t c = lv_color_make(300, 400, 500);  // 值超出255

// ✅ 正确：RGB值在0-255范围内
lv_color_t c = lv_color_make(255, 87, 51);

// ❌ 错误：混淆16进制格式
lv_color_t c = lv_color_hex(0xF53);  // 3位格式，结果不正确

// ✅ 正确：6位格式用于标准颜色
lv_color_t c = lv_color_hex(0xFF5733);

// ✅ 正确：3位格式用于简写
lv_color_t c = lv_color_hex3(0xF53);  // 等同于0xFF5533
```

### 性能优化建议

| 场景 | 建议 |
|------|------|
| **静态颜色** | 使用 `LV_COLOR_MAKE` 编译时常量 |
| **动态颜色** | 预计算常用颜色，避免运行时重复创建 |
| **渐变动画** | 使用 HSV 模式计算中间色，再转为RGB |
| **透明度混合** | 频繁混合时考虑缓存结果 |

### 颜色对比度

```c
// 根据背景色自动选择文字颜色（黑或白）
lv_color_t bg = lv_color_hex(0x333333);
lv_color_t text_color = lv_color_brightness(bg) < 128 ? 
                        lv_color_white() : lv_color_black();
```

**建议**：文字与背景对比度至少为 4.5:1，确保可读性。

### 常见问题

**Q: 颜色显示不正确？**
A: 检查：1) `LV_COLOR_DEPTH` 设置；2) 显示屏颜色格式；3) 是否使用了正确的颜色创建函数。

**Q: 如何在不同颜色深度的设备间移植代码？**
A: 始终使用 `lv_color_t` 类型和LVGL提供的颜色函数，不要直接操作颜色值。

**Q: 透明度如何影响颜色混合？**
A: `lv_color_mix()` 的 ratio 参数是百分比（0-255），不是0-100。
