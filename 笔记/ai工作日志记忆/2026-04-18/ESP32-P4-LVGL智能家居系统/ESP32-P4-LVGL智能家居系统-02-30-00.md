### 关键决策
- 使用 LVGL 内置思源黑体而非 FreeType 运行时加载，避免增加复杂度和内存开销
- 所有 CJK 字体大小统一使用 14px 和 16px 两个版本
- emoji 替换为 LVGL 符号常量（LV_SYMBOL_CHARGE/DOWNLOAD/SETTINGS），因为 CJK 字体不包含 emoji

### 遇到的问题
- **问题1**: 从 MSYS2 bash 环境无法烧录，串口访问超时/端口忙
- **根因**: MSYS2 的 POSIX 层与 Windows 串口驱动不兼容
- **解决**: 使用 python -m esptool 直接调用，COM28 端口可用

- **问题2**: idf.py 在 MSYS2 环境下无法正常运行
- **根因**: 缺少 esp_idf_monitor 模块，且 MSYS 环境变量干扰
- **解决**: 创建 run_idf.py 清理环境变量，使用 ESP-IDF venv Python

- **问题3**: 中文显示为"口口口"方框
- **根因**: Montserrat 字体仅支持拉丁字符集
- **解决**: 启用 Source Han Sans SC CJK 字体，全局替换字体引用

- **问题4**: LV_SYMBOL_LIGHT 编译错误
- **根因**: LVGL v9 中不存在此符号常量
- **解决**: 替换为 LV_SYMBOL_CHARGE
