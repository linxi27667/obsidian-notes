# GUI Guider UI 设计任务计划书

## 项目概览

**屏幕参数**: 1024x600 (MIPI DSI)
**触摸**: GT911 I2C 触摸控制器
**目标平台**: ESP32-P4 + ESP-IDF v5.5.3 + LVGL v9.4.0
**集成方式**: GUI Guider 1.10.1 生成代码 → 移植到现有 ESP-IDF 项目

---

## 整体布局架构

### 屏幕层级结构
```
lv_scr_act() (根屏幕, 背景色: #ECF0F1)
├── 状态栏 (顶部, 宽100%, 高40px, 背景: #2C3E50)
│   ├── 时间标签 (左对齐, 距左15px, 白色文字, 16px)
│   ├── 温度标签 (右侧, 距右100px, 白色文字, 16px, 显示"25C")
│   └── WiFi图标 (右侧, 距右50px, 16px, 颜色动态: 已连接#2ECC71/未连接#E74C3C)
│
└── TabView (tab栏在底部, 高70px, 背景: #34495E)
    ├── Tab 1: 首页 (图标: LV_SYMBOL_HOME "🏠")
    ├── Tab 2: 设备 (图标: LV_SYMBOL_SETTINGS "⚙")
    ├── Tab 3: 场景 (图标: LV_SYMBOL_EYE_OPEN "👁")
    ├── Tab 4: WiFi (图标: LV_SYMBOL_WIFI "📶")
    └── Tab 5: 设置 (图标: LV_SYMBOL_SETTINGS "⛭")
```

### 全局样式规范
- **页面背景色**: #ECF0F1 (浅灰)
- **卡片背景色**: #FFFFFF (纯白)
- **卡片圆角**: 12px
- **主色调**: #3498DB (蓝色)
- **成功色**: #2ECC71 (绿色)
- **警告色**: #E74C3C (红色)
- **文字主色**: #2C3E50 (深蓝灰)
- **文字副色**: #7F8C8D (灰色)
- **字体**: 思源黑体 (Source Han Sans SC), 14px 和 16px 两种字号
- **中文支持**: 必须包含所有UI中出现的中文字符

## GUI Guider 创建步骤提示

1. **新建项目**:
   - Board: 选择 "Custom" 或 "ESP32"
   - Resolution: 1024 x 600
   - LVGL Version: v9.x

2. **设置全局字体**:
   - 添加 14px 和 16px 两种中文字体
   - 确保覆盖所有中文Unicode范围

3. **创建页面结构**:
   - 主屏幕 (Main Screen) → 添加状态栏 → 添加 TabView
   - TabView 设置 tab_bar_position = BOTTOM, tab_bar_size = 70
   - 添加 5 个 Tab

4. **逐个 Tab 设计**:
   - 按照上述 5 页的详细规格逐个放置控件
   - 注意坐标、尺寸、颜色、字体的一致性

5. **导出代码**:
   - 生成 C 代码 (ESP-IDF 目标)
   - 导出后我来负责集成到现有项目中
