### 关键决策
- 选择 lv_font_conv 生成 C 文件而非 FreeType：无需 SD 卡/文件系统，字体直接编译进固件，启动速度快，无运行时加载风险
- 仅提取 UI 实际使用的 204 个汉字而非完整字体集：将 ~10MB TTF 缩减到 343KB C 文件，大幅节省 Flash 空间
- 字体占用 343KB Flash 在可接受范围内（固件总大小 ~1MB，分区 8MB，剩余充足）

### 遇到的问题
- **问题1**: npm 安装 lv_font_conv 时报 ETIMEDOUT
  - **根因**: npm 配置了无效的本地代理 http://127.0.0.1:20055
  - **解决**: `npm config delete proxy` 和 `npm config delete https-proxy`

- **问题2**: CMakeLists.txt 被覆盖为 demo 项目格式（引用 guider_gen/ 目录）
  - **根因**: 之前操作意外覆盖
  - **解决**: 重写为正确的格式，包含自定义字体源文件

- **问题3**: main.c 和 task_ui.c 引用了不存在的 Guider 头文件
  - **根因**: 项目从 SquareLine GUI Guider 模板创建但未使用 Guider 生成代码
  - **解决**: 移除无用引用，task_ui.c 中用 lv_timer_handler() 替代 Guider_UI_Loop()
