## 一、配置参数

```c
// main.h
#define MAX_HEIGHT_MM   4000          // 最大高度 4m（可修改）
#define MAX_PULSES      (MAX_HEIGHT_MM / 6)  // 丝杆导程 6mm/脉冲

// app_w25qxx.c → g_config
screw_lead_mm   = 6       // 每脉冲 6mm
max_pulses      = MAX_PULSES  // 上限位脉冲数
stall_timeout_ms = 2000   // 堵转判定 2s
tolerance_up    = 4       // 上升允差 4脉冲
tolerance_down  = 4       // 下降允差 4脉冲
```

## 二、任务执行流程（伪代码）

### Control_Task — 10ms

```
loop 10ms:
  if alarm_state != NONE → skip

  // 冲突检测
  if UP+DOWN both pressed && moving → Motor_Stop_All(), "Status: STOP (conflict)"

  // 松手停
  if UP released && dir==UP → Motor_Stop_All(), "Status: STOP"
  if DOWN released && dir==DOWN → Motor_Stop_All(), "Status: STOP"

  // 启动
  if UP pressed && dir==STOP:
    if at_upper_limit → "At top (XXXXmm)", skip
    else → Motor_Start_All(UP), "Status: UP"
  if DOWN pressed && dir==STOP:
    if at_lower_limit → "At bottom", skip
    else → Motor_Start_All(DOWN), Buzzer_Beep(500ms), "Status: DOWN"

  // 运行时
  if moving:
    Sim_Encoder_Run()           // 每50ms: UP→pulse++ / DOWN→pulse--
    print "Height: XXmm/XXmm"   // 每10ms
    Balance_Run()               // 双柱同步
    Safety_Check_Upper_Limit()  // pulse>=MAX_PULSES → 停+at_upper_limit
    Safety_Check_Lower_Limit()  // pulse<=0 → 停+at_lower_limit
    Safety_Check_Secondary_Descent()

  // Flash 存储（高度变化且间隔>5s → 写入）
  if pulse_count changed && now-last_save > 5s:
    App_W25Qxx_Height_Save()
```

### Safety_Task — 10ms

```
loop 10ms:
  Safety_Check_Stall():
    for each col: if RUNNING && now-last_pulse > 2s → Motor_Stop_All, alarm=STALL, Buzzer_On

  if confirm_key && secondary_descent_triggered → confirmed=1

  if alarm_state && button_stop → Safety_Alarm_Reset()
```

### Key_Task — 20ms

```
loop 20ms:
  Key_Scan():
    read 4 GPIO pins (PB12-15)
    for each key: if raw != pending → start debounce timer(2 ticks=20ms)
    update g_command = key_stable[]
    print "Key: UP/DOWN/STOP/CONFIRM" on press only

  Buzzer_Poll():
    if buzzer_off_tick reached → Buzzer_Off
```

### TIM7 ISR — 10ms

```
TIM7_IRQHandler → HAL_TIM_IRQHandler → HAL_TIM_PeriodElapsedCallback(tim.c):
  if TIM7 → Key_Debounce_Tick():
    for each key: debounce timer--, if timer=0 → accept pending value
```

## 三、ISR 中断

| 中断 | 触发 | 处理 |
|------|------|------|
| TIM2_CH1(PA0) | 1#霍尔脉冲 | Encoder_Capture_ISR → pulse_count++, last_pulse_tick=tick |
| TIM2_CH2(PA1) | 2#霍尔脉冲 | 同上 |
| EXTI0(PB0) | 下限位开关 | Encoder_Reset_Count(双柱清零), at_lower_limit=1 |
| EXTI1(PB1) | 1#防碰杆 | COLLISION_ENABLE=0时忽略 |
| EXTI2(PB2) | 2#防碰杆 | COLLISION_ENABLE=0时忽略 |

## 四、电机继电器时序

```
上升: 主接触器ON → 20ms → 上升接触器ON → 刹车释放
下降: 反转ON → 20ms → 主接触器ON → 20ms → 上升接触器ON → 刹车释放
停止: 两路上升OFF → 10ms → 两路主OFF → 反转OFF → 刹车抱紧
      (Motor_Stop_All 并行操作，单次10ms延时)
```

低电平吸合 (RELAY_ON = GPIO_PIN_RESET)。

---

## 五、限位行为

| 事件 | 动作 | 后续 |
|------|------|------|
| 上限位(pulse>=MAX_PULSES) | 停上升，at_upper_limit=1 | 可直接按下降 |
| 下限位(pulse<=0) | 停下降，at_lower_limit=1 | 可直接按上升 |
| 在顶按上升 | 阻止，打印 "At top (XXXXmm)" | — |
| 在底按下降 | 阻止，打印 "At bottom" | — |
| UP+DOWN同时按 | 立即停机，打印 "Status: STOP (conflict)" | — |

##六、Flash 存储策略

- **运动时**：高度变化 + 距上次存储 > 5秒 → 写入一次
- **停止时**：不写入（节省擦除寿命）
- **上电时**：从 Sector 2 (0x00002000) 恢复高度

## 七、告警状态

| 告警 | 触发 | 蜂鸣器 | 复位 |
|------|------|:---:|------|
| NONE | 正常 | — | — |

| STALL(2) | 2秒无脉冲 | 持续响 | 停止键 |
| BALANCE_TIMEOUT(3) | 平衡等待超10s | — | 停止键 |
